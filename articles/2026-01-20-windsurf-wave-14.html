Release Notes

Windsurf Wave 14: Arena Mode & Plan Mode による意思決定の革新

複数モデルの同時比較と、より深化したタスクプランニングを実現。

リリース日 2026-01-20
ツール Windsurf Editor
Wave Wave 14

モデルを選ぶ楽しみ、「Arena Mode」

2026 年 1 月 20 日にリリースされた Windsurf「Wave 14」における最大の目玉は、LMSYS Chatbot Arena の体験をデスクトップに持ち込んだ「Arena Mode」の実装です。

Arena Mode では、2 つの異なるモデルを並べて同じプロンプトに対する回答を生成し、パフォーマンスを直接比較することが可能です。 これにより、特定のプロジェクトや複雑なロジックに対して、Claude 3.5 Sonnet、GPT-4o、あるいは独自チューニングモデルのどれが最適かを、開発者が自らの目で確かめることができます。

賢い設計、Plan Mode

同時に追加された「Plan Mode」は、エージェントがコードを書く前に「どのようにタスクを遂行するか」の設計図を提示する機能です。 大規模な修正において、ファイルの変更順序や影響範囲を事前に予測・可視化することで、意図しない破壊的変更を防ぎます。

MCP エコシステムの深化

Windsurf は Model Context Protocol (MCP) の公式サポートにおいても先行しており、今回のアップデートではさらに安定性が向上しました。 JetBrains 系のプラグインとの連携も強化され、Cascade (エージェント) から MCP 経由で外部ツールを自在に操ることができるようになっています。

まとめ

Wave 14 は、AI モデルの性能を最大限に引き出しつつ、人間の監督責任（Human-in-the-loop）を「快適な UI」で解決しようとする Windsurf らしい意欲的なアップデートです。
