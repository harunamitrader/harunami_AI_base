Claude Opus 4.7: コーディングと視覚性能を強化して一般提供

長時間のコーディング、画像理解、自己検証を強化した Opus 4.7 が Claude 全製品と API で一般提供になりました。

リリース日 2026-04-16
バージョン Opus 4.7
ツール Claude

主な変更点

Anthropic は Claude Opus 4.7 を一般提供し、難度の高いソフトウェア工学タスクや長時間のエージェント作業での安定性を高めました。画像理解も高解像度寄りに強化され、文書、スライド、UI のような視覚要素を含む作業でも扱いやすくなっています。

  • コーディング性能: もっとも難しい開発タスクで Opus 4.6 より改善し、複数ステップの作業を落ち着いて進めやすくなりました。
  • 視覚理解: 画像をより高い解像度で扱えるようになり、図表やインターフェースの読み取りに強みがあります。
  • 利用範囲: Claude 全製品に加え、API、Amazon Bedrock、Vertex AI、Microsoft Foundry でも利用できます。モデルコードは `claude-opus-4-7` です。

影響や使いどころ

長時間のリファクタリング、複数ファイルにまたがる修正、コードレビュー、資料やインターフェースの下書き生成のように、推論の深さと出力の一貫性が必要な場面で向いています。

注意点

Anthropic は、Opus 4.7 を Mythos Preview ほど広くはしない一方で、サイバー用途の安全制御を強めたモデルとして出しています。正規のセキュリティ用途で使う場合は Cyber Verification Program の案内も確認したほうがよいです。

まとめ

Opus 4.7 は、Claude を会話相手から長時間作業の実務モデルへ寄せる更新です。コーディングと視覚系の処理を重く使うなら、まず試す価値があります。
