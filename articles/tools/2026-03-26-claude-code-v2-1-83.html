Claude Code v2.1.83: 会話履歴検索とサンドボックス制御の強化

2026-03-26 anthropic-ai/claude-code

Claude Code v2.1.83 がリリースされました。このアップデートでは、過去の会話履歴を素早く見つける Transcript Search や、画像対応を強化する Image Chips など、利便性を高める新機能が多数導入されています。また、企業利用を見据えたサンドボックスのポリシー管理機能も強化されています。

主な更新内容

  • Transcript Search: 過去のセッション履歴から特定のやり取りを検索できる機能。
  • Image Chips: ターミナル上で画像を扱いやすくするための新しいチップ UI。
  • ポリシー制御の柔軟化: managed-settings.d/ によるチーム別のポリシー適用をサポート。
  • 安全性向上: サンドボックスが起動できない場合に異常終了する sandbox.failIfUnavailable 設定の追加。
  • 操作性改善: 一言で指示を出す /effort コマンドや、セッションに名前を付ける -n フラグの追加。

パフォーマンス面でも起動と再開の速度が向上しており、よりスムーズな開発体験を提供します。
