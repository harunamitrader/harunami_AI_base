Cursor 3.1: Agents Window と Design Mode で開発体験を革新

複数のAIエージェントを並列実行できる「Agents Window」や、UIを直接指示できる「Design Mode」が登場。

リリース日 2026-04-15
バージョン 3.1
ツール Cursor

主な変更点

Cursor 3.0および3.1では、AIとの協調作業を次のレベルに引き上げる画期的な機能が導入されました。

  • Agents Window: 複数のAIエージェントをタイル状のレイアウトで並列実行・管理できる新しいインターフェース。ローカル、クラウド、SSH経由など、様々な環境でエージェントを同時に動かせます。
  • Design Mode: ブラウザで表示したUIに直接アノテーション（注釈）を付け、エージェントに具体的な修正指示を出せる機能。フロントエンド開発のフィードバックループを劇的に高速化します。
  • 音声入力のアップグレード: Ctrl+Mでの音声入力がより正確になり、波形表示やタイマーなどのUIも改善されました。
  • Bugbotの進化: プルリクエストのフィードバックから学習し、プロジェクト固有のルールを自動で適用する「Learned Rules」機能が追加されました。
  • セルフホスト・クラウドエージェント: 企業独自のインフラ内でエージェントを実行できるセルフホストオプションが提供され、セキュリティと柔軟性が向上しました。

影響や使いどころ

Agents Windowは、複雑なタスクを複数のエージェントに分担させたり、異なるモデルの出力を比較検討したりする際に絶大な効果を発揮します。Design Modeは、デザイナーと開発者の連携をスムーズにし、視覚的なフィードバックを直接コードに反映させることを可能にします。

まとめ

Cursor 3.1は、単なるAIコーディングアシスタントから、複数のエージェントを指揮して開発を進める「統合開発環境」へと進化を遂げました。マルチエージェントによる並列作業と、より直感的な指示方法が、開発の生産性を新たな高みへと導きます。
