OpenAI Codex: GPT-5.5 モデルのプレビュー対応と CLI v0.124.0

次世代モデル GPT-5.5 への前方互換性と、CLI の大幅な機能追加により、開発自動化がさらに加速。

リリース日 2026-04-24
種別 Update
ツール Codex

主な変更点

OpenAI Codex エコシステムにおいて、次世代モデルへの橋渡しとなる重要な更新が行われました。

  • GPT-5.5 プレビュー対応: まもなく登場する次世代モデル GPT-5.5 への接続オプションが開発者向けに公開されました。
  • Codex CLI v0.124.0: CLI が大幅にアップデートされ、プロジェクト全体の依存関係を解析し、最適なプロンプトを自動構築する「Smart Context」機能が追加されました。
  • マルチモーダル・フック: 画像や設計図からのコード生成において、特定のレイヤーを指定して指示を出す機能が強化されました。

影響や使いどころ

GPT-5.5 のプレビュー対応により、次世代の AI コーディング能力をいち早く体験・検証することが可能になります。また、CLI の Smart Context 機能は、複雑な既存コードベースの理解を劇的に助けます。

まとめ

Codexは、単一のモデル名を超え、高度な開発自動化を実現するための包括的なプラットフォームへと進化しています。
