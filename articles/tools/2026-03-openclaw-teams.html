OpenClaw 3月アップデート: Microsoft Teams 対応と Control UI の刷新

Published: 2026-03-24 | Feature: Teams SDK / Control UI

個人用 AI アシスタント OpenClaw は 2026年3月、エンタープライズプラットフォームとの親和性と、管理インターフェースのユーザビリティを向上させました。

1. Microsoft Teams への正式対応

公式 SDK への移行により、Teams 上でのウェルカムカード、ストリーミング返信、タイピングインジケーター等の AI らしいリッチな体験が可能になりました。メッセージの編集/削除もサポートされています。

2. Control UI のステータス管理刷新

スキルの状態（準備完了、要設定、無効）をタブで一括管理できる新 UI に刷新されました。また、各スキルの設定ダイアログが改善され、API キーの入力や設定がより直感的に行えるようになりました。

3. OpenAI 互換レイヤーの強化

`/v1/embeddings` への対応やモデルオーバーライド機能が追加され、既存の OpenAI 向けエコシステムとの連携がさらに強固になっています。
