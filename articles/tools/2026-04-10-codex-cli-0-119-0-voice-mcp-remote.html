Codex CLI 0.119.0: voice、MCP、remote workflow を強化

Realtime voice の v2 WebRTC 化、MCP Apps / custom server 対応、remote workflow と TUI の改善が入ったリリースです。

リリース日 2026-04-10
バージョン 0.119.0
ツール Codex

主な変更点

OpenAI の公式 Codex changelog では、2026-04-10 付で Codex CLI 0.119.0 が公開されています。Realtime voice sessions は v2 WebRTC path が既定になり、transport、voice selection、TUI media、app-server coverage の改善が入っています。

影響や使いどころ

MCP Apps と custom MCP servers では、resource read、tool-call metadata、custom-server tool search、server-driven elicitations、file-parameter upload などの対応が広がりました。Codex を MCP 中心の開発環境に組み込む場合、扱える入力とツール呼び出しの幅が広がります。

補足

remote / app-server workflow では egress websocket transport、remote --cd forwarding、sandbox-aware filesystem API、experimental codex exec-server が追加されています。TUI では Ctrl+O で最新回答をコピーでき、/resume もセッション ID や名前で直接ジャンプしやすくなりました。

まとめ

0.119.0 は、音声、MCP、remote 実行、TUI 操作をまとめて底上げする基盤更新です。ローカル CLI だけでなく app-server や MCP 連携まで使っているユーザーほど影響が大きいリリースです。
