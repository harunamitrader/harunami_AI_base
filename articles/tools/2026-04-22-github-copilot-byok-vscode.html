GitHub Copilot: VS Code 内での BYOK (Bring Your Own Key) 対応

独自の API キーを使用して Copilot を動作させることが可能に。より柔軟なモデル選択を実現。

リリース日 2026-04-22
種別 Update
ツール GitHub Copilot

主な変更点

以前からアナウンスされていた BYOK 機能が VS Code 拡張機能で利用可能になりました。

  • BYOK (Bring Your Own Key): OpenAI や Anthropic の独自 API キーを設定し、Copilot のインターフェースで使用可能。
  • モデルスイッチャーの改善: カスタムキー使用時に、利用可能なモデルをリストから即座に切り替え可能。

影響や使いどころ

標準プランに含まれない最新モデルや、特定の組織ポリシーに沿ったモデルを Copilot 上で活用したい開発者に最適です。

まとめ

プラットフォームとしての開放性が進み、ユーザーの選択肢が大幅に増えた重要なアップデートです。
