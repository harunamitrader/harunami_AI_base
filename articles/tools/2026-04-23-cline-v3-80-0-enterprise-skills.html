Cline v3.80.0: エンタープライズ向けリモートスキルと動的モデル推奨機能

組織内で共有可能なカスタムスキルセットを導入。タスクに応じた最適な AI モデルを Cline が提案。

リリース日 2026-04-23
バージョン v3.80.0
ツール Cline

主な変更点

チーム開発とコスト効率を重視した新機能が追加されました。

  • エンタープライズ・リモートスキル: Git リポジトリ経由でカスタムスキルをチーム全体に配信・同期。
  • 動的モデル推奨 (Smart Route): 入力されたタスクの複雑度を判定し、最適なコスト/性能のモデルを推奨。

影響や使いどころ

企業内でのベストプラクティス共有が容易になり、かつ API コストの最適化を自動で行えるようになります。

まとめ

大規模組織での導入を意識した、管理性と効率性に優れたアップデートです。
