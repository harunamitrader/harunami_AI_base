Grok CLI v1.0.0-rc 系列登場: マルチエージェント対応と TUI 刷新

Grok CLI (Community Edition) の最新アップデート情報の要約。

リリース日 2026-03-23
バージョン v1.0.0
ツール Grok CLI (Community Edition)

2026年3月、xAI の高性能モデル Grok をターミナルから操作するための CLI ツール（superagent-ai/grok-cli）が、大規模なバージョンアップ（v1.0.0-rc 系列）を遂げました。

1. マルチエージェント機能の完全サポート

3月23日適用の v1.0.0-rc4 / rc5 では、バックエンドの Grok-4.20 Multi-agent API に正式対応しました。複数のサブエージェントを自動生成し、複雑なタスクを並列処理する「AI チーム」を CLI 経由で直接指揮することが可能になっています。

2. ターミナル UI (TUI) の劇的な進化

Bun と OpenTUI をベースにしたリッチなターミナルインターフェースが導入されました。ストリーミング応答時のローディングアニメーションや、JSON 出力モードの追加により、対話型のみならずヘッドレス（自動化スクリプト）での運用性能も向上しています。

3. 最新モデルへの追従 (Grok-4 / Grok-3 mini)

推論オプション `reasoningEffort` の調整が追加され、Grok-3 mini や最新の Grok-4.20 向けに最適化された推論実行をコマンドラインからスイッチできるようになりました。
