Release Notes

Antigravity v1.21.6: Linux サンドボックス対応と MCP 認証の強化

より安全なコード実行環境の提供と、外部ツール連携（MCP）の認証フローを最適化。

リリース日 2026-03-31
バージョン v1.21.6
ツール Antigravity

主な変更点

Google の公式 AI エージェント環境である Antigravity が v1.21.6 にアップデートされました。このバージョンでは、セキュリティ基盤の飛躍的な向上が主要な成果となっています。

1. Linux サンドボックスの公式サポート

エージェントが生成・実行したコードがシステム全体に影響を与えないよう、Linux 環境下での強力なサンドバッシング（隔離環境）がデフォルトで利用可能になりました。これにより、より信頼性の高い開発体験が実現します。

2. MCP (Model Context Protocol) 認証の強化

外部のツールセットやデータリポジトリとの通信プロトコルである MCP において、OAuth 等の認証フローがシームレスになりました。複数のサービス間で、エージェントが安全かつ簡単に情報をやり取りできるように設計されています。

まとめ

v1.21.6 への移行により、Antigravity は「安全性」と「接続性」の両面を強化しました。大規模プロジェクトにおけるセキュアな開発基盤としての信頼性は、さらに強固なものとなっています。
