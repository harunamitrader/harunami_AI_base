Gemini CLI v0.39.0-nightly.20260411: 最新 nightly タグを更新

Gemini CLI の nightly 系が v0.39.0-nightly.20260411.0957f7d3e に進み、次期安定版へ向けた検証対象が更新されました。

リリース日 2026-04-11
バージョン v0.39.0-nightly.20260411
ツール Gemini CLI

主な変更点

Google Gemini CLI の公式 GitHub Releases では、2026-04-11 付で nightly の最新タグが v0.39.0-nightly.20260411.0957f7d3e へ更新されています。stable ではなく pre-release 系の更新なので、安定版利用者が即時に切り替えるべきリリースではありません。

実務で見るポイント

nightly は次期 stable に入る可能性がある変更を早期に検証するための参照先です。CLI の挙動、認証まわり、サンドボックス、エージェント実行フローを固定しているチームでは、検証環境だけで先に動作差分を確認するのが安全です。

何を確認すべきか

既定の運用が stable の v0.37.1 にある場合は、そのまま維持して問題ありません。nightly を追う環境では、セットアップスクリプトや CI の固定タグを更新する前に、既存プロンプト、MCP 連携、権限設定、非対話実行で回帰が出ないかを確認しておきたい更新です。

まとめ

v0.39.0-nightly.20260411 は、Gemini CLI の次期系列を先取りして検証するための nightly 更新です。production の既定値を動かすリリースというより、pre-release 追従環境の検証対象を更新するタイミングとして扱うのが現実的です。
