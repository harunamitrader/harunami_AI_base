Cline v3.78.0: Spend Limit UI と read_file 表示を改善

支出上限到達時の専用エラー UI と、チャット内で実際の read_file 行範囲を表示する修正が入った小粒な安定化リリースです。

リリース日 2026-04-10
バージョン v3.78.0
ツール Cline

主な変更点

公式 GitHub Releases では、2026-04-10 付で v3.78.0 が公開されています。追加点は、spend cap に到達したときに専用の Spend Limit Reached エラー UI を表示することです。

影響や使いどころ

AI コーディングツールでは、利用上限や課金制限に当たったときの表示が曖昧だと原因切り分けに時間を使います。今回の UI 追加により、モデルエラーなのか、ネットワーク問題なのか、単純に spend limit なのかを判断しやすくなります。

補足

修正点として、チャット UI に実際の read_file line range が表示されるようになりました。エージェントがどの範囲を読んだのかを追いやすくなり、レビューやデバッグ時の透明性が少し上がります。

まとめ

v3.78.0 は派手な新機能ではありませんが、運用時の誤解を減らす実務的な更新です。Cline を長く動かすユーザーほど、上限到達とファイル参照範囲の可視化は効いてきます。
