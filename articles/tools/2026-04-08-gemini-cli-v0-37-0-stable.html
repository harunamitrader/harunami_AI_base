Release Notes

Gemini CLI v0.37.0 Stable: Plan Mode の強化とセキュリティサンドボックスの改善

プレビュー版での実験的機能が正式に安定版として昇格。Plan Mode の信頼性向上と、.env ファイルなど機密情報のサンドボックス保護を強化しました。

リリース日 2026-04-08
バージョン v0.37.0
ツール名 Gemini CLI

主な変更点と最新情報

Gemini CLI v0.37.0 は、数週間にわたるプレビュー・サイクルを経て、多くの革新的な機能を Stable 版へと昇格させました。特筆すべきは「Plan Mode」の進化で、信頼できないフォルダでの動作や、承認前のディスカッションを優先するフローが導入され、より安全に高度なタスクを委任できるようになりました。

注目の新機能: ブラウザ・エージェントの進化

ブラウザ・エージェント機能も大幅に強化され、永続的なセッション管理や、読み取り専用ツールの動的発見が可能になりました。これにより、ブラウジングを伴うタスクの文脈維持が容易になり、不必要なリソース消費を抑えた効率的な動作を実現しています。

高度なサンドボックス・セキュリティ

セキュリティ面では、Windows および Linux におけるサンドボックスの動的拡張機能が導入されました。特に .env ファイルなどの機密情報へのアクセスを厳密に制限する機能が追加され、OS レベルでのパス禁止設定もサポートされました。

まとめ

「Tokyo Night」テーマの追加やメッセージ・キュー機能など、ユーザー体験面でも多くのアップデートが含まれています。今回のリリースは、セキュリティとパフォーマンス、そして使いやすさを高いレベルで統合した、まさに「次世代の CLI ツール」と呼ぶにふさわしい内容です。
