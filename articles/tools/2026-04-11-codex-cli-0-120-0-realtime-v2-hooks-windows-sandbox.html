Release Notes

Codex CLI 0.120.0: Realtime V2 と hooks 表示、Windows sandbox を改善

background agent の進捗ストリーミング、hooks 表示の整理、code-mode outputSchema、Windows sandbox 修正が入った更新です。

リリース日 2026-04-11
バージョン 0.120.0
ツール Codex

主な変更点

OpenAI の公式 Codex changelog では、2026-04-11 付で Codex CLI 0.120.0 が公開されています。Realtime V2 は background agent の進捗を作業中にストリームでき、実行中の response が終わるまで follow-up response をキューできるようになりました。

影響や使いどころ

hooks 周辺では、TUI で実行中 hooks と完了済み出力を見分けやすくなり、status line には rename 済み thread title を含められます。code-mode tool declaration には MCP outputSchema details が入るようになり、構造化 tool result の型付けがより正確になります。

補足

Windows elevated sandbox では split filesystem policies、read-only carveouts、symlinked writable roots の扱いが修正されています。さらに codex --remote wss://... の TLS websocket panic、tool search result の順序保持、app-server MCP cleanup なども修正対象です。

まとめ

0.120.0 は、Codex を長時間・遠隔・hooks 付きで運用するユーザーに効くリリースです。特に Realtime V2 と Windows sandbox の修正は、エージェント実行の安定性に直結します。
