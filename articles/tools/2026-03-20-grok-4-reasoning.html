xAI Grok-4 発表: PhD レベルの推論と科学・コーディング特化

Grok (xAI) の最新アップデート情報の要約。

リリース日 2026-03-20
バージョン Latest
ツール Grok (xAI)

xAI は 2026年3月、次世代モデル Grok-4 の開発状況と一部 API のアップデートを発表しました。競合モデルを圧倒する推論性能を目指しています。

1. Grok-4: 科学・数学における「PhD レベル」の推論

最新の Grok-4 は、単なる知識量だけでなく、多段階の複雑な論理展開を伴う「科学的推論」において PhD レベルのパフォーマンスを発揮するように設計されています。特に数学やコーディングの難問解決能力が大幅に向上しました。

2. Grok Imagine のアップデート

画像生成機能 Grok Imagine が 3 月に更新され、新しいスタイライズド・テンプレート（Chibi テンプレート等）が追加されました。Elon Musk 氏はこの機能のさらなる「エピックな」更新を示唆しています。

3. 企業向けマルチエージェント API

Enterprise API において Grok 4.20 Multi-agent Beta が公開されました。複数の Grok インスタンスを協調させて複雑なパイプラインを構築することが可能です。
