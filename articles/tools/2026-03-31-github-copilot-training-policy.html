Important Announcement

GitHub Copilot: インタラクションデータの学習利用（4月24日より）に関する重要告知

Free/Pro ユーザーのコード断片等がモデル学習に活用される方針に。オプトアウト設定が可能。

リリース日 2026-03-31
カテゴリ Policy Update
ツール GitHub Copilot

概要

GitHub は、Copilot (Free, Pro, Pro+) ユーザーを対象とした新しいプライバシーポリシーを発表しました。2026年4月24日以降、AI とのやり取り（入力内容、提案、コード断片、コンテキスト等）が、将来のモデル改善や学習のためにデフォルトで利用されることになります。

重要なポイント

  • 対象ユーザー: 個人向けの Free、Pro、および Pro+ プランが対象です。
  • 対象外: 企業向けの Business および Enterprise プランをご利用のユーザーは、この変更の影響を受けません（データは引き続き学習に使用されません）。
  • オプトアウト: 設定画面より、データの学習利用を拒否する設定（Opt-out）が可能です。

対応の推奨

社外秘のソースコードや機密情報を取り扱っている個人開発者は、規約が変更される4月24日までに、設定画面から現在の設定内容を再確認することをお勧めします。データの安全性と利便性のトレードオフを考慮した判断が求められています。

まとめ

今回の変更は、より高機能なモデルを生成するためのデータエコシステム構築の一環ですが、ユーザーの「コードの所有感」や「プライバシーへの懸念」を呼ぶ可能性も指摘されています。
