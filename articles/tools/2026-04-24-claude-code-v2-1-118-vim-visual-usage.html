Claude Code v2.1.118: Vimビジュアルモードの実装と /usage への統合

Vimビジュアルモードの追加やコマンド統合により、ターミナルでの操作性と統計管理が向上。

リリース日 2026-04-24
バージョン v2.1.118
ツール Claude Code

主な変更点

今回のアップデートでは、エディタ操作の利便性向上とコマンドの整理が行われました。

  • Vimビジュアルモード: ターミナル内での範囲選択が可能になり、コード編集の直感性が向上しました。
  • コマンド統合: /cost と /stats が /usage に統合され、利用状況を一括で確認できるようになりました。
  • カスタムテーマ: TUIの配色をユーザー設定で変更できるテーマ機能が正式にサポートされました。
  • MCPフック強化: MCPツールの直接呼び出しにおける安定性とフック処理の柔軟性が向上しました。

影響や使いどころ

ターミナル作業がメインの開発者にとって、Vim操作の充実と統計情報の集約は、作業リズムを崩さない効率的な開発を支えます。特に利用統計の統合は、APIコストの管理をより簡便にします。

まとめ

Claude Codeは単なるチャットUIを超え、ターミナル上の完全な開発環境へと一歩近づきました。操作性の細かな改善が、長期的な生産性向上に寄与します。
