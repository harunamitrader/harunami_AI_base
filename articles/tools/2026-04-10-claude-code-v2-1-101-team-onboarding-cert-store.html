Claude Code v2.1.101: team onboarding と企業 TLS 対応を正式化

/team-onboarding、OS CA certificate store の既定信頼、remote session の初期環境作成、resume や権限制御の修正がまとまった更新です。

リリース日 2026-04-10
バージョン v2.1.101
ツール Claude Code

主な変更点

Anthropic の公式 GitHub Releases では、2026-04-10 付で v2.1.101 が Latest として公開されています。新しく /team-onboarding が追加され、ローカルの Claude Code 利用状況からチームメイト向けの ramp-up guide を生成できるようになりました。

影響や使いどころ

企業環境では、OS の CA certificate store を既定で信頼するようになった点が重要です。TLS inspection や社内プロキシを挟む環境でも追加設定なしで動きやすくなり、必要なら CLAUDE_CODE_CERT_STORE=bundled で従来寄りの挙動に戻せます。

補足

/ultraplan などの remote-session 系機能は、事前に Web 側で環境を作らなくても既定の cloud environment を自動作成するようになりました。さらに resume、permissions.deny、MCP outputSchema、plugin、LSP、Grep fallback、長時間セッションのメモリリークなど、実運用で詰まりやすい領域の修正も多く含まれます。

まとめ

v2.1.101 は、個人向けの小改善というより、チーム導入と企業ネットワーク下の安定運用を前に進めるリリースです。Claude Code を標準ツールとして展開する組織ほど確認優先度が高い内容です。
