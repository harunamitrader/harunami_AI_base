Claude Code v2.1.116: `/resume` の劇的な高速化と MCP 起動最適化

大規模セッションの復帰が最大67%高速化。MCPサーバーの起動負荷も軽減されました。

リリース日 2026-04-21
バージョン v2.1.116
ツール Claude Code

主な変更点

今回のアップデート v2.1.116 では、特にヘビーユーザー向けのパフォーマンス改善が中心となっています。

  • `/resume` の高速化: 40MBを超えるような大規模なログを持つセッションにおいて、コンテキストの復元プロセスが最大67%高速化されました。
  • MCP 起動の効率化: 複数の stdio サーバーを使用している環境において、起動時のオーバーヘッドを削減。テンプレートリストの取得などをメンション時まで遅延させることで、初期起動がよりスムーズになりました。
  • ターミナルスクロールの改善: VS Code、Cursor、Windsurf 内蔵のターミナルで実行する際のスクロール描画が、より滑らかに調整されました。

影響や使いどころ

複雑なプロジェクトで長時間の作業を行い、過去のコンテキストを頻繁に呼び出す開発者にとって、`/resume` の待ち時間短縮は大きな利便性向上となります。また、多数の MCP サーバー（MCP Servers）を常時稼働させている場合も、シェル起動時のもたつきが解消されます。

まとめ

v2.1.113 でのネイティブバイナリ化に続き、今回の v2.1.116 でも「体感速度」に直結する重要な最適化が行われました。特に大規模なコーディングタスクを AI と協力して進める際のボトルネックが、着実に排除されています。
