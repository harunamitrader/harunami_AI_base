Claude: サードパーティ経由の Pro 枠利用を制限

OpenClaw などの外部ツールでの Pro サブスクリプション利用が不可となり、従量課金が必須となりました。

更新日 2026-04-04
種別 利用規約変更
ツール Claude

主な変更点

Anthropic は、サードパーティ製クライアントを通じた Claude Pro サブスクリプション枠の利用を制限しました。これまで OpenClaw などの外部ツールで Web 版のセッションを利用できていたユーザーは、今後 API 経由の従量課金（Pay-as-you-go）に移行する必要があります。

影響と使いどころ

この変更は、外部ツールを常用していたパワーユーザーに大きな影響を与えます。公式の Web/Mobile アプリ以外で Claude を利用する場合は、API キーを取得し、トークン量に応じた支払いを設定する必要があります。

まとめ

セキュリティとサービスの安定性確保を目的とした措置と考えられますが、サードパーティ製エコシステムへの接続方法が、より公式の API モデルへと明確に一本化される形となりました。
