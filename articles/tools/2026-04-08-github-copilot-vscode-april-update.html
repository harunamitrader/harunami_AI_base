Release Notes

GitHub Copilot VS Code 4月アップデート: 動画のプレビュー対応と回答コピーの改善

VS Code 版での画像カルーセルやエクスプローラー・メニューが動画再生に対応。また、エージェントの思考プロセスを除いた「最終回答のみ」のコピー機能が追加されました。

リリース日 2026-04-08
バージョン Extension v1.114
ツール名 GitHub Copilot

主な変更点と最新情報

GitHub Copilot VS Code エクステンションの 2026年4月アップデートでは、開発者のワークフローをより快適にする細かな、しかし強力な改善が多く含まれています。特に TypeScript 6.0 への完全対応により、最新の型システム機能を活用したコード生成が可能になりました。

注目の新機能: 動画プレビュー

チャットのアタッチメントやエクスプローラーのコンテキストメニューにおいて、動画ファイルが直接プレビューできるようになりました。再生コントロールも完備されており、UI の動作サンプルやデモ動画を確認しながらのコーディングを強力にサポートします。

注目の新機能: 最終回答のみをコピー

新しいコンテキストメニュー・コマンドにより、エージェントの「思考プロセス」や「ツール呼び出しログ」を除去し、整形された最終的な Markdown 回答のみをワンクリックでコピーできるようになりました。ドキュメントへの転記がこれまで以上にスムーズに行えます。

セマンティック検索の純粋化

ワークスペース検索を担う #codebase ツールは、精度にばらつきがあった従来のあいまい検索を廃止し、セマンティック検索（意味論的検索）のみにフォーカスするようになりました。これにより、関連性の高いコード片をより確実に見つけ出せるようになっています。

まとめ

過去のチャットセッションをトラブルシューティングに活用できる #session 変数の導入など、Copilot は単なるコード補完ツールから、開発の「文脈」をより深く理解するパートナーへと進化し続けています。
