GitHub Copilot: モデルセレクターの整理と GPT-5.3-Codex の段階的廃止

GitHub Copilot のモデル選択肢が整理され、旧世代の GPT-5.3-Codex モデルの提供が段階的に終了します。

リリース日 2026-04-27
カテゴリ AI Coding
ツール GitHub Copilot

主な変更点

GitHub Copilot において、モデルの整理とラインナップの更新が行われました。

  • GPT-5.3-Codex の廃止: 旧世代のコーディング特化モデルであった GPT-5.3-Codex が、モデル選択メニューから段階的に削除されます。
  • モデルセレクターの簡素化: 自動選択機能（Auto）の精度向上に伴い、ユーザーが迷わずに最適なモデルを利用できるようインターフェースが整理されました。
  • 推奨モデルの移行: 今後は GPT-5.5 や Claude Opus 4.7 への利用が推奨され、より高度な推論を必要とするタスクへの対応が強化されています。

影響や使いどころ

特定の旧モデルに依存したワークフローを持っているユーザーは、新しいモデルへの移行準備が必要です。最新の GPT-5.5 世代は Codex よりも汎用性と推論能力で勝っており、通常の開発業務では性能向上が期待できます。

まとめ

GitHub はモデルのポートフォリオを継続的に更新しており、旧世代モデルの廃止は最新のフロンティアモデルへのリソース集中を意味しています。
