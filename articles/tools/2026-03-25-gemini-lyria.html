Google AI Studio 3月更新: Lyria 3 音楽生成と新課金プラン

Google AI Studio / Gemini API の最新アップデート情報の要約。

リリース日 2026-03-25
バージョン Latest
ツール Google AI Studio / Gemini API

Google は 2026年3月、AI Studio および Gemini API においてマルチモーダル能力と開発者向けプラットフォーム機能を大幅に強化しました。

1. Lyria 3 音楽生成モデルの公開

3月25日に発表された Lyria 3 (`lyria-3-pro-preview`) は、テキストや画像からの高品質な 48kHz ステレオ音源生成をサポートします。API 経由でクリエイティブな音響制作が可能になりました。

2. 柔軟な課金体系 (Pay-as-you-go)

AI Studio において Prepay および Postpay の課金プランが正式にロールアウトされました。これにより、個人の実験から大規模なサービス展開まで、予算に合わせたスケーリングが容易になります。

3. Gemini 3.1 Pro プレビュー統合

最新の Gemini 3.1 Pro への一本化が進み、より高速で精度の高いレスポンスが標準となりました。また、マルチモーダル埋め込みモデル `gemini-embedding-2-preview` も提供開始されています。
