ChatGPT: オープンソースのエージェント仕様「Symphony」を発表

OpenAI がマルチエージェントのオーケストレーションを定義するオープンソース仕様「Symphony」を発表。エージェント間の協調とタスク分散の標準化を目指します。

リリース日 2026-04-27
カテゴリ AI Others
ツール ChatGPT / Symphony

主な変更点

OpenAI は、自律型 AI エージェントが互いに協力して複雑な問題を解決するための新しい標準プロトコル「Symphony」を公開しました。

  • Symphony プロトコルの公開: 複数のエージェント間でタスクを分割、委譲、結果を統合するための標準的なデータ形式と通信仕様を定義しました。
  • オープンソース化: 仕様は GitHub で公開されており、OpenAI 以外のモデルやプラットフォームでも Symphony 準拠のエージェントを構築可能です。
  • オーケストレーションの自律化: 親エージェントが子エージェントの能力を把握し、最適なエージェントに動的にタスクを振り分けるためのメタデータ定義が含まれています。

影響や使いどころ

これまではプラットフォームごとに独自実装されていた「マルチエージェント連携」が共通化されることで、異なるベンダーのエージェント同士を組み合わせた高度な自動化が可能になります。OpenAI 自身の Workspace Agents も、この Symphony 仕様に基づいて設計されています。

まとめ

OpenAI は MCP (Model Context Protocol) に対抗する形で、より「実行」と「協調」に重きを置いた標準化を進めており、エージェントエコシステムの主導権を握る狙いが見て取れます。
