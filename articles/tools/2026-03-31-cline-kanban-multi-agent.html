Release Notes

Cline Kanban: 複数の CLI エージェントを Git Worktree でオーケストレーション

並列実行される AI エージェントを視覚的に管理し、大規模タスクの分業を強力にサポート。

リリース日 2026-03-31
新機能 Cline Kanban
ツール Cline

主な変更点

Cline のエコシステムから、複数の自律型エージェントを指揮（Orchestration）するための新ツール「Cline Kanban」が発表されました。これは単一のエージェントではなく、チームとしての AI 開発を加速させる革新的なアプローチです。

1. Git Worktree による並列開発

Cline Kanban は、各エージェントを独立した Git Worktree（作業ツリー）へ自動的にアサインします。一方のエージェントが環境構築を行う間に、別の方のエージェントがテストを書くといった、物理的に分断された空間での安全な並列作業が可能になります。

2. 視覚的なステータス管理

カンバン形式のボードを通じて、各エージェントが進めているタスクの進捗、成果物のレビュー待ち、エラーによる停止などを一目で俯瞰できます。個別の CLI エージェントを追いかける負担が劇的に軽減されます。

まとめ

Cline は個人用エージェントの枠組みを越え、複数の AI コンポーネントをマネジメントする「開発プラットフォーム」へと進化しつつあります。Cline Kanban はその中心的な役割を果たすことになるでしょう。
