主な変更点

GitHub Blog の Changelog には、2026 年 4 月 1 日付で「GPT-5.4 mini is now available in Copilot Student auto model selection」と記載されています。対象は Copilot Student の自動モデル選択フローです。

今回確認できる公式情報は、あくまで auto model selection の候補に GPT-5.4 mini が追加されたという点です。手動モデル選択の仕様変更や他プランへの即時展開までは、この changelog 断片からは読み取れません。
