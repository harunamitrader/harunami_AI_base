Release Notes

Claude Code v2.1.87: 「Auto Mode」導入による自律的な権限管理の実現

安全なアクションを AI が自動判定し、開発者の承認頻度を大幅に削減。Computer Use も一般開放。

リリース日 2026-03-31
バージョン v2.1.87
ツール Claude Code

主な変更点

Anthropic の公式 CLI ツールである Claude Code が v2.1.87 にアップデートされ、待望の自律実行機能「Auto Mode」が搭載されました。

1. Auto Mode による権限管理の極致

これまではファイル読み込みやターミナル実行のたびにユーザーの承認が必要でしたが、Auto Mode では「安全な操作」を分類器が自動判定します。これにより、環境構築やコード解析といった長時間のタスクを、AI が途切れることなく自律的に完遂できるようになりました。

2. Computer Use の一般提供 (GA)

ブラウザ操作や OS ツールへのアクセスを含む Computer Use 機能が、すべての Pro / Max ユーザーに開放されました。CLI から直接ウェブサイトの情報を取得したり、外部アプリを操作するタスクの自動化がさらに加速します。

まとめ

v2.1.87 へのアップデートにより、Claude Code は「対話型ツール」から、自ら判断し行動する「自律型エージェント」へと一歩前進しました。開発者は、クリエイティブな設計により集中できるようになります。
