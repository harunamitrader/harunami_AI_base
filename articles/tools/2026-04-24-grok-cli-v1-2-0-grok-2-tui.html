Grok CLI v1.2.0: Grok-2 対応と TUI 刷新

最新モデル Grok-2 への完全対応と、より直感的になった TUI により開発体験が劇的に進化。

リリース日 2026-04-24
バージョン v1.2.0
ツール Grok CLI

主な変更点

メジャーマイナーアップデートとなる v1.2.0 では、モデルサポートとインターフェースの両面で大きな進化がありました。

  • Grok-2 サポート: xAIの最新モデル Grok-2 に対応し、より高度な推論とコーディング能力を活用できるようになりました。
  • TUI の刷新: ターミナルインターフェースが再設計され、マルチウィンドウ表示やセッション切り替えがよりスムーズになりました。
  • パフォーマンス向上: 大規模なディレクトリ解析時のメモリ消費量が約30%削減されました。

影響や使いどころ

Grok-2 の推論能力を最大限に引き出す TUI の刷新は、複雑なプロジェクトのコード修正やデバッグにおいて、より高い集中力を維持させます。

まとめ

Grok CLIは、最新モデルのポテンシャルをいち早く開発者の手元に届けるツールとして、その地位を確固たるものにしています。
