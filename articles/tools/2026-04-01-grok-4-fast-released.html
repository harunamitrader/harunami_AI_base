Grok 4 Fast: 推論速度を極めた新モデルがリリース

xAI が最新の「Fast」モデルを公開。回答までの時間が半分以下に短縮され、API コストも削減。

リリース日 2026-04-01
モデル Grok 4 Fast
ツール Grok (xAI)

主な変更点

  • 推論速度の向上: Grok 4 通常モデルと比較して、約 2.5 倍の生成速度を達成しました。
  • API コストの削減: 開発者向け API において、Fast モデルのトークン単価が大幅に抑えられています。
  • 利用範囲: X Premium ユーザーへの展開が開始されており、順次 API (x.ai console) でも提供が開始されます。

まとめ

速度と推論のバランスを最適化した Grok 4 Fast の登場により、リアルタイム性が求められるチャットやエージェント開発がより円滑になります。
