Claude Code v2.1.98: Vertex AI 導入と Monitor tool を追加

Google Cloud の Vertex AI 経由で Claude を使えるようにしつつ、新しい Monitor tool と複数の安定性修正が加わった更新です。

リリース日 2026-04-09
バージョン v2.1.98
ツール Claude Code

主な変更点

公式 GitHub Releases では、2026-04-09 付で v2.1.98 が公開されています。今回の目玉は Google Cloud Vertex AI のサポートで、Claude API 直結ではなくクラウド基盤経由での採用を進めたい組織にとって導入しやすい構成になりました。

新機能の見どころ

あわせて Monitor tool が追加され、長く走る処理や状態監視が必要なワークフローの扱いが広がっています。CLI ベースのエージェント運用では、単純な実行だけでなく経過観察の導線が重要になるため、地味でも効く追加です。

修正面の意味

リリースではそのほかに UI 表示やツール周辺の不具合修正も含まれています。クラウド接続先の選択肢を広げつつ、セッション運用の細かな詰まりを減らすバランス型の更新と見てよさそうです。

まとめ

v2.1.98 は、大きな見出しとしては Vertex AI 対応、実務上は Monitor tool の追加が効くリリースです。特に企業環境で Claude Code を採用したいチームにとって確認優先度が高い版です。
