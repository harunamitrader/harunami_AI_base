Cursor 3月25日アップデート: 自社ホスト型エージェントと JetBrains 統合

2026-03-26 getcursor/cursor

Cursor エディタの最新アップデート（2026年3月25日発表）により、エンタープライズ向けの「Self-hosted Cloud Agents」と、JetBrains IDE ユーザー待望の統合機能が発表されました。これにより、Cursor の強力な AI エージェント機能をより広い環境で、安全に利用できるようになります。

主な発表内容

  • Self-hosted Cloud Agents: 自社ネットワーク内のインフラ上で AI エージェント（VM）を実行可能に。データの外部流出リスクを抑えつつ、Cursor 独自のエージェント機能を利用できます。
  • JetBrains IDEs 統合 (ACP): IntelliJ IDEA、PyCharm、WebStorm 等で、Cursor の AI エージェント機能（マルチモデル・ハーネス等）が利用可能になりました。Agent Client Protocol (ACP) を介して動作します。
  • Composer 2 の更なる展開: 先週発表された Moonshot Kimi K2.5 ベースの独自モデル「Composer 2」が、さらに安定し、大規模プロジェクトでの精度が向上しています。

Cursor Marketplace にもパートナー企業のプラグインが30種類以上追加されており、エコシステムの拡大が加速しています。
