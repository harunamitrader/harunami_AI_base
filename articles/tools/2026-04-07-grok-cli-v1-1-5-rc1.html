Grok CLI v1.1.5-rc1: Plan Mode の安定化とマルチファイル・トランザクション

複数ファイルへのアトミックな変更、失敗時の自動ロールバック、Plan Mode のユーザビリティ向上。

リリース日 2026-04-07
バージョン v1.1.5-rc1
ツール Grok CLI

アトミックなマルチファイル変更

Grok CLI の最新プレリリース版では、開発プロセスをより安全にするための「マルチファイル・トランザクション」機能が導入されました。これにより、複数のファイルにまたがる変更がひとつの「トランザクション」として扱われ、一部の書き込みが失敗した場合でも、システム全体をクリーンな状態に保つための自動ロールバックが実行されます。

Plan Mode の安定化

リリース前の設計やリサーチを安全に行う「Plan Mode」の安定性が向上しました。複雑な依存関係を持つプロジェクトにおいても、より正確な戦略を策定可能になり、開発の不確実性を軽減します。

プロバイダー管理の強化

新たに導入された `/provider` コマンドにより、Grok 4、Claude、Gemini、ローカルの Ollama などの AI モデル間を動的に切り替えることが容易になりました。タスクの種類に応じて最適なモデルを瞬時に選択できる柔軟性が提供されます。

まとめ

v1.1.5-rc1 は、開発者が CLI エージェントに任せる「自動化」の範囲を、より複雑でリスクの高い操作へと安全に拡張するための重要な一歩です。
