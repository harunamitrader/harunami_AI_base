Security Advisory

OpenClaw: セキュリティ懸念による逆風と偽トークン詐欺への警告

自律エージェントによるデータ漏洩リスクの指摘と、プロジェクトを騙るフィッシング詐欺が急増。

リリース日 2026-03-31
ステータス Advisory
ツール OpenClaw

概要

爆発的な人気を博しているオープンソースの自律型 AI エージェント「OpenClaw」に対し、セキュリティ専門家から重大な警告が出されています。特に、自律的な指令実行プロセスにおける「誤操作によるファイル削除」や「外部への意図しないデータ送信」のリスクが表面化しています。

主な懸念事項

  • サンドボックスの未徹底: 一部の環境下で OS 操作権限が広範囲に渡り、エージェントが誤った判断を下した際に取り返しのつかない変更が加わるリスク。
  • フィッシング詐欺の横行: GitHub リポジトリや公式を装ったサイトにおいて、偽の $CLAW トークン（仮想通貨）を配布してウォレットのアクセス権を奪う攻撃が多数確認されています。

注意喚起

公式チームは「現在進行形でセキュリティパッチを配布中」としていますが、エンタープライズ環境での利用には慎重な判断が求められます。また、仮想通貨トークンの公式な発行予定はなく、SNS 等での勧誘はすべて詐欺であると強く警告しています。

まとめ

OpenClaw は非常に強力な開発支援・自動化ツールですが、その自律性の高さが仇となり、攻撃者の標的や不注意によるトラブルの温床となっています。利用の際は、必ず権限を制限した独立した VM 上で動作させることを推奨します。
