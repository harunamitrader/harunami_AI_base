Release Notes

Claude Code v2.1.108: プロンプトキャッシュとセッション要約(recap)を強化

プロンプトキャッシュのTTL設定や、セッション復帰時の要約機能追加で、より効率的な利用が可能になりました。

リリース日 2026-04-15
バージョン v2.1.108
ツール Claude Code

主な変更点

Claude Code の最新アップデートでは、開発効率とセッション管理を向上させるための機能が複数追加されました。

  • プロンプトキャッシュの強化: APIキーやBedrock、Vertex等でのプロンプトキャッシュTTL（有効期限）を1時間に設定できる環境変数が追加され、繰り返しの呼び出しにおけるコストと時間を削減します。
  • セッション管理の改善: セッション復帰時にコンテキストを要約する「recap（要約）」機能が追加され、長時間の作業中や中断後もスムーズに開発を再開できます。
  • スキルの統合: /init や /review といった組み込みコマンドを、モデルが「Skillツール」を通じて自律的に呼び出せるようになり、より複雑なタスクの自動化が期待できます。
  • パフォーマンス改善: ファイル読み込み時のメモリ使用量が削減され、より大規模なプロジェクトでも安定して動作するようになりました。

影響や使いどころ

今回のアップデートにより、長時間のコーディングセッションや大規模なリポジトリを扱う開発者にとって、Claude Code はさらに強力なツールとなります。特に、キャッシュ設定や要約機能は、API利用コストを意識するユーザーや、断続的に作業を行うリモートワーカーにとって大きなメリットとなるでしょう。

まとめ

v2.1.108は、パフォーマンスの改善と、よりインテリジェントなセッション管理機能の追加を両立させた、実用的なアップデートです。
