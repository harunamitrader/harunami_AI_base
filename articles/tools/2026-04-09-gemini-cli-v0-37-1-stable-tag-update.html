Gemini CLI v0.37.1: Stable の最新版タグを更新

大規模な新機能追加ではなく、Stable 系の最新参照先が v0.37.1 に進んだことを確認できる更新です。

リリース日 2026-04-09
バージョン v0.37.1
ツール Gemini CLI

主な変更点

Google Gemini CLI の公式 GitHub Releases では、2026-04-09 付で stable の最新タグが v0.37.1 へ更新されています。リリースノートの粒度は大きくありませんが、安定版の参照先が進んだこと自体が運用上の重要ポイントです。

実務で見るポイント

nightly や preview を追わず stable を基準に導入している環境では、どのタグを既定として配布するかがそのまま再現性につながります。今回のような小さな版上げでも、CI やセットアップ手順に固定バージョンを書いている場合は追従判断が必要です。

何を確認すべきか

特にチーム導入では、README、インストール手順、社内テンプレートに書かれた stable バージョンとの差分を確認しておくと安全です。派手な機能更新がない日でも、配布基準の版が変わると実際の挙動基準は更新されます。

まとめ

v0.37.1 は目立つ新機能というより、Gemini CLI の stable 系列の現行基準を示す更新です。安定版を採用している運用では見落としにくくしておきたいリリースです。
