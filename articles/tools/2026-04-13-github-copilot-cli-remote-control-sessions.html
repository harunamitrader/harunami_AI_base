GitHub Copilot CLI: remote control sessions が public preview

Copilot CLI の実行セッションを GitHub の web や mobile から監視・操作できる remote control が public preview になりました。

リリース日 2026-04-13
機能 Remote control CLI sessions
ツール GitHub Copilot

主な変更点

GitHub Changelog では、2026-04-13 付で Copilot CLI の remote control sessions が public preview として公開されています。copilot --remote を使うと、ローカル CLI の進行状況を GitHub の web や GitHub Mobile から確認し、追加指示も送れるようになります。

影響や使いどころ

長めの実装や調査を CLI に任せたまま、別端末から進行を確認したり、途中で方針を追加したりできます。CLI にはリンクや QR code が表示され、web / mobile 側と CLI 側の操作が同期される設計です。

補足

remote session では、実装前の plan review、plan / interactive / autopilot mode の切り替え、権限リクエストの承認・拒否、ask_user への応答などを扱えます。Business / Enterprise では、管理者による remote control と CLI policy の有効化が必要です。

まとめ

今回の更新は、Copilot CLI を純粋なローカル作業から、web / mobile で監視できる分散型の開発セッションへ広げるものです。外出中の確認や長時間タスクの監督には便利ですが、組織利用では policy と権限設定を先に確認する必要があります。
