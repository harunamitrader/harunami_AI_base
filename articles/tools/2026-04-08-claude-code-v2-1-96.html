Claude Code v2.1.96: Amazon Bedrock 対応と AWS 認証の修正

Bedrock サポートの強化と、v2.1.94 で発生していた認証情報の欠落バグを修正。インタラクティブな学習コマンド /powerup も追加されました。

リリース日 2026-04-08
バージョン v2.1.96
ツール名 Claude Code

主な変更点と最新情報

今回の v2.1.96 リリースは、直前のメジャーアップデート v2.1.94 で導入された Amazon Bedrock サポートに関する重要な修正を含んでいます。特に、特定の環境変数を使用している場合に発生していた 403 認証エラーが解消され、AWS 上での利用がよりスムーズになりました。

注目の新機能: /powerup

新しく導入された /powerup コマンドは、Claude Code の高度な機能を学ぶためのインタラクティブなレッスンを提供します。アニメーション付きのデモを通じて、複雑なコマンドやツール呼び出しのベストプラクティスを効率的に習得できるようになりました。

MCP の改善と大容量サポート

Model Context Protocol (MCP) のサポートも強化され、最大 50万文字の結果を保持できるようになりました。また、Slack #channel ヘッダーの改善など、コラボレーションツールとの連携時の視認性も向上しています。

まとめ

v2.1.96 は、エンタープライズ利用を支えるインフラ対応の修正と、ユーザーの習熟を助ける新機能がバランスよく配合されたアップデートです。Bedrock ユーザーは速やかな更新を推奨します。
