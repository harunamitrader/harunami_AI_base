724-office: 自己進化し続ける pure Python AI エージェント

2026-03-26 wangziqi06/724-office

GitHub Trending で注目を集めている 724-office は、24時間365日働き続けることを目指した「自己進化型」AI エージェント・フレームワークです。Pure Python で実装されており、外部サービスに依存せず、独自のメモリシステムとツールキットを備えています。

主な特徴

  • 自己修復能力: 動作中に発生したエラーをエージェント自身が分析し、コードや設定を修正しながらタスクを継続します。
  • 3層メモリシステム: 短期記憶、長期記憶、知識ベースを組み合わせ、文脈を失わずに長期にわたるタスクを完遂します。
  • MCP (Model Context Protocol) サポート: 既存の MCP ツール群を即座に取り込み、ブラウジングやファイル操作などの能力を拡張可能です。

人間が不在の間も「オフィス」を守り、学習し続けるエージェントの姿は、次世代のオートメーションを感じさせます。
