claude-token-counter: Claude Code のトークン使用量とコストをリアルタイム監視

進行中のセッションの消費トークンと見積コストを秒単位で追跡。コスト意識の高い開発者向け Rust ツール。

対象 repo rubenmendoza1290/claude-token-counter

これは何か

Claude API および Claude Code のトークン使用状況をリアルタイムで監視・可視化し、コストを追跡するための Rust 製 CLI ツールです。複雑な API 使用料を瞬時に計算し、美しいターミナル出力で現在の消費状況と、最新の Claude 3.7 や 4.5 などのモデルに基づいた正確な見積コストを表示します。

何ができるか

  • ライブ監視: live コマンドにより、進行中のセッションのトークン量と見積コストをリアルタイム表示。
  • ヒストリ表示: 過去 7 日間や 30 日間の累計使用量（トークン数、コスト）を一覧表示。
  • ローカルログ解析: Claude Code が出力するローカルの JSONL ログを直接解析するため、API 連携なしでも即座に状況把握が可能。
  • 最新価格対応: Claude 3.5/3.7/4.5 Sonnet や Haiku など、最新の価格体系に基づいてコストを算出。

目立つポイント

最大の魅力は、Claude Code の実行ログを直接読み取ることで実現される「高いリアルタイム性」です。作業中に別ターミナルで起動しておくだけで、現在のターンでいくら消費したかを秒単位で確認できるため、トークン節約の強力な指針となります。Rust 製による軽快な動作も、開発者にとって嬉しいポイントです。

セットアップや使い方の流れ

1. インストール: cargo install を使用して GitHub リポジトリから直接ビルド・インストールします。

2. ライブ監視の開始: ターミナルで claude-token-counter live を起動します。

3. 作業の継続: 別ターミナルで通常通り Claude Code を使用します。

4. ステータス確認: status コマンドで、現在の請求期間におけるクォータ残量や使用量を素早く確認できます。

どんな人向けか

Claude Code を日常的に活用し、コーディングを自動化している開発者に最適です。また、Anthropic API のコストを詳細かつ正確に把握・管理したいエンジニアや、クォータ制限をリアルタイムで監視したいシステム管理者にも強く推奨されます。

注意点

現在は 0.1.x のベータ版であり、今後のアップデートで仕様が変更される可能性があります。ライブモードは主に Claude Code の標準的なログ保存環境を前提としています。

まとめ

claude-token-counter は、AI 投資における「見えないコスト」を透明化する強力なユーティリティです。特に Claude Code 利用者にとっては、作業効率とコスト意識を両立させるための唯一無二のツールとなるでしょう。
