mobile-use: 自然言語でスマホを操る次世代エージェント

2026-03-26 minitap-ai/mobile-use

minitap-ai による mobile-use は、Anthropic の computer use モデルのような体験を iOS/Android のモバイルデバイスで実現するためのオープンソース・プロジェクトです。「メッセージを送って」「このアプリの情報をまとめて」といった自然言語の指示を、モバイル画面の操作に変換します。

主な機能

  • マルチプラットフォーム対応: Android と iOS の両方をサポートし、エミュレータまたは実機を通じて自動操作を実現します。
  • 画面解析とアクション: 視覚的なフィードバックを基に、ボタンのクリック、スワイプ、文字入力、通知の読み取りを行います。
  • 情報の構造化出力: スクレイピングしたデータを解析し、使いやすい JSON フォーマットに変換して他のツールに渡すことができます。

モバイルアプリの自動化テストや、個人のパーソナル・アシスタントとしての活用に大きな期待がかかります。
