これは何か

openscreen は、製品デモや操作説明動画を、録画から簡単な編集まで 1 つで進められるオープンソースアプリです。README では Screen Studio の無料代替として位置づけられており、ウォーターマークなし、商用利用可、サブスク不要が打ち出されています。

何ができるか

全画面や特定ウィンドウの録画、マイクとシステム音声の収録、ズーム演出、トリミング、再生速度調整、背景差し替え、注釈追加、複数アスペクト比での書き出しまでをカバーしています。単なる録画ツールではなく、「そのまま見せられるデモ動画」に近づける編集機能が一通り揃っています。

目立つポイント

全部入りの動画編集ソフトを目指すのではなく、README でも「90% of use cases」に絞る方針が明確です。TypeScript、React、Electron、PixiJS 構成で UI と演出を作っており、プロダクト紹介動画の用途に特化している点が分かりやすいです。

セットアップや使い方の流れ

基本は GitHub Releases から OS 向けバイナリを取って起動するだけです。macOS では署名未対応のため Gatekeeper 回避と画面録画権限が必要で、Linux では AppImage 実行権限や PipeWire 確認など、各 OS 向けの初期手順が README にまとまっています。

どんな人向けか

SaaS、開発ツール、社内業務フローの紹介動画を気軽に作りたい個人開発者や小規模チームに向いています。特に「本格的な動画編集ソフトは重いが、見栄えのするデモは欲しい」という人にはちょうどよい立ち位置です。

注意点

README でも beta とされており、成熟度は商用の専用ツールほど高くありません。高度なモーショングラフィックや細かなタイムライン編集が必要なら不足もあるので、まずは短いデモ動画で使い勝手を確かめるのが安全です。

まとめ

openscreen は、録画と最低限の演出編集を 1 つに寄せて、デモ動画作成を軽くする実用的なオープンソースです。プロダクト紹介を日常的に作る人なら、試す価値はかなりあります。
