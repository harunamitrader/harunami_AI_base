これは何か

Prompt Engineering Guide は、prompt engineering を中心に、RAG、AI agents、関連論文、講義、tool をまとめた学習ハブです。README でも guide、papers、lessons、notebooks、resources の集合として位置づけられています。

何ができるか

基礎から techniques、applications、models、risks まで体系的に辿れます。Zero-shot、Few-shot、CoT、ReAct、RAG などの手法だけでなく、lecture、notebook、prompt hub、datasets まで入っているのが便利です。

目立つポイント

単なる README 集ではなく、外部 web 版 promptingguide.ai と連動した教材基盤になっています。学習者向けにも実務者向けにも使いやすく、LLM 周辺の知識整理にかなり向いています。

セットアップや使い方の流れ

普段は web 版を読むのが楽ですが、repo からローカルでも起動できます。README では Node と pnpm を入れた上で pnpm dev でローカル閲覧する手順も案内されています。

どんな人向けか

prompt engineering を体系的に学びたい人、RAG や agent の手法を実装前に整理したい人、社内勉強会や研修の参照先を探している人に向いています。

注意点

内容量が非常に多いので、最初から全部読むより、自分の用途に近い technique や application から入る方が効率的です。また web 版の方が更新が追いやすい場面もあります。

まとめ

Prompt-Engineering-Guide は、prompt / context engineering 周辺を一気に俯瞰したいときの定番 repo です。学習用ブックマークとして持っておく価値があります。
