対象 repoFlowiseAI/Flowise

これは何か

Flowise は、AI agent を視覚的に組み立てるための platform です。README でも Build AI Agents, Visually と打ち出しており、GUI ベースで node をつなぎながら workflow を構成できます。

何ができるか

RAG、chatflow、agentflow を UI 上で組み、self-host したり Cloud で使ったりできます。Node backend、React frontend、third-party node integration 群を monorepo で持っているので、利用者としても拡張者としても扱えます。

目立つポイント

非エンジニア寄りの導入しやすさと、開発者向け self-host の両立が強みです。npm install や Docker Compose ですぐ動き、AWS や GCP など複数のデプロイ先も案内されています。

セットアップや使い方の流れ

最小導入は npm install -g flowise の後に npx flowise start、または Docker / Docker Compose です。開発側は pnpm install、pnpm build、pnpm start でローカル起動できます。

どんな人向けか

RAG や agent workflow をコードだけでなく GUI でも触りたい人、社内デモや PoC を早く形にしたい人、または self-host 가능한 visual AI stack を探している人に向いています。

注意点

大規模 monorepo なので、自前で深くカスタムする場合は依存や build コストもそれなりです。GUI で組みやすい一方、細かい制御は code-first 案件と比較して設計判断が必要です。

まとめ

Flowise は、visual AI workflow の代表格です。agent や RAG の構築を手早く体験したいなら、依然として有力な選択肢です。
