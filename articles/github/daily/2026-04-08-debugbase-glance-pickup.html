これは何か

Anthropic の Claude Code などの AI 開発ツールに、強力なブラウザ操作機能を追加するための Model Context Protocol (MCP) サーバーです。ターミナル上の AI と対話しながら、実際のブラウザを制御して情報の収集、動作確認、スクリーンショットの取得などを行わせることができます。

何ができるか

このツールを導入した AI エージェントは、以下の操作を自律的に代行できるようになります。

  • ブラウザナビゲーション: 指定された URL へのアクセスと情報の読み取り。
  • 要素操作: リンクのクリック、フォームへの入力、ボタン操作。
  • 視覚的確認: 特定の画面範囲やページ全体のスクリーンショット撮影。
  • 自動テストの実行: UI が意図通りに動いているかをブラウザ上で実際に動かして検証。

目立つポイント

AI エージェントにとって「ブラウザが見える」ことは、ドキュメントの調査や UI デバッグにおいて決定的なアドバンテージとなります。glance は軽量かつ TypeScript で実装されており、特に Claude Code との親和性が高く設計されているため、導入コストを低く抑えつつ開発ワークフローを大幅に拡張できる点が魅力です。

セットアップや使い方の流れ

  1. リポジトリをクローンし、必要な依存関係をインストールします（npm install など）。
  2. ビルドを実行後、Claude Code の MCP 設定（claude config）にこのサーバーを追加します。
  3. Claude Code 起動後、「example.com を開いてヘッドラインを確認して」などの指示を送ります。
  4. AI が glance 経由でブラウザを立ち上げ、結果をテキストや画像パスで返します。

どんな人向けか

  • ターミナルから出ることなく Web 調査を完結させたい効率重視のエンジニア。
  • AI エージェントにフロントエンドの UI テストや動作チェックを任せたい開発者。
  • MCP サーバーを活用して自律的なワークフローを構築したい AI ユーザー。

注意点

同じ名前の dashboard アプリケーション（glanceapp/glance）が存在しますが、それとは全く別の「AI 向けブラウザ自動化ツール」であることに注意してください。使用には Node.js 環境が必要です。

まとめ

glance は、AI コーディングエージェントに「実社会（Web）」への窓口を与えるツールです。コードを書くだけでなく、そのコードが Web 上でどう動いているかを AI 自らが見て確認できる環境を作ることで、デバッグ効率は飛躍的に向上するでしょう。
