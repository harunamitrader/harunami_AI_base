これは何か

Alibabaが開発した「Page Agent」は、Webページ内に直接組み込んで自然言語でブラウザ操作を可能にするJavaScriptベースのGUIエージェントです。ブラウザ拡張機能や外部サーバーを介さず、数行のJavaScriptコードをページに追加するだけで動作します。

何ができるか

  • テキストベースのDOM操作により、軽量かつ高速なブラウザ操作を実現します。
  • Qwenなど様々な大規模言語モデル（LLM）と連携し、自然言語での指示を解釈・実行します。
  • SaaS製品へのAIコパイロットの実装、複雑なフォーム入力の自動化、アクセシビリティの向上など、多様な用途に活用できます。
  • Chrome拡張機能版やMCPサーバーも提供されており、複数タブにまたがるタスクや外部からの制御にも対応します。

目立つポイント

  • **簡単な統合**: 外部サービス不要で、JavaScriptコードの追加のみでWebページにAIエージェント機能を組み込めます。
  • **軽量・高速**: スクリーンショットやマルチモーダルLLMに依存せず、DOMを直接解析するため効率的です。
  • **柔軟なLLM対応**: ユーザーがAPIキーを持ち込み、任意のLLMを利用できるため、幅広いAIニーズに対応可能です。

セットアップや使い方の流れ

基本的には、対象のWebページに数行のJavaScriptコードを追加するだけで動作します。詳細な設定や使い方は、公式GitHubリポジトリを参照してください。

どんな人向けか

  • SaaS製品にAIコパイロット機能を導入したい開発者
  • Webアプリケーションのユーザー体験を向上させたい企業
  • ブラウザ操作の自動化やアクセシビリティ改善に関心のあるエンジニア

注意点

記事作成時点では、利用可能なLLMの種類や具体的なセットアップ手順については、公式ドキュメントで詳細を確認する必要があります。

まとめ

Page Agentは、WebページにAIの力を手軽に組み込むための強力なツールです。簡単な統合、軽量な動作、柔軟なLLM対応により、Webアプリケーションの可能性を大きく広げます。
