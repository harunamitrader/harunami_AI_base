これは何か

stagewise は、Web 開発者のためにゼロから構築された、AI コーディングエージェント内蔵の次世代ブラウザです。 既存のブラウザで開発・検証し、エディタで修正するという「往復作業」を解消し、ブラウザそのものがコードを理解・編集・テストする、エンジニアにとっての司令塔となります。

何ができるか

  • コンソール・デバッガへの完全アクセス： 内蔵エージェントがタブのコンソール出力や実行状態を直接読み取り、バグの原因特定や解決策の提案をリアルタイムに行います。
  • ライブ編集・プレビュー： 表示されているウェブサイトのコンポーネントやスタイルをその場で解析。一時的なテスト変更から、接続したコードベースへの永続的な反映まで一気通貫で行えます。
  • インテリジェント・リバースエンジニアリング： 既存のサイトからコンポーネント構造、スタイル、カラーパレットなどを瞬時に抽出し、自身の開発に即座に活用可能です。
  • IDE との深い統合： VS Code 拡張機能を通じて、ブラウザ上での変更をお気に入りのエディタへ即座に同期できます。
  • BYOK (Bring Your Own Key) サポート： Anthropic, OpenAI など主要な AI プロバイダーを自身の API キーで利用できます。

目立つポイント

最大の特徴は、**「ブラウザが開発プロセスにおける『受動的な表示機』から『能動的な開発パートナー』へと進化した点」** です。 開発者がデバッグ中にコンソールを見て悩み、コードを書き直すという一連の思考フローを、AI がブラウザ内部のコンテキストを共有した状態で支援。 これにより、ブラウザとエディタの往復による文脈の切り替え（コンテキストスイッチ）を極限まで減らし、開発の密度を飛躍的に高めています。

セットアップや使い方の流れ

stagewise.io からアプリをダウンロードしてインストールし、API キーを登録します。 調査したいサイトや開発中の URL を開き、サイドバーの AI エージェントに「このバグを直して」「このデザインを抽出して」と指示を出すだけです。 プロジェクトフォルダを接続すれば、修正をそのまま自身のコードベースに書き込むことも可能になります。

どんな人向けか

フロントエンド開発のイテレーションを爆速化したいエンジニアや、既存の Web サイトのデザインシステムを効率的に分析・模倣したいデザイナー兼エンジニアに最適です。 Cursor 等の AI エディタを使いこなしている方にとって、さらに「実行環境側」からの強力な支援を得られるこの環境は、代替不可能な道具になるでしょう。

注意点

オープンソース（AGPLv3）で提供されていますが、コードベースへの自動変更などは強力であるため、重要なプロジェクトへの適用時には人間によるレビューを併用することが推奨されます。 また、BYOK 形式のため、利用するモデルや頻度に応じたトークンコストをユーザー自身で管理する必要があります。

まとめ

stagewise は、Web 開発の「現場」であるブラウザのあり方を根本から再定義しようとしています。 AI が DOM や実行コンテキストを真に理解し、ブラウザ内部から開発を加速させるこの体験は、これからの Web 開発における新たなスタンダードを示唆しています。
