これは何か

Open Interpreter は、LLM にローカルでコードを実行させる natural language interface です。ChatGPT の Code Interpreter を手元の PC に持ち込む発想で、Python、JavaScript、Shell などを自然文経由で動かせます。

何ができるか

画像や PDF の生成編集、ブラウザ操作、データ分析、ファイル加工など、一般的な PC 作業を会話で進められます。CLI の interpreter だけでなく Python API もあり、streaming や profile、OpenAI 互換 local model 接続もサポートされています。

目立つポイント

強みは hosted sandbox ではなくローカル環境で動く点です。時間制限や upload 制限がきつい hosted code interpreter と違い、ネット接続や任意 package 利用まで含めてかなり自由度があります。

セットアップや使い方の流れ

導入は pip install git+https://github.com/OpenInterpreter/open-interpreter.git の後に interpreter を実行するだけです。Python からは from interpreter import interpreter で埋め込み、LM Studio などの OpenAI 互換 endpoint へも向けられます。

どんな人向けか

AI にローカル automation をやらせたい人、code execution を伴う assistant を自前環境で扱いたい人、あるいは「PC 操作 agent」の OSS 実装を学びたい人に向いています。

注意点

ローカル実行ゆえに権限は強く、README でも user confirmation の重要性が強調されています。安全性を重視するなら sandbox や safe mode、隔離環境での利用前提で考えた方がよいです。

まとめ

open-interpreter は、ローカル code execution agent の代表格です。自然言語で PC を動かす体験を OSS で見たいなら、今でも押さえておく価値があります。
