これは何か

guppylm は、900万パラメータ（約9M）という極めて軽量な言語モデルです。水槽の中に住む魚「グッピー」になりきって会話をすることを目的に設計されています。

このプロジェクトの真の価値は、LLM（大規模言語モデル）の構築と訓練が「魔法」ではないことを証明することにあります。データ生成からトークナイザーの準備、モデルアーキテクチャ、訓練ループ、そして推論まで、すべての工程がシンプルに提示されており、LLMの基礎を学ぶための最高の実践教材となっています。

何ができるか

  • 魚との対話: 水、食べ物、光、水槽の中の生活について、小文字の短い文章で喋るグッピーとチャットできます。
  • ブラウザ実行: WebAssembly と ONNX を使用し、サーバー不要でブラウザ上で直接モデルを動かせます（量子化されたモデルは約10MB）。
  • 5分間の訓練: Google Colab を使用して、スクラッチから自分だけのモデルを約5分で訓練できます。
  • ローカル実行: pip を介してインストールし、ターミナルから直接グッピーと対話可能です。

目立つポイント

徹底して「シンプルさ」にこだわっている点が特徴です。現代の主流モデルで使われる GQA（Grouped Query Attention）や RoPE（Rotary Positional Embedding）などの複雑な手法をあえて排除し、Vanilla Transformer アーキテクチャを採用しています。

また、9Mパラメータというサイズにより、高価な GPU クラスタを必要とせず、ノート PC やブラウザで軽快に動作します。性格は 6万件の合成会話データによって「焼き付けられて」おり、システムプロンプトなしでも安定したキャラクター性を維持します。

セットアップや使い方の流れ

最も手軽なのは、GitHub ページに用意されているブラウザデモでの試遊です。開発者として深く触りたい場合は、以下の手順でローカル実行が可能です。 

pip install torch tokenizers
python -m guppylm chat

訓練フローを自作したい場合は、付属の train_guppylm.ipynb を Google Colab で開くことで、データ生成から訓練までの全工程を体験できます。

どんな人向けか

  • LLM がどのように動いているか、中身をブラックボックスなしで理解したい学習者。
  • 特定のキャラクター（この場合は魚）を持った超軽量モデルの構築手法に興味があるエンジニア。
  • ブラウザやエッジデバイス上で AI を動作させる方法（WASM/ONNX）のリファレンスを探している開発者。

注意点

教育用かつ極小サイズであるため、128 トークンの短いコンテキスト制限があります。数ターン以上の長い会話は記憶が曖昧になりやすく、また、魚の理解を超える高度な話題（経済、政治、科学など）には対応していません。

まとめ

guppylm は、巨大化の一途をたどる LLM の世界において、あえて「最小構成」を見せることでその仕組みを民主化する素晴らしいプロジェクトです。AI エージェントやツール開発に携わるなら、一度その内部コードをのぞいて、LLM の本質に触れてみることをお勧めします。
