これは何か

Penpotは、デザインと実装の境界をなくすために開発された、オープンソースのデザイン・プロトタイピングプラットフォームです。Figmaの強力な競合でありながら、SVGやCSSといったWeb標準技術を基盤としている点が最大の特徴です。

何ができるか

ベクターベースのUIデザイン、リアルタイムの共同編集、プロトタイプの作成、そして開発者への正確なスペック伝達が可能です。デザイナーが作成したレイアウトがそのまま CSS Flexbox や CSS Grid の構造を反映しているため、実装時の乖離を最小限に抑えることができます。

目立つポイント

完全にオープンソースであり、Dockerを使用して自身のサーバーでホスト（セルフホスト）できるため、プライバシーやデータの制御を重視するチームにとって理想的です。また、最近のアップデートではAIによるオートレイアウト補助機能なども追加され、制作効率が飛躍的に向上しています。

セットアップや使い方の流れ

最も手軽なのは公式のクラウド版（無料プランあり）を利用することです。チームで独自のインフラを構築したい場合は、リポジトリ内の `docker-compose.yaml` を使用して、数分で自分たちの Penpot インスタンスを立ち上げることができます。Figma や Sketch からのインポート機能も備えています。

どんな人向けか

プロプライエタリなツール（Figma等）への依存を減らしたいデザインチームや、開発者とのコミュニケーションを円滑にしたいプロダクトマネージャー、そして自由なカスタマイズ性を求めるオープンソース愛好家に強くおすすめします。

注意点

急速に進化していますが、プラグインのエコシステムなどはFigmaと比較するとまだ発展途上です。また、セルフホストする場合は、データのバックアップやサーバーのメンテナンスを自前で行う必要があります。

まとめ

Penpotは「デザインの民主化」を象徴するプロジェクトです。オープンなWeb標準技術に根ざしたその設計思想は、デザイナーとデベロッパーが対等に協力し合える新しいワークフローを提示しています。
