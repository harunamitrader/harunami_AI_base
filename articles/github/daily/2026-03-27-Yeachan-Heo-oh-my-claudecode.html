これは何か

Anthropic の自律型 CLI ツール「Claude Code」の機能を大幅に拡張し、複数の AI エージェントを協調させて動かすためのオーケストレーションツールです。単体のエージェントでは困難だった長時間のタスクや、多角的な検証が必要な開発ワークフローを「チーム」として自動実行することを目指しています。

何ができるか

「Team Mode」による並列タスク実行と自動検証が最大の特徴です。一つの指示から計画策定（Team-Plan）、実装（Team-Exec）、検証（Team-Verify）、修正（Team-Fix）という一連の開発サイクルを自動で回します。また、MCP（Model Context Protocol）の統合や、過去の開発パターンを「カスタムスキル」として自動学習し、次回以降に再利用する機能も備えています。

目立つポイント

32種類もの専門エージェント（アーキテクチャ、リサーチ、テスト等）を内蔵しており、タスクの難易度に応じて Opus や Haiku などのモデルを賢く使い分けます。これにより、推論の質を保ちつつトークンコストを 30-50% 削減できるとされています。また、HUD（ステータスライン）で現在のエージェントの動きがリアルタイムに可視化される点も強力です。

セットアップや使い方の流れ

コマンドラインで omc を実行してセットアップを開始します。/team コマンドでマルチエージェントモードを起動し、自然言語でタスクを依頼するだけで、チームが自律的に動き出します。特定のエージェントに直接質問を投げたり、学習したスキルを管理したりすることも可能です。

どんな人向けか

Claude Code を単なる「便利なチャットインターフェース」としてだけでなく、数十分から数時間に及ぶ本格的な開発プロジェクトを自律的に進める「AI 開発チーム」として運用したいエンジニアに最適です。

注意点

現時点では tmux の環境が推奨されており、複数のエージェントがバックグラウンドで並行動作するため、API のレート制限管理には注意が必要です（ツール側で自動待機機能が備わっています）。

まとめ

oh-my-claudecode は、Claude Code を「最強の自律開発チーム」へと進化させる強力なアドオンです。モデルの使い分けによるコスト最適化と、スキル学習による自己進化という、次世代の AI エージェント開発に必要な要素が凝縮されています。
