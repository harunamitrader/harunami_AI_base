これは何か

Translumo は、ゲームやデスクトップアプリの画面に表示された文字を OCR で認識し、翻訳結果をその場にオーバーレイ表示する Windows 向けツールです。README では JRPG やビジュアルノベルのように、画面内テキストを読みながら遊ぶ用途を主眼に置いています。

何ができるか

認識対象はフルスクリーン、ウィンドウ、選択領域に対応し、OCR は PaddleOCR と Windows OCR を切り替えられます。翻訳先も DeepL、Google、Gemini、ChatGPT、Ollama など複数のバックエンドを選べるため、精度重視・速度重視・ローカル運用など目的に応じて構成を変えやすいのが強みです。

目立つポイント

単なるスクリーン翻訳に留まらず、履歴表示、コンテキスト蓄積、ログ保存、TTS、ルビや字幕向けの表示調整までまとめて扱えます。README のスクリーンショットを見ると、縦長のゲーム文面や短い会話文を連続で処理する前提で設計されており、一般的な OCR デモより現場寄りです。

セットアップや使い方の流れ

基本は Releases から Windows 向けビルドを入手し、必要に応じて翻訳 API キーやローカルモデル接続先を設定します。起動後は翻訳したい画面領域を指定し、OCR エンジンと翻訳プロバイダを選んで、オーバーレイ表示を有効化する流れです。ローカル運用を重視するなら Ollama、精度を優先するならクラウド翻訳 API を組み合わせる使い方が自然です。

どんな人向けか

海外ゲームや未翻訳アプリを読みながら使いたい個人ユーザーに向いています。特に、画面キャプチャベースの翻訳をワンボタンで済ませたい人や、機械翻訳を補助的に使ってストーリーを追いたい人には分かりやすい選択肢です。

注意点

性能は OCR 精度と表示フォント、画面解像度にかなり左右されます。また、利用する翻訳プロバイダによっては API キー、ネット接続、料金、レート制限が関わります。さらに、ゲームによってはオーバーレイや画面取得を嫌うケースがあるので、実運用前に対象タイトル側の挙動は確認した方が安全です。

まとめ

Translumo は、画面翻訳を「OCR の実験」ではなく「実際に読みながら使う補助ツール」として仕上げているのが良い点です。OCR と翻訳バックエンドの選択肢が広く、クラウド依存にもローカル志向にも振れるので、ゲーム翻訳用途の実用ツールとして見やすいリポジトリです。
