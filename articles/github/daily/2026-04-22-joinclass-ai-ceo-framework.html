AI-CEO Framework

Claude Codeで「1人企業」を完全自動化。15体のAIエージェントと承認パイプラインを備えた、実戦投入済みの経営フレームワーク。

記事作成日 2026-04-22
種別 Pickup
対象 repo JOINCLASS/ai-ceo-framework
Star数 17

これは何か

Claude Code を「経営陣」として機能させるための、統合エージェント・フレームワークです。15種類の専門エージェント（CTO, CMO, CFOなど）と、実務を遂行するためのスキル群、そしてAIの暴走を防ぐ承認システムがセットになっており、1人で会社を運営するための「AI OS」として機能します。

何ができるか

ユーザーが「CEO」として「今日のステータスを教えて」「この施策のブログ記事を書いて」と自然言語で伝えるだけで、適切な部門のエージェントがタスクを引き継ぎ、実行します。メール送信やデプロイなどの対外的なアクションは必ず「承認待ちリスト」に入り、CEOの許可を得てから実行されるため、安全な自動化が可能です。

目立つポイント

単なるプロンプト集ではなく、1年以上の実ビジネス運用を経て磨き上げられた「実戦仕様」である点が最大の特徴です。特に「仮説検証ゲート」機能は、新しいアイデアに対してデータに基づいた厳格な評価を行い、無駄な投資を防ぐ仕組みが組み込まれています。また、コンテキスト消費を最小限に抑える設計により、大規模なプロジェクトでも安定して動作します。

セットアップや使い方の流れ

既存のリポジトリにフレームワークのファイルをコピーし、setup.sh を実行するだけで導入できます。社名やビジネスドメイン、使用ツールなどの質問に答えることで、Claude Code が即座にあなたの会社の「バーチャル経営陣」としてセットアップされます。あとは /ai-ceo:morning などのコマンドで業務を開始できます。

どんな人向けか

複数のプロジェクトやサイドビジネスを1人で回している個人開発者や、リソースが限られたスタートアップの創業チームに最適です。Claude Code を単なるコーディングアシスタントとしてだけでなく、マーケティング、財務、法務までをサポートする「万能の部下」に変えたい全てのユーザーに向けた野心的なツールです。

注意点

エージェントが相互に連携し、サブタスクを自律的にこなす機能をフル活用するには、Claude Code Max プランまたは Anthropic API キーによる従量課金が必要です。また、現時点でのテンプレートは英語がベースとなっているため、日本国内での運用にはエージェント定義のカスタマイズが必要になる場合があります。

まとめ

「AIエージェントによる組織運営」という未来を、今すぐ手元で実現できる極めて実用的なフレームワークです。AIを単なる道具から「自律的な組織」へと進化させたい方にとって、これ以上の出発点はないでしょう。
