これは何か

Anthropic の「Claude Code」のような高性能な AI エージェント（コーディング・ハーネス）を、自分たちの手でゼロから構築する方法を学ぶための教育用プログラムです。全 11 セッションで構成されており、AI を単なる「チャットボット」としてではなく、「道具を使いこなし自律的に働くエージェント」として一から再構築するプロセスを体系的に示しています。GitHub で驚異的な 4.2 万ものスターを獲得し、大きな話題となっています。

何ができるか

「エージェント・ループ」の基礎から始まり、ツール・エンジニアリング（AI に外部ツールを与える方法）、さらには複数のエージェントが協力して一つのタスクを完遂する「自律チーム」の構築までを、Python の具体的なコード（MVP 実装）を交えて学習できます。日本語、英語、中国語の三か国語に対応しており、グローバルな知見を得ることが可能です。このプログラムを通じて、独自のエージェント・アプリケーションを開発するための土台を築くことができます。

目立つポイント

「モデル（脳）は運転手であり、コード（枠組み）は車両（ハーネス）である」という独自の設計哲学（Manifesto）を提唱している点が非常にユニークです。単なる技術解説にとどまらず、AI エージェントをどのように設計・運用すべきかという高い視点からの示唆が含まれています。また、段階的に難易度が上がるように設計されているため、読み進めるうちに知識が確実に定着していく点も優れています。

セットアップや使い方の流れ

基本的には、このリポジトリを `git clone` して各セッションのフォルダにある Jupyter ノートブックや Python スクリプトを順次実行していきます。それぞれのセッションには明確な学習目標が設定されており、実際に動くコードを手元で確認しながら、AI エージェントの内部構造を深く理解していくことができます。

どんな人向けか

「既存の AI ツールを使うだけでは物足りない」「自分たちで独自の AI エージェント・プラットフォームを構築したい」と考えているエンジニアや研究者に最適です。また、LLM をどのようにアプリケーションに組み込み、実用的な「ツール」として昇華させるかを学びたいプロフェッショナルにとっても、これ以上ない教材となります。

注意点

このリポジトリはあくまで「教育用」の学習リソースであり、特定の業務（例えば、そのままウェブサイトを全自動で作る等）を即座に遂行するための完結した完成品ツールではありません。実用的なエージェントを構築するためには、ここで得た知識をベースに、自ら開発を行う必要があります。

まとめ

「learn-claude-code」は、AI エージェント開発という新しい分野に飛び込むための最高級の教科書です。Claude Code の背後にある「魔法」の正体を突き止め、自分だけのエージェント・チームを作り上げたい開発者にとって、この 11 セッションからなる旅は非常に価値のある体験になるでしょう。
