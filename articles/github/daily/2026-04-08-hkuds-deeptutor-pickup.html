これは何か

DeepTutorは、香港大学（HKU）のData Science（HKUDS）チームが開発した、学習者一人ひとりに最適化された支援を提供するオープンソースのAI学習アシスタントです。単に質問に回答するだけでなく、自律的に学習をサポートする「エージェント・ネイティブ」な設計が最大の特徴です。

何ができるか

主な機能は以下の5つの「ケイパビリティ（能力）」として提供されています。

  • 対話型支援: 質疑応答とパーソナライズされたフィードバック。
  • Deep Solve: 複雑な問題を段階的に解説・解決するマルチステップ推論。
  • Deep Question: 学習者の理解度を評価するための適切な問題生成。
  • Deep Research: 学術論文やWeb情報を自律的に検索・集約。
  • Math Animator: 数学的概念を Manim を用いたアニメーションで可視化。

目立つポイント

DeepTutor 2.0 で刷新されたアーキテクチャは、GeoGebra、Python実行、Web検索などの「専門ツール」と、それらを組み合わせてパイプライン実行する「ケイパビリティ」を統合しています。また、永続的なメモリシステムを備えており、セッションを跨いで学習者の進捗や個々のデバッグ履歴、好みの学習スタイルを「記憶」し続けることができます。

セットアップや使い方の流れ

ユーザーは、Next.js で構築された公式の Web ワークスペースからすぐに利用を開始できます。さらに、Telegram、Discord、Slack、WeChat などの主要なコミュニケーションツール向けの Bot も公開されています。開発者は WebSocket API や Python SDK を使用して、自身の教育アプリに DeepTutor の推論エンジンを組み込むことも可能です。

どんな人向けか

個別指導を求めている学生、高度な学術調査を効率化したい研究者に最適です。また、エージェント・アーキテクチャの実践例を探している開発者や、AIを教育現場に導入したい教育関係者にとっても非常に参考になるリポジトリです。

注意点

Manim を用いた数学アニメーションの生成や高度な推論などは計算コストが高いため、利用する LLM バックエンド（GPT-4o や Claude 3.5 等）の利用制限に注意が必要です。

まとめ

DeepTutor は、AI が「答えを出力するだけの機械」から、学習者に並走し理解を助ける「賢いチューター」へと進化したことを示す好例です。モジュール化されたアーキテクチャにより、教育以外の専門領域への応用も期待できる、ポテンシャルの高いプロジェクトです。
