これは何か

RuView は、市販の WiFi 信号を利用して、「カメラを一切使わずに」物体の存在、人体、姿勢、さらにはバイタルインジケータを推定・監視するためのオープンソースフレームワークです。 WiFi DensePose 技術をベースにしており、電波の反射や散乱を AI で解析することで、壁を隔てた向こう側の動きさえもリアルタイムで可視化します。

何ができるか

  • リアルタイム人体姿勢推定： WiFi 信号の位相変化から、人間の関節位置や動きを 3D で再現。
  • 非接触バイタルモニタリング： 対象に何も装着させることなく、心拍や呼吸、睡眠の質を追跡。
  • 存在・転倒検知： 暗闇や煙の中でも、プライバシーを侵害することなく高齢者の転倒や不審な侵入を検知。
  • マルチモーダル統合： ウェアラブルデバイスや他のセンサーと組み合わせた高度なヘルスケア・セキュリティ基盤として機能。

目立つポイント

最大の注目点は、**「一切のビデオ・ピクセル（映像）を使用しないことによるプライバシー保護の両立」**です。 カメラを設置できない寝室や浴室、介護施設などでも、利用者のプライバシーを守りながら、AI による高度な見守りと分析を実現。市販の WiFi デバイスがあれば導入できるという汎用性の高さも魅力です。

セットアップや使い方の流れ

対応する WiFi チップセット（ESP32 や特定のルーター）からチャネル状態情報 (CSI) を収集する環境を構築します。 RuView の Python バックエンドを起動し、収集した CSI データを流し込むことで、AI モデルが即座に姿勢推定のレンダリングを開始。Web ベースのダッシュボードで結果を確認できます。

どんな人向けか

プライバシーを最優先しながらスマートホーム、介護監視、遠隔医療を実現したい開発者や研究者に最適です。 また、映像を使わずに物理空間を「デジタルツイン」へと変換したい、工場や公共施設の DX 担当者にとっても非常に価値のある技術です。

注意点

WiFi 環境の電波干渉や遮蔽物の材質によっては、推定精度に影響が出ることがあります。現時点では特定のチップセットへの依存度があるため、ハードウェア要件の確認が必須です。

まとめ

RuView は、私たちの身の回りに満ちている WiFi 電波を、新たな「感覚器」へと変貌させます。 映像を介さない見守りは、次世代のパーソナル・セキュリティとヘルスケアの標準となる可能性を秘めており、今後のマルチモーダル AI との融合が強く期待されます。
