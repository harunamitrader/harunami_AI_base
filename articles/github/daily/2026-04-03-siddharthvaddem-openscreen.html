これは何か

openscreen は、Screen Studio のようなプロダクトデモ動画を無料で作れる Electron ベースの録画アプリです。README では有料サブスクやウォーターマークなしで使える代替として位置づけられており、個人利用だけでなく商用利用も前提にしています。

何ができるか

全画面または特定ウィンドウの録画、マイクとシステム音声の収録、自動ズームや手動ズーム、動画のトリミング、区間ごとの速度変更、背景差し替え、注釈追加、複数アスペクト比での書き出しまでを一通りカバーしています。単純なスクリーンレコーダーではなく、デモ動画編集まで一体で扱えるのが強みです。

目立つポイント

無料であることだけでなく、README が「全部入りではなく、よく使う基本機能を押さえる」設計を明言している点が実務向きです。TypeScript / React / Electron / PixiJS 構成で開発されており、見た目の演出を伴う録画アプリとしては構成もわかりやすいです。

セットアップや使い方の流れ

使い始め方は単純で、GitHub Releases から OS 向けインストーラを取得して起動します。macOS では Gatekeeper 回避や画面録画権限の付与、Linux では AppImage 実行権限や PipeWire の有無など、プラットフォームごとの注意点が README に整理されています。

どんな人向けか

プロダクト紹介、チュートリアル、社内デモを素早く作りたいが、まずは有料ツールに縛られたくない人に向いています。特に SaaS や開発ツールの紹介動画を頻繁に作る個人開発者や小規模チームと相性が良さそうです。

注意点

README でも beta であることが明記されており、Screen Studio の完全代替ではありません。高度な演出や成熟した安定性を最初から求めるなら不足もあり得るので、まずは短いデモ動画で試すのが安全です。

まとめ

openscreen は、「無料でも見栄えの良い録画をしたい」という需要に対して、必要十分な編集機能を実装した現実的なオープンソースです。2026-04-03 の GitHub Trending 1 位という伸び方も含め、今の勢いはかなり強いです。
