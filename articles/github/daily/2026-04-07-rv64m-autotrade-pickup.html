autotrade

AI (Claude/Gemini) にトレード戦略を自律的に考案・改善させる、最先端の自動売買検証フレームワーク。

対象 repo rv64m/autotrade

これは何か

autotrade は、Claude Code や Gemini、OpenAI (Codex) などの AI エージェントを活用して、仮想通貨（特に Binance 等）のトレード戦略を自律的に生成・改善・実行し続けるためのフレームワークです。 人間がロジックを書くのではなく、AI に「利益を最大化する戦略を考えて実行しろ」とまるごと委ねるアプローチをとっています。

何ができるか

  • AI による戦略の自律改善: バックテストの結果（損失や利益、ドローダウン）を AI が読み取り、それを改善するためのコード修正を AI 自らが行います。
  • 高速バックテスト連携: Python スクリプト (train.py) を通じて、AI が生成した戦略を即座にヒストリカルデータで検証します。
  • マルチ AI 対応: 最新の強みを持つ AI ツール（Claude Code, Gemini CLI, Codex 等）をエンジンとして切り替えて利用可能です。
  • 取引所データの自動取得: CCXT ライブラリを統合しており、Binance 等から最新の市場データをシームレスに取得します。

目立つポイント

最大のポイントは「ループの自動化」です。これまでの自動売買は、人間が考えたロジックをプログラムにするものでしたが、このプロジェクトでは「戦略の考案 → 行動（コード生成）→ 結果（バックテスト）→ フィードバック（AIによる内省・修正）」というサイクルを自律的に回す仕組みを提供しています。

セットアップや使い方の流れ

  1. Python 3.10 以上の環境を用意し、pip install -r requirements.txt で依存関係をインストール。
  2. Binance 等の API キーを設定（バックテストのみならヒストリカルデータの取得に利用）。
  3. Claude Code や Gemini CLI がパスに通っている環境で python train.py を実行。
  4. AI が複数の戦略案を生成し、その中で最も優秀なものを自律的に選定・洗練させていきます。

どんな人向けか

  • 自律型 AI エージェント（AI Agents）による自動投資の可能性を探っている開発者。
  • アルゴリズムトレードのロジック構築を AI に外注し、スピード感を持って検証・改善したい人。
  • 複数の AI モデルを組み合わせて、複雑な金融データの解析を行いたい人。

注意点

AI が生成するコードをそのまま実行するため、サンドボックス環境での運用が推奨されます。また、仮想通貨取引には常に高いリスクが伴うため、実運用前に十分なバックテストと資金管理の徹底が必要です。

まとめ

autotrade は、単なる「自動売買ツール」ではなく、AI が投資というフィールドで「試行錯誤して成長する」ための基盤です。Claude Code などの進化したコーディングエージェントの力を、金融取引という最もシビアな領域に適用した意欲的なプロジェクトと言えます。
