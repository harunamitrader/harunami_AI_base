概要

ChatDev は、2026-04-02 時点で GitHub Trending 13 位に入っていたマルチエージェント基盤です。README では ChatDev 2.0 を DevAll と位置づけ、従来の「仮想ソフトウェア会社」型から、より汎用的なオーケストレーション基盤へ拡張したと説明しています。

できること

中心となるのは、コードを書かずにエージェント、ワークフロー、タスクを定義して動かせる点です。README では、データ可視化、3D 生成、深い調査のような複雑なシナリオまで、設定ベースで素早く構築できるとしています。

一方で旧来の ChatDev 1.0 系は、CEO、CTO、Programmer などの役割を持つエージェント群で、設計、実装、テスト、文書化までを自動化する構成として残されています。用途に応じて発想がかなり違う二系統を同一リポジトリで追えるのが特徴です。

導入と向いている人

リポジトリ直下には frontend、server、workflow、yaml_template、yaml_instance などが並んでおり、UI と実行基盤、テンプレートを分けた構成になっています。ゼロコードで試したい人にも、内部構造を追って独自運用に寄せたい人にも入口があります。

複数エージェントを業務フローに組み込みたいチームには相性がよい一方、役割設計やワークフロー設計が曖昧だと期待通りに動かしにくいはずです。便利な土台ですが、運用設計まで自動で決まるわけではありません。

まとめ

ChatDev は「AI エージェントを何体動かすか」よりも「どう連携させるか」に重心を置いたリポジトリです。マルチエージェント運用を実験段階から実務寄りに進めたい人ほど、見る価値があります。
