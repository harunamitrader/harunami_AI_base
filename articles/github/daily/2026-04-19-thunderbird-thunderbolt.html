これは何か

Thunderbolt は、Mozilla Thunderbird チームによって開発されている、オープンソースかつクロスプラットフォームの AI クライアントアプリケーションです。特定の AI ベンダーにロックインされることなく、自分自身でモデルを選び、データを管理し、プライバシーを守りながら高度な AI 機能を利用することを目指しています。

何ができるか

Web、iOS、Android、Windows、macOS、Linux のすべてに対応。Ollama や llama.cpp を介したローカル AI の実行はもちろん、OpenAI 互換のあらゆる API 経由で外部モデルも統合可能です。チャット機能にとどまらず、自律的なスキル（Claude Code 互換のスキルなど）を実行するエージェント的な側面も備えており、オンプレミスでのデプロイも強力にサポートしています。

目立つポイント

信頼の Mozilla ブランドのもと、「AI You Control（あなたが支配する AI）」という哲学が貫かれている点です。現在は特にエンタープライズ向けのオンプレミス導入を視野に入れた開発が進んでおり、データの外部流出を許さないセキュアな AI 利用環境を、美しい UI とクロスプラットフォーム対応で提供します。

セットアップや使い方の流れ

現在は開発の初期段階（アクティブ開発中）ですが、Docker を用いたバックエンドのデプロイや、各プラットフォーム向けのビルドが提供されています。起動後は Ollama などのローカル推論エンジンを連携させるか、API キーを設定することで即座に利用を開始できます。

どんな人向けか

「AI は使いたいが、社外やクラウドにデータを送りたくない」と考える企業の管理職やエンジニア、または個人のプライバシーを徹底的に守りたいテック・エンスージアストに最適です。

注意点

まだ非常に新しいプロジェクトであり、一部の機能（オフライン専用化、検索機能の統合など）は開発途上です。特にローカルで実行する場合は、マシンのグラフィック性能やメモリ容量に依存するため、適切なハードウェア構成が必要です。

まとめ

Thunderbolt は、メールソフトの歴史を築いてきた Thunderbird チームが、AI 時代の「情報の自由とコントロール」を再定義しようとする意欲作です。ベンダーに支配されるのではなく、ユーザーが AI を道具として完全に使いこなすための、オープンな足場がここにはあります。
