これは何か

NOFX は、オープンソースで開発されている自律型の AI トレードプラットフォームです。従来のトレード Bot とは異なり、AI 自体が市場の状況を認識し、最適な LLM モデルを選択して、必要なデータを自動で取得・分析して取引を行います。ユーザーは戦略の大枠を決めるだけで、あとは AI が自律的に動く「AI Coworker」としてのトレード体験を提供します。

何ができるか

  • 自律型マーケット分析： AI が市場の動きを「知覚」し、人間に代わって意思決定を行います。
  • マルチアセット対応： 仮想通貨だけでなく、米国株、FX、貴金属など幅広い市場をサポート。
  • API キー不要の AI 利用： x402 規格のマイクロペイメントに対応しており、USDC の入ったウォレットがあれば API キーの設定なしで 15 種類以上の LLM を即座に利用可能です。
  • 多数の取引所連携： Binance、Bybit などの大手 CEX から、Hyperliquid などの最新の Perp-DEX まで幅広く接続可能。
  • 直感的な Web UI： Go 製のバックエンドと React/TradingView 製のフロントエンドにより、デスクトップから手軽に管理・運用できます。

目立つポイント

最も革新的な点は、AI モデルの利用料金を API キーではなく USDC で「従量課金」支払う仕組み（x402）を組み込んでいる点です。これにより、複数の LLM サービスにアカウントを作ったり、API キーの管理に追われたりすることなく、ウォレット一つですべての AI 機能を解放できます。開発に貢献すると将来的に収益の一部がエアドロップされる仕組みなど、コミュニティ主導のプロジェクトとしても注目されています。

セットアップや使い方の流れ

  1. Linux や macOS の場合は、ワンラインの install コマンド（curl -fsSL ... | bash）でセットアップが完了します。
  2. Windows の場合は、Docker Desktop を使用してクイックに起動可能です。
  3. ブラウザで http://127.0.0.1:3000 にアクセスし、初心者向けのガイド付きオンボーディングに従ってセットアップを進めます。
  4. AI の接続（API キーまたは x402 ウォレット）、取引所との連携、そして戦略の選択を行い、ダッシュボードから起動します。

どんな人向けか

  • AI を使った自律的なトレードに興味がある研究者やトレーダー。
  • API キーの管理や複雑なモデル構成を避け、シンプルに高性能な AI アシスタントを導入したい人。
  • Web3 時代の新しい決済プロトコル（x402）を活用した実用的なツールを体験したいパワーユーザー。

注意点

AI による自動取引には常に高いリスクが伴います。まずは少額、あるいは教育・研究目的での利用が推奨されます。また、モデルの利用には USDC による支払いが必要になる（x402 モードの場合）ため、ウォレットの管理には十分注意してください。

まとめ

NOFX は、AI・金融・決済の 3 つが高度に融合した次世代のトレード基盤です。「API Key ではなく Wallet で支払う」という新しい概念をトレードの世界に持ち込み、誰でも手軽に、かつ自律的な AI の力を借りて市場に挑むことができる環境を提供しています。
