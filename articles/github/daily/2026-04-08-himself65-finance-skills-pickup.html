これは何か

AI エージェントをプロの投資アナリストへと昇華させるための、金融特化型スキルセットです。オープンな「Agent Skills」規格に基づいて作成されており、Claude Code などの AI ツールに読み込ませることで、複雑な市場データの解析を専用コマンド一つで実行可能にします。

何ができるか

現在、特に「stock-liquidity（銘柄流動性分析）」スキルが注目されており、以下の高度な分析を AI が代行します。

  • 流動性スコアリング: 出来高プロファイルやビッド・アスク・スプレッドに基づき、その銘柄がどれだけ安定して売買可能かを評価。
  • 市場インパクトの予測: 大口注文を出した際に価格がどれだけ変動するか（マーケットインパクト）のシミュレーション。
  • Amihud 比率の算出: 出来高に対する価格変動の感度を示す非流動性指標（Amihud 比率）を用い、隠れたボラティリティリスクを特定。
  • マルチプラットフォーム対応: 一度導入すれば、サポートされている様々な AI エージェント上でこれらの金融コマンドが利用可能になります。

目立つポイント

汎用的な AI は「株価」は知っていても、その裏側にある「注文情報の薄さ（板の厚み）」や「執行リスク」までは自律的に評価できません。このスキルは、AI に金融工学的な分析手法を直接インストールすることで、より現実的でシビアな投資判断のアシストを可能にしている点が最大の特徴です。

セットアップや使い方の流れ

  1. リポジトリ内の SKILL.md ファイルを確認し、必要なコマンドを確認します。
  2. AI エージェント（Claude Code 等）のスキル管理ディレクトリに配置、もしくは npx skills add 等のコマンドを用いて導入します。
  3. AI との対話で「AAPL の流動性を分析して」といった命令を出します。
  4. AI が背後で各種指標を計算し、グラフや数値データの解釈を含む専門的なレポートを出力します。

どんな人向けか

  • AI を使って独自のアルゴリズム取引や高度な銘柄調査を行いたいトレーダー。
  • 金融データを扱う際の「AI の解釈精度」をさらに高めたい開発者。
  • データサイエンスと投資を組み合わせ、効率的な意思決定を行いたい人。

注意点

提供される情報は教育および情報提供のみを目的としており、実際の投資判断は自己責任で行う必要があります。データの取得には別途マーケットデータソースへのアクセスが必要な場合があります。

まとめ

finance-skills は、AI の汎用的な知性に「金融の専門知識」というプラグインを差し込むようなツールです。マーケットの荒波において、人間と AI が高度なデータに基づいたチームを組むための強力な武器となるでしょう。
