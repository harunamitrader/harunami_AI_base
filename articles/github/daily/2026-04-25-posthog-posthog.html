これは何か

Webやアプリのユーザー行動を分析するためのオープンソースプラットフォームです。Google Analytics などの代替として知られていますが、単なるアクセス解析にとどまらず、開発者がプロダクトを改善するために必要なツール一式を統合した「プロダクトOS」を目指しています。

何ができるか

イベントトラッキング、ユーザーの画面操作を動画で確認する「セッションリプレイ」、特定の機能のオンオフを制御する「機能フラグ（Feature Flags）」、さらに A/B テストやサーベイ機能まで備えています。最近では LLM オブザーバビリティ機能も追加され、AI アプリの開発・監視もサポートしています。

目立つポイント

最大の特徴は、これら全ての機能を単一のインターフェースで利用でき、かつオープンソース（MITライセンス）として提供されている点です。プライバシーやデータガバナンスが厳しい環境でも、自分のサーバーにインストールして全てのデータをコントロール下に置くことができます。また、SQL を直接記述して高度な分析ができる柔軟性も、エンジニアから高く評価されています。

セットアップや使い方の流れ

PostHog Cloud を利用して即座に開始することもできますが、セルフホストする場合は Docker や Kubernetes を使用します。Web サイトやアプリに JavaScript のスニペットを数行追加するだけでイベントの収集が始まり、ダッシュボード上でリアルタイムに分析結果を確認できるようになります。

どんな人向けか

ユーザー行動に基づいてプロダクトを高速に改善したい PM やエンジニアに向いています。特に、データプライバシーを重視する企業や、複数のツール（分析、フラグ管理、リプレイ）を一つにまとめたいチームに最適です。

注意点

多機能ゆえに、全ての機能を使いこなすには相応の学習コストがかかります。また、セルフホスト版は大規模なトラフィックを扱う場合、インフラ管理（特に ClickHouse などのデータベース）に専門的な知識が必要になることがあります。

まとめ

「開発者が真に欲しかった分析ツール」を具現化したプロジェクトです。プロダクトの成長に必要な全ての武器が揃っており、現代のソフトウェア開発において不可欠なインフラストラクチャの一つと言えるでしょう。
