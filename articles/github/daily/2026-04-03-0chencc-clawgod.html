これは何か

clawgod は、Anthropic 公式の Claude Code を置き換えるのではなく、その上にランタイムパッチを当てて機能拡張するツールです。README でも「サードパーティ製クライアントではない」と明記されており、既存の claude コマンドに対して追加機能を差し込む作りです。

何ができるか

README の説明では、hidden command 群の解放、feature flag 上書き、agent teams、Claude Code Remote を使う ultraplan / ultrareview、macOS での computer use などが挙がっています。加えて /share、/teleport、/issue、/bughunter のような追加コマンドや、ログ表示強化、メッセージフィルタ表示も含まれます。

目立つポイント

目立つのは、Claude Code 本体を fork して保守する形ではなく、更新後も再適用しやすい patch 層として設計している点です。README では「Claude Code が更新されても patch が効き続ける」と説明されており、さらに patched 版は緑、original 版はオレンジのロゴで見分けられるようになっています。

セットアップや使い方の流れ

導入はかなり単純で、macOS / Linux は install.sh、Windows は PowerShell 用の install.ps1 を 1 行で実行する形です。要件は Node.js 18 以上と Claude Code アカウントで、適用後は通常の claude が patch 済み版になり、元の実行系は claude.orig として残ります。

どんな人向けか

公式 Claude Code の体験を残したまま、より深い制御や独自コマンドを足したい人に向いています。特に Claude Code Remote や multi-agent 系の流れを試したい人、あるいは標準の安全策や制約を研究目的で外したい人には刺さりやすいです。

注意点

README にもある通り Anthropic 非公式で、制限解除や挙動変更を含む patch なので自己責任で使う前提です。特に destructive action 確認や安全制約を外す項目まで含まれるため、本番運用よりまず検証環境で試した方が安全です。

まとめ

clawgod は、Claude Code を別物に置き換えるのではなく、公式本体へ patch を重ねて拡張したい人向けのリポジトリです。通常の plugin より踏み込みが深く、その分リスクもあるので、Claude Code を強くカスタマイズしたい上級者向けと見ておくのが妥当です。
