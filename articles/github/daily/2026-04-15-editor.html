プロジェクトの概要

Pascal Editor は、ブラウザ上で直感的に3D建築モデルを構築・編集できるオープンソースのプラットフォームです。 最新の Web 規格である WebGPU を活用することで、従来の WebGL ベースのエディタを凌駕する描画性能と、複雑な 3D オブジェクトのリアルタイムな操作感を実現しています。 建築設計のプロトタイピングから、詳細なインテリアレイアウトまで、シームレスな編集体験を提供することを目指しています。

機能・できること

  • 直感的な建築編集: 壁の描画、床の生成、ゾーン（部屋）の定義、家具や設備の配置などをマウス操作で簡単に行えます。
  • 階層的ノード管理: 「サイト ＞ 建物 ＞ 階層」という論理的な構造でプロジェクトを管理。
  • リアルタイム保存・復元: IndexedDB を利用した自動保存機能と、高度な Undo/Redo システムを搭載。
  • 検証システム: 3D空間上での衝突判定や、配置ルールの検証をリアルタイムで実施。

リポジトリの構成

React 19、Next.js 16 という最新のフロントエンドスタックに加え、Three.js (WebGPU) と React Three Fiber を描画エンジンのコアに採用しています。 状態管理には Zustand、スキーマ検証には Zod を使用し、モノレポ管理には Turborepo、パッケージマネージャーには Bun を採用した非常にモダンな構成です。

セットアップ・導入のイメージ

Bun がインストールされた環境であれば、以下の手順で即座に開発環境を起動できます。 bun install bun dev 最新のブラウザ（WebGPU 対応）で localhost:3000 にアクセスするだけで、リッチなエディタが立ち上がります。

ターゲット層 / 注目ポイント

建築家やインテリアデザイナーはもちろん、3D Web アプリケーションの開発手法を学びたいエンジニアにとっても垂涎のリポジトリです。 特に WebGPU を本格的に活用した大規模なアプリケーションの実装例として、技術的な注目度が非常に高まっています。

制約・リスク

WebGPU などの最新技術に依存しているため、古いブラウザや一部のハードウェア環境では動作しません。 また、現在は v0.5.1 のプレリリース段階であり、API の破壊的変更や、複雑なシーンにおけるパフォーマンスのチューニングが課題として挙げられています。

まとめ

PASCALORG/Editor は、Web ブラウザがプロフェッショナルな設計ツールとしてどこまで進化できるかを示す野心的なプロジェクトです。 オープンソースでありながら極めて高い完成度を誇り、今後の 3D Web 開発の指標となる可能性を秘めています。
