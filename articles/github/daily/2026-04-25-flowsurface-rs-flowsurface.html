Flowsurface

Rust製の高速・高機能な仮想通貨チャーティングアプリ。ヒートマップ、フットプリント、DOMなど、プロ仕様の分析機能をネイティブデスクトップ環境で提供。

これは何か

仮想通貨市場に特化した、オープンソースのネイティブデスクトップ・チャーティングプラットフォームです。Rust 言語と iced GUI ライブラリを用いて構築されており、ブラウザベースのツールでは困難な、極めて低遅延で滑らかなユーザー体験を提供します。かつての Cryptowatch (Kraken Desktop) にインスパイアされた、プロ向けの分析ツールを目指しています。

何ができるか

Binance, Bybit, Hyperliquid, OKX, MEXC といった主要な取引所からリアルタイムの板情報（L2 orderbook）と約定データ（Trades）を取得し、多彩な形式で可視化します。標準的なローソク足チャートに加え、ヒートマップ（Historical DOM）、フットプリントチャート（Clustered Trades）、DOM（Ladder）、さらには複数銘柄の相関を比較する比較チャートなどを備えています。

目立つポイント

単なるチャート表示にとどまらず、トレードの音響効果やマルチウィンドウ対応、パネル間の銘柄同期（Pane linking）など、デイトレーダーが求める細かな機能が充実しています。特に、過去の板の厚みを時間軸で可視化するヒートマップ機能は、マーケットの流動性の変化を直感的に把握するのに非常に強力です。

セットアップや使い方の流れ

Windows, macOS, Linux 向けのプリビルド・バイナリが Releases ページから提供されており、インストール不要ですぐに使い始めることができます。また、Rust ツールチェーンがインストールされている環境であれば cargo install でビルドすることも可能です。起動後、設定画面から取引所を選択し、銘柄を検索してチャートを表示させます。

どんな人向けか

ブラウザベースのチャートツール（TradingView 等）では満足できない、より高度なオーダーフロー分析（板読み）を行いたい仮想通貨トレーダーに向いています。また、Rust によるモダンな GUI アプリケーションの実装例としても非常に参考になります。

注意点

GPLv3 ライセンスで公開されているオープンソースプロジェクトであり、現在は精力的に開発が進められている段階です。バイナリが署名されていないため、OS のセキュリティ警告が出ることがありますが、開発元は信頼できるオープンソースリポジトリとしての透明性を維持しています。

まとめ

「自分たちが本当に使いたいツールを作る」というオープンソースの精神が詰まった、高品質なチャーティングアプリです。Rust のパフォーマンスを最大限に活かした、次世代のトレーディング環境の基盤として、今後さらなる進化が期待されます。
