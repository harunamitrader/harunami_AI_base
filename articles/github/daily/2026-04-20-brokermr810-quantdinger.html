これは何か

QuantDingerは、AIによる市場リサーチから、Pythonを用いた戦略開発、バックテスト、そして実取引の執行までを一つのスタックで提供する、セルフホスト型のクオンツ・トレード・プラットフォームです。自身のインフラ（Docker）上で動作させることを前提として設計されており、取引データやAPIキー、独自の戦略コードを個人の管理下に置きつつ、最新のAI技術をトレードプロセスに統合することができます。

何ができるか

  • AI市場分析: 複数のLLM（OpenAI, Gemini, DeepSeek等）を切り替えて、市場データやニュースに基づいた高度な分析・判断支援を受けられます。
  • Python戦略開発: Pythonで標準的なインジケーターや複雑なロジックを直接記述できます。自然言語からコードを生成するAIアシストも内蔵されています。
  • バックテストと可視化: 過去データを用いた手法の検証、資産曲線の確認、AIによるテスト結果へのフィードバックが可能です。
  • マルチアセット実取引: 仮想通貨（主要取引所）、米国株（IBKR経由）、FX（MT5経由）の執行に対応したコネクタを備えています。
  • 運用管理: Webベースのダッシュボード、マルチユーザー管理、Telegram/Discord等への通知、USDT決済による課金・メンバーシップ管理機能まで含まれます。

目立つポイント

最大の特徴は、単なる「AIチャット付きトレードアプリ」ではなく、「クオンツ運用のためのOS」を目指している点です。多くのOSSトレードツールが欠いている「ユーザー管理」「課金基盤」「通知システム」を標準で備えており、個人利用から小規模チームでの商業展開までを視野に入れたフルスタックな構成になっています。また、セルフホスト型であるため、秘匿性の高い独自のアルゴリズムを外部サービスに預けることなく、かつモダンなUI/UXで運用できる点は強力なメリットです。

セットアップや使い方の流れ

  1. リポジトリを git clone し、ディレクトリに移動します。
  2. backend_api_python/env.example をコピーして .env を作成し、必要なAPIキーを設定します。
  3. 用意されているスクリプト ./scripts/generate-secret-key.sh を実行して秘密鍵を生成します。
  4. docker-compose up -d --build でシステムを起動します。
  5. ブラウザで http://localhost:8888 にアクセスし、デフォルト設定でログインします。
  6. AIによる市場リサーチから手法を検討し、Pythonで実装・バックテストを行い、準備が整えば実取引の設定を行います。

どんな人向けか

  • プライバシー重視の投資家: 自身の手法やAPIキーの安全性を確保したい個人。
  • クオンツ開発者: Pythonの柔軟性とAIのスピードを組み合わせて、効率的にアルゴリズムを開発・検証したいエンジニア。
  • 小規模トレードチーム: 共通のインフラで分析・実行環境を構築し、役割に応じたユーザー管理を行いたい組織。
  • 商用プラットフォーム構築者: 課金やユーザー管理の仕組みが整ったトレード基盤をベースに、独自のサービスを展開したい運営者。

注意点

本システムはセルフホスト（自己運用）が前提であるため、サーバーの維持管理やセキュリティ設定はユーザー自身が行う必要があります。また、実取引（Live Trading）を行う場合は、APIキーの権限設定を適切に行い、十分なバックテストと低資金でのテスト運用を経てから本番稼働させることを強く推奨します。

まとめ

QuantDingerは、リサーチから執行、さらにはプラットフォームとしての運用機能までを統合した、極めて包括的なオープンソース・プロジェクトです。AIとPythonという現在のテクノロジーの核を、個人の管理するインフラ上で有機的に結びつけることで、従来のプラットフォームでは得られなかった「自由度」と「高度な分析能力」を両立させています。
