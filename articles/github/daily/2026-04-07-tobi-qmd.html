リポジトリ概要

qmd (Query Markup Documents) は、個人のノートや企業のドキュメントベースを対象とした、ローカル完結型のインテリジェント検索エンジンです。従来のキーワード検索（BM25）にベクトル検索（意味検索）を組み合わせ、さらに LLM による再ランキングを行うことで、極めて精度の高い検索結果を提供します。

主な特徴と機能

  • ハイブリッド検索: 全文検索とベクトル検索を融合し、文脈に即した最適な情報を抽出します。
  • 完全ローカル実行: node-llama-cpp を活用し、機密情報を外部に送信することなくデバイス上で全処理を完遂します。
  • MCP サーバー対応: Model Context Protocol に対応しており、Claude Desktop や Claude Code などのエージェントから直接ドキュメントを検索可能です。
  • AST チャンキング: Markdown だけでなく、コード構造を理解した高度な分割アルゴリズムを搭載しています。

最近のアップデート (2026年4月)

バージョン 2.1.0 がリリースされ、TypeScript や Python などのコードファイルに対する AST 対応のチャンキング機能が強化されました。また、HTTP トランスポートへの対応により、チーム内での共有サーバーとしての運用も容易になっています。

導入と活用方法

  1. npm または GitHub からリポジトリをクローンし、依存関係をインストールします。
  2. 検索対象となる Markdown ファイルやドキュメントのディレクトリを指定してインデックスを作成します。
  3. MCP サーバーとして登録することで、AI エージェントがあなたの知識ベースを自在に参照できるようになります。

まとめ

AI エージェントの性能を最大限に引き出すためには、良質なコンテキストの供給が不可欠です。qmd は、プライバシーを守りながら自分だけの「第二の脳」を AI に提供するための、現代的な検索インフラストラクチャと言えるでしょう。
