これは何か

Claude Code にプラグインして利用できる、オープンソースの AI マーケティング & セールス自動化ワークフロー集です。単なるプロンプト集ではなく、Python スクリプトによるデータ解析と、AI エージェントへの精密な指示書（SKILL.md）がセットになった「実戦的なツールキット」を提供します。

何ができるか

  • Growth Engine: 統計手法（マン・ホイットニーの U 検定等）を用いた客観的なグロース実験の分析。
  • Sales Pipeline: 失注案件からの再三の機会創出や、LinkedIn 連携によるパイプライン構築。
  • Content Ops: 複数の専門家ペルソナによる反復修正を行い、高品質なコンテンツを自律生成。
  • SEO & Finance Ops: キーワード調査の自動化や ROI（投資対効果）の自動追跡。

目立つポイント

AI の「直感的出力」を「統計学」で補完し、マーケティング判断の信頼性を高めている点が秀逸です。また、Claude Code のスキルシステムをフル活用しており、各カテゴリの SKILL.md を導入するだけで、複雑なワークフローを自然言語の指示（例：「LinkedIn での A/B テストを計画して」）のみで実行可能にします。

セットアップや使い方の流れ

1. インストール: リポジトリをクローンし、pip install -r requirements.txt で依存環境を導入します。

2. 環境設定: .env ファイルを作成し、OpenAI 等の必要な API キーを設定します。

3. スキルの導入: 使用したいカテゴリの SKILL.md ファイルを、自身のプロジェクトの .claude/skills/ 等へ配置します。

4. 実行: Claude Code を起動し、「実験をレポートして」「新しいコンテンツを作成して」などの指示を出します。

どんな人向けか

マーケティング担当者やグロースハッカー、SalesOps の専門家に最適です。特に Claude Code を活用して、単なるテキスト生成を超えた「自律的な業務遂行」を実現したい開発者やビジネスオーナーにとって強力な武器となります。

注意点

利用する AI モデルや LinkedIn 等の外部サービスに応じて、各種 API の利用料金が発生する場合があります。基本的には Claude Code 環境での利用を前提とした設計になっています。

まとめ

ai-marketing-skills は、AI エージェントを単なるアシスタントから、高度なビジネスロジックを自律実行する「デジタルエンジニア」へと進化させるライブラリです。マーケティングの自動化を次のレベルへ引き上げたいチームにとって、必見の内容です。
