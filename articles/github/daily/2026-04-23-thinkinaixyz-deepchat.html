これは何か

DeepChatは、モデル、ツール、エージェント実行環境を1つのデスクトップアプリケーションに統合した強力なオープンソースAIプラットフォームです。OpenAI、Gemini、AnthropicなどのクラウドAPIや、Ollamaを用いたローカルモデルをシームレスに利用できるだけでなく、MCP（Model Context Protocol）やACP（Agent Client Protocol）といった次世代のエージェントプロトコルを標準でサポートしています。

何ができるか

複数モデルの統合管理、Ollamaを用いたローカルモデルの容易なデプロイ・実行、そしてMCPを通じたコード実行やWebアクセスなどのツール呼び出しが可能です。さらに、Agent Client Protocol (ACP) を介して、外部の自律エージェントをモデルの1つとして扱うこともでき、構造化されたプランやターミナル出力用の専用ワークスペースUIを提供します。

目立つポイント

  • マルチモデル対応: クラウドAPIからローカルモデル(Ollama)まで、主流なLLMを幅広くサポートし、1つのアプリで管理可能。
  • MCPエコシステム内包: リソース、プロンプト、ツールの3つのコア機能をサポート。組み込みのNode.jsランタイムにより、設定不要ですぐにコード実行やWeb検索などを活用できます。
  • ACP統合: ACP互換の自律エージェントを統合し、専用のワークスペースUIで高度なタスクを効率よく実行可能です。
  • 洗練されたUI/UX: CodeMirrorベースのコード表示、マルチウィンドウ・タブでの並行操作、各種Artifactsレンダリングなど、モダンなチャットクライアントとしての完成度が非常に高いです。

セットアップや使い方の流れ

Windows、macOS、Linux向けのバイナリが提供されており、公式サイトやGitHubのReleasesページからダウンロードしてインストールします。macOSの場合はHomebrew（brew install --cask deepchat）でも導入できます。インストール後、設定画面からAPIキーを入力するかOllamaを連携させれば、すぐにチャットやエージェント機能を利用開始できます。

どんな人向けか

複数のLLMを切り替えて使いたい開発者やクリエイター、最新のMCPツールや自律型エージェント(ACP)をコマンドラインではなく使いやすいGUI上で一元管理・活用したいパワーユーザーに最適です。

注意点

クラウドAPIの利用には各自でAPIキー（従量課金等）の準備が必要です。また、MCPやACPによるコード実行ツールなどは強力な反面、意図しないファイル操作などが発生する可能性があるため、実行環境には注意して利用してください。

まとめ

クラウドAPIとローカルモデルの壁をなくし、最新のエージェントプロトコル（MCP・ACP）の恩恵を直感的かつ美しいデスクトップUIで享受できる、まさに次世代のAIワークステーションです。
