これは何か

bolt.new は、ブラウザ内で full-stack 開発を完結させる AI-powered web development agent です。README では、prompt でアプリを作り、そのまま実行、編集、deploy まで進められる環境として説明されています。

何ができるか

WebContainers 上で npm install、Node.js server 起動、third-party API 連携、production deploy まで扱えます。単なるコード補完ではなく、AI が filesystem、terminal、browser console、package manager を跨いで触れるのが大きいです。

目立つポイント

ポイントは「ローカルセットアップ不要で full-stack を回せる」ことです。Claude や v0 系の UI 生成より一段踏み込み、backend 実行や依存追加もブラウザ内で片づくので、試作速度がかなり高いです。

セットアップや使い方の流れ

利用者としては bolt.new にアクセスして prompt を投げるだけですが、この repo は Bolt の open source codebase そのものでもあります。自前の AI web dev agent を作りたい人はここから構造を読めます。

どんな人向けか

フロントだけでなく backend も含めた試作を最短で回したい人、あるいは browser-native な agent IDE の作り方を学びたい人に向いています。

注意点

README でも beta 扱いで、対応フレームワークや運用制約は StackBlitz / WebContainers の特性に依存します。安定運用というより、まずは高速試作と体験価値に強い系統です。

まとめ

bolt.new は、AI coding をブラウザだけで完結させる代表例として見ておく価値があります。特に WebContainers と agent をどう結びつけるかの参考になります。
