これは何か

Page Agent は、Webページ内で直接動作する JavaScript ベースの GUI エージェントです。自然言語を使用して、Web インターフェースを直感的に操作できます。

従来の自動化ツール（ブラウザ拡張機能、Python、ヘッドレスブラウザなど）とは異なり、ページ内の JavaScript だけで完結するのが最大の特徴です。

何ができるか

  • **簡単な統合**: ブラウザ拡張や外部サーバー不要で、インページ JavaScript のみで動作。
  • **テキストベースの DOM 操作**: スクリーンショットやマルチモーダル LLM を使わず、テキストベースで DOM を解析・操作。
  • **任意の LLM 連携**: OpenAI 互換 API など、任意の LLM をバックエンドとして利用可能。
  • **拡張性**: マルチページタスク用の Chrome 拡張機能や、外部からブラウザを制御するための MCP (Model Context Protocol) サーバーも提供。

目立つポイント

完全にインページで動作するため、デプロイや統合が非常に簡単です。数行のコードで既存のWebアプリケーションにAIアシスタント機能を追加でき、開発コストを大幅に削減できます。

また、テキストベースでの DOM 操作により、従来の画像認識に頼る方法よりも堅牢で安定した自動化を実現します。

セットアップや使い方の流れ

CDN を使用したクイックデモ

HTML に以下のスクリプトを追加するだけで、すぐにデモ用の LLM を使用して試すことができます。

<script src="https://cdn.jsdelivr.net/npm/page-agent@1.7.1/dist/iife/page-agent.demo.js" crossorigin="true"></script>

npm によるインストールと利用

プロジェクトに組み込む場合は、npm を使用します。

npm install page-agent

基本的な TypeScript コード例:

import { PageAgent } from 'page-agent'

const agent = new PageAgent({
    model: 'qwen3.5-plus', // 使用するモデル名
    baseURL: 'https://dashscope.aliyuncs.com/compatible-mode/v1', // API エンドポイント
    apiKey: 'YOUR_API_KEY', // API キー
    language: 'ja-JP', // 言語設定
})

// 自然言語で操作を実行
await agent.execute('ログインボタンをクリックして')

どんな人向けか

  • 自社サービスにAIアシスタント機能を素早く導入したい SaaS 開発者。
  • 複雑なWebフォーム入力を自動化したい業務担当者。
  • Webアプリケーションのアクセシビリティを向上させたい開発者。
  • ブラウザベースのマルチページ自動化ツールを構築したい開発者。

注意点

LLM の精度やレスポンス速度に依存するため、安定した性能を発揮するには適切なモデルの選定とチューニングが必要です。

また、DOM 構造の変更には影響を受ける可能性があるため、定期的なテストとメンテナンスが推奨されます。

まとめ

Page Agent は、Webページの GUI 操作を自然言語で行うための強力な JavaScript ライブラリです。ブラウザ拡張機能や外部サーバーに依存せず、インページで完結する手軽さと、任意の LLM と連携できる柔軟性により、SaaS AI コパイロット、自動フォーム入力、アクセシビリティ向上など、幅広い分野での応用が期待されます。

Web オートメーションの新しい形を提案し、開発者がより効率的かつ革新的な Web アプリケーションを構築する手助けとなるでしょう。
