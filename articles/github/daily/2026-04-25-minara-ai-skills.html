これは何か

Claude Code や OpenClaw、Hermes といった AI エージェントに、高度な金融・トレード機能を付与するための公式スキルセットです。AI エージェントを単なるコーディング補助やチャットボットから、資産運用や市場分析を自律的にこなす「パーソナル AI CFO」へと進化させます。

何ができるか

仮想通貨（現物・無期限先物）、米国株、コモディティ、為替（Forex）のリアルタイム市場データの取得、分析、そして実際の取引実行までをサポートします。EVM 互換チェーン（Ethereum, Base, Arbitrum 等）や Solana、Hyperliquid といった主要なネットワークに対応しており、オンチェーン・オフチェーンを問わず、AI への自然言語指示だけで複雑な金融操作を実現します。

目立つポイント

プロダクトの完成度と、AI への統合の容易さが最大の特徴です。単純な情報取得だけでなく、リミットオーダー（指値）の設定やマルチウォレット管理、さらには「x402」による API 直接支払い機能まで備えています。また、Claude Code 向けにはワンクリックでセットアップできるシェルスクリプトが用意されており、即座に「取引可能な AI エージェント」を構築できます。

セットアップや使い方の流れ

公式のセットアップスクリプトをターミナルで実行するだけで、Minara CLI のインストールとスキルの登録が自動で行われます。セットアップ後は、エージェントに対して「ポートフォリオを表示して」「ETH を 100 USDC 分買って」「トレンドのトークンを教えて」といった自然言語で話しかけるだけで、裏側で Minara の API と連携してタスクが実行されます。

どんな人向けか

AI エージェントを実務や資産運用に活用したいエンジニア、独自の投資戦略を AI に自律実行させたいトレーダー、または AI による高度な金融リサーチ環境を構築したいリサーチャーに最適です。

注意点

実際の資産を扱うツールであるため、API キーや秘密鍵の管理には細心の注意が必要です。本リポジトリ自体は設定スクリプトとドキュメントのみを含んでおり、実際のロジックは Minara CLI が担う構造になっています。また、金融取引には常にリスクが伴うため、AI の提案を鵜呑みにせず、十分な検証が必要です。

まとめ

AI エージェントの役割を「金融」という専門領域にまで広げる、非常に野心的なプロジェクトです。特にエージェントフレームワークとの密接な連携が考慮されており、これからの「自律型金融エージェント」時代のスタンダードになり得るポテンシャルを感じさせます。
