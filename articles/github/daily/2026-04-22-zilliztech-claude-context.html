これは何か

claude-context は、Anthropic 社の Claude デスクトップアプリのための強力なメモリ（記憶）拡張ツールです。ベクターデータベースの Milvus などを活用し、Claude との過去の対話履歴や、ユーザーが指定したドキュメントを「長期記憶」として保持・検索可能にします。

何ができるか

デスクトップアプリのサイドバー等に統合され、現在の会話に関連する過去の情報を自動的に検索・提示します。これにより、「3日前のあの件について教えて」といった曖昧な指示にも対応でき、長期プロジェクトにおけるコンテキストの維持が劇的に容易になります。また、特定のローカルディレクトリを「知識ベース」として Claude に学習させることも可能です。

目立つポイント

公式の機能にはまだ存在しない（あるいは限定的な）「真の長期記憶」を、ユーザーの手元で実現している点が最大の特徴です。開発元がベクター検索の専門企業である Zilliz であるため、数百万件のメッセージがあったとしても、ミリ秒単位で関連情報を引き出すことが可能です。

セットアップや使い方の流れ

Node.js 環境と、バックエンドとして動作する Milvus Lite (または Docker 上の Milvus) が必要です。リポジトリをクローンし、必要な依存関係をインストールした後、npm start でコンパニオンアプリを起動します。その後、Claude デスクトップアプリの設定で MCP (Model Context Protocol) サーバーとして登録することで、Claude 内からシームレスに記憶へのアクセスが可能になります。

どんな人向けか

Claude を仕事や創作のメインパートナーとして活用しており、セッションをまたいだ文脈の共有や、自身の膨大なナレッジベースとの連携を必要としているヘビーユーザーに最適です。

注意点

現時点では非公式の拡張ツールであるため、Claude アプリ側のアップデートによって動作が不安定になる可能性があります。また、会話履歴という機密性の高いデータを扱うため、セルフホスト環境のセキュリティ管理は自身で行う必要があります。

まとめ

claude-context は、Claude の高い知能を「記憶」という翼でさらに羽ばたかせるためのツールです。これ一つで、AI アシスタントとの対話はより個人的で、より一貫性のあるものへと進化するでしょう。
