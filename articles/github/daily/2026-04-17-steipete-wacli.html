これは何か

wacli は、世界的に普及しているメッセージングアプリ「WhatsApp」をターミナル（CLI）から直接操作するためのオープンソースツールです。GUI を開くことなく、Rust 製の高速なインターフェースを通じてメッセージのやり取りが可能になります。

何ができるか

  • メッセージ送受信： コマンドラインからのリアルタイムチャット。
  • メディア対応： 画像やファイルなどの添付情報の管理。
  • コンタクト管理： 連絡先の検索やメッセージ履歴の表示。
  • ヘッドレス動作： サーバー上での実行やスクリプト連携。

目立つポイント

公式 SDK でないにも関わらず、非常に安定しており、WhatsApp Web の API を効率的にラップしている点です。特に開発者が作業中に画面を切り替えることなく、ターミナル内で連絡を取れるため、生産性の向上が期待できます。また、Rust で記述されているためレスポンスが非常に軽快です。

どんな人向けか

ターミナルの外に出たくない開発者、WhatsApp を使った通知の自動化を検討しているエンジニア、または Vimmer/Emacser。

注意点

WhatsApp の利用規約に基づき、スパム行為や不適切な自動配信に使用するとアカウント停止のリスクがあります。また、利用には事前の QR コード認証（WhatsApp 側のセキュリティ）が必要です。

まとめ

「ターミナルこそがホーム」であるエンジニアにとって、WhatsApp への窓口となるこのツールは非常に魅力的です。
