これは何か

「検閲なし（No Guardrails）」を掲げるオープンソースの統合生成AIプラットフォームです。画像生成から動画、シネマティックな映像制作、リップシンクまで、クリエイティブなワークフローに必要な機能を一つのインターフェースに集約しています。

何ができるか

Image Studio, Video Studio, Cinema Studio, Lip Sync Studio の 4 つのモジュールを使い分けることで、Flux, Stable Diffusion, Kling, Sora といった 200 種類以上の高度なモデルを自在に操れます。ローカルの Apple Silicon での実行（Metal 加速）や、クラウド API を介した高速生成が可能です。

目立つポイント

大手企業の AI サービスで見られるような過度なフィルタリングやプロンプト拒絶がなく、自由な表現を追求できる点が最大の特徴です。また、Cinema Studio では「仮想レンズ」や「絞り設定」などのカメラパラメータを用いて生成を制御できる、プロフェッショナル志向の UI を備えています。

セットアップや使い方の流れ

リポジトリをクローンし、必要なモデルや API キーを設定して起動します。Mac ユーザーであれば Metal 加速を有効にすることで、ローカル環境でも快適に動作します。目的の「Studio」を選択し、モデルを選んでプロンプトを入力するだけで、検閲を気にせずクリエイティブ活動を開始できます。

どんな人向けか

制限に縛られず自由な表現をしたいデジタルアーティスト、最新の AI モデルを一つの場所で使い倒したい研究者、またはプライバシーやデータ制御を重視してセルフホストしたいユーザーに最適です。

注意点

「検閲なし」は強力な自由を意味しますが、同時にユーザー自身の倫理観と法的責任が問われます。また、多くのモデルを統合しているため、全ての機能をフル活用するには相応のマシンスペックや API コストの管理が必要です。

まとめ

集中管理される AI へのカウンターとして登場した、極めて自由度の高いオープンソースプロジェクトです。クリエイターが「道具」としての AI を完全に支配下に置くための、強力な選択肢となるでしょう。
