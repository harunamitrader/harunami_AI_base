これは何か

Recordly は、画面録画とデモ動画向けの編集を 1 つにまとめたデスクトップアプリです。README では walkthrough、tutorial、product video、social clip 向けの open-source screen recorder and editor と説明されており、録画後すぐに編集へ入れるのが前提になっています。

何ができるか

画面やウィンドウの録画、マイクとシステム音声の収録、ズーム演出、カーソル装飾、背景やフレームの調整、注釈追加、webcam bubble、速度変更、トリミング、MP4 / GIF 書き出しまでをまとめて扱えます。単に録るだけでなく、プロダクトデモとして見せやすい形まで持っていけるのが強みです。

目立つポイント

README を見ると、auto-zoom suggestions、cursor smoothing、cursor motion blur、cursor loop mode など「デモ動画で見栄えが変わる細部」にかなり力が入っています。さらに .recordly の project file で編集状態を保存して再開できるので、録って終わりではなく繰り返し仕上げる運用もやりやすいです。

セットアップや使い方の流れ

使い始め方は、GitHub Releases から各 OS 向け build を入れるのが基本です。ソースから試す場合は git clone 後に npm install、npm run dev で起動できます。macOS は 12.3 以上、Windows は 19041 以上が推奨で、Linux は modern distro と PipeWire 前提の説明になっています。

どんな人向けか

SaaS や開発ツールの紹介動画を自分で作る人、あるいはチュートリアルを短時間で整えたい人に向いています。特に「録画ソフト」と「編集ソフト」を分けたくない個人開発者や小規模チームにはかなり相性が良さそうです。

注意点

README では Linux の cursor hiding 未対応、Windows の古い build では real cursor が残る可能性、system audio の OS 依存など制約も明記されています。クロスプラットフォーム対応は広いですが、録画まわりの完成度は OS ごとに差がある前提で見た方が安全です。

まとめ

Recordly は、録画から編集までを 1 本化して、デモ動画制作の手数を減らしたい人に向いた実用的なオープンソースです。見せ方のための細かな機能が最初からまとまっているので、プロダクト紹介用途では特に使いどころがはっきりしています。
