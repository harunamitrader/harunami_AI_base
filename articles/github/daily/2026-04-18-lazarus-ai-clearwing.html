これは何か

Clearwing は、Lazarus AI（著名な Eric Hartford 氏ら）によって開発された、自律型の脆弱性スキャナー兼ソースコードハンターです。 Anthropic が非公開で行っていた「Glasswing」の挑戦に触発され、LangGraph をベースに構築されました。 誰もがアクセス可能な LLM を使いながら、ネットワークペネトレーションテストと深層ソースコード解析の両方を自律的に実行することを目指しています。

何ができるか

  • ソースコード・ハンティング： 膨大なファイルをランク付けし、ハンターエージェントが脆弱性を探索。ASan/UBSan 等での「クラッシュの再現」を真実の根拠（Ground Truth）として使用します。
  • ネットワーク・ペンテストエージェント： 63 のツールを装備し、Kali Linux のツールセットをサンドボックス内で実行。ライブターゲットのスキャンから脆弱性検知、エクスプロイト試行までを行います。
  • 検証済みパッチ生成： 見つけた脆弱性に対し、ルート原因の解明だけでなく、検証済みの修正パッチ生成まで試みます。
  • 段階的な証拠レベル： 報告書は「疑い（Suspicion）」から「エクスプロイト実証（Exploit Demonstrated）」、そして「パッチ検証済み（Patch Validated）」まで、厳密なエビデンスレベルでランク付けされます。

目立つポイント

最大の特徴は、**「実証ベース（Logic & Evidence First）」** の設計です。 AI がありがちな脆弱性を指摘するだけでなく、実際にコードを走らせ、クラッシュを再現し、原因を説明するという「人間であれば数時間かかる作業」を自律的に高速実行します。 また、Eric Hartford 氏の関与により、特定の巨大プロバイダーに依存せず、OpenRouter や自前の Ollama 等でも動作可能なオープンな設計が貫かれています。

セットアップや使い方の流れ

Python 3.10+ 環境で uv sync を行い、clearwing setup ウィザードでプロバイダー（Anthropic や OpenAI 互換 API）を設定します。 準備ができれば clearwing sourcehunt [REPO_URL] でリポジトリの深層分析を開始、あるいは clearwing interactive でエージェントと直接対話しながらのペンテストも可能です。

どんな人向けか

セキュリティエンジニア、レッドチーマー、そして「自社のコードの安全性を、静的解析ツール以上、人間以下のコストで徹底的にチェックしたい」開発者にとって、これ以上ない強力な武器となります。

注意点

Clearwing は強力な「攻撃ツール」としての側面も持ちます。 必ず自分が所有する、あるいは明示的な許可を得たターゲットに対してのみ使用してください。 また、高度な分析には相応の LLM トークンコスト（予算設定機能あり）と実行時間がかかります。

まとめ

Clearwing は、AI が単なる「コードの書き手」から「コードの守り手（および厳格な監査官）」へと進化したことを象徴するプロダクトです。 実証に裏打ちされたその報告能力は、従来のセキュリティツールの限界を大きく超え、AI エージェントの実用性を新たなフェーズへと押し上げています。
