SEO特化型ブログコンテンツ作成ワークスペース：Claude Code上で動作する調査から最適化までを網羅した自動化システム。

記事作成日 2026-04-08
作成時順位 5位
対象 repo TheCraigHewitt/seomachine

これは何か

seomachineは、AnthropicのCLIツール「Claude Code」上で動作するように設計された、SEO（検索エンジン最適化）に特化したブログ記事制作ワークスペースです。コンテンツの調査から執筆、分析、既存記事の最適化まで、一連のSEOワークフローをAIエージェントの力を借りて効率化します。

何ができるか

専用のコマンド（/research, /write, /analyze-existing, /optimize等）を用いて、各工程を自動化できます。内蔵された26種類のマーケティングスキルを活用し、ターゲットキーワードに基づくコンテンツ分析や内部リンクの提案、Google Search ConsoleやGA4のデータと連携したリアルタイム分析が可能です。

目立つポイント

Claude Codeの強力なコンテキスト management機能を最大限に活用している点が特徴です。役割が定義された複数の専門エージェント（コンテンツアナリスト、SEOオプティマイザー等）が連携し、高品質な記事作成をサポートします。また、BeautifulSoupやScikit-learnを用いた本格的なスクレイピング・分析モジュールも組み込まれています。

セットアップや使い方の流れ

Claude Codeをインストールした環境で、このリポジトリのワークスペース設定を読み込むことで利用を開始します。Google AnalyticsやSearch Console、DataForSEOといった外部APIキーを設定することで、実データに基づいた高度な最適化提案が受けられるようになります。

どんな人向けか

コンテンツマーケティングを効率化したいエンジニアやブロガー、SEO専門家、そしてClaude Codeを実務に活用したい開発者にとって非常に価値のあるプロジェクトです。SEOのベストプラクティスをAIワークフローに落とし込みたいと考えている人にも最適です。

注意点

Claude Code（β版）の利用を前提としているため、ツール自体のアップデートによる影響を受ける可能性があります。また、外部APIの利用には別途トークンや費用が発生する場合があるため、事前に確認が必要です。

まとめ

seomachineは、AIエージェントが「開発」だけでなく「マーケティング」領域においても、専門的で構造化されたワークフローを自律的に遂行できることを示した実例です。ターミナル完結型のSEOツールとして、今後の進化が期待されます。
