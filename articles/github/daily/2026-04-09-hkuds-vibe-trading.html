これは何か

Vibe-Tradingは、香港大学の研究チーム（HKUDS）が開発した、AI駆動のマルチエージェント型金融ワークスペースです。ユーザーが自然言語で「〜といった戦略をテストして」「最新の暗号資産市場のセンチメントを分析して」といった依頼を投げるだけで、AIエージェントが自律的にコードを書き、データを取得し、分析レポートを作成します。

何ができるか

単なるチャットボットではなく、実行能力（Execution）に重点が置かれています。トレード戦略の自動生成、マルチソースからの市場データ取得、過去データを用いたバックテスト、そして投資委員会や暗号資産デスクといった仮想的な「専門家チーム」による多角的なリサーチが可能です。米国株、香港株、中国A株、そして暗号資産といった主要市場を横断してサポートしています。

目立つポイント

最大の特徴は、金融ドメインに特化した「64のスキル」と「29のチームプリセット」です。テクニカル分析、クオンツ、財務諸表分析、グローバルマクロなど、人間のアナリストが行うような高度なタスクがモジュール化されており、これらを組み合わせて「投資委員会」エージェントが議論を戦わせるようなワークフローを簡単に構築できます。

セットアップや使い方の流れ

導入は非常に柔軟で、pip install vibe-trading-ai によるインストール、Dockerによるワンコマンド起動、さらには MCP（Model Context Protocol）サーバーとしての利用も可能です。起動後は対話型の CLI や Web UI を通じてエージェントに指示を与えます。OpenAI 互換の API キーがあれば、すぐに分析を開始できます。

どんな人向けか

自分のトレードアイデアを素早く検証したい個人トレーダーや、膨大なデータから特定の市場シグナルを抽出したいクオンツ・リサーチャーに最適です。また、AIエージェントを金融実務でどう活用すべきかのフレームワークを探している開発者にとっても、非常に完成度の高いリファレンス実装となります。

注意点

現時点ではバックテストやリサーチが主眼であり、直接的なライブ取引（自動売買の執行）については、各ブローカーの API 連携を含め、自身の責任で環境を構築する必要があります。また、高度な分析には強力な LLM（Claude 3.5 Sonnet や GPT-4o 等）の使用が推奨されます。

まとめ

Vibe-Tradingは、金融分析という専門性の高い領域において、AIエージェントが「思考」だけでなく「実行」までを代替する強力なツールです。自然言語という最も手軽なインターフェースで、プロフェッショナルなリサーチチームを手に入れたかのような体験を提供してくれる、次世代の金融分析基盤と言えるでしょう。
