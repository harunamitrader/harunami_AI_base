これは何か

yt-short-clipper は、ポッドキャストやインタビューのような長尺 YouTube 動画を、TikTok、Instagram Reels、YouTube Shorts 向けの短尺動画に変換する自動処理パイプラインです。README では desktop app とソース実行の両方を案内しており、非技術者向けの導入ガイドも用意されています。

何ができるか

YouTube 動画の取得、字幕解析、AI による見どころ抽出、FFmpeg での切り出し、16:9 から 9:16 への縦変換、フック用テキストと TTS 生成、Whisper による単語単位字幕、ウォーターマーク追加、SEO 用タイトルと説明文生成まで一連で処理できます。単なる切り抜きではなく、公開用のショート動画素材をまとめて作る設計です。

目立つポイント

特徴は、切り抜きの前後工程までかなり揃っている点です。AI ハイライト検出は GPT-4 ベース、顔追従は OpenCV と MediaPipe の 2 モード、字幕は CapCut 風のハイライト付き、導入面では desktop app まであるため、Python スクリプトだけのツールより使い始めやすい構成です。

セットアップや使い方の流れ

利用者向け README では、アプリ取得後に yt-dlp、FFmpeg、Deno、YouTube cookies、AI API を設定して動画処理に入る流れが示されています。開発者向けには Python 3.10 以上を前提にソース実行やビルド手順も分かれており、用途に応じて GUI 利用と開発運用を切り替えやすいです。

どんな人向けか

長尺コンテンツを継続的にショート動画へ再利用したい個人運営者や小規模チームに向いています。特に切り抜き候補探し、字幕付け、縦動画化を毎回手作業で回している人には、工程圧縮の効果が見えやすいはずです。

注意点

依存として yt-dlp、FFmpeg、Deno、Whisper、外部 AI API、YouTube cookies など複数の前提があり、権利処理や動画利用規約も別途整理が必要です。また顔追従は精度と速度のトレードオフがあり、MediaPipe モードは精度が高い代わりに 2 から 3 倍遅いと説明されています。

まとめ

yt-short-clipper は、YouTube の長尺動画を「見どころ抽出から Shorts 向け仕上げまで」一気に持っていくための実務寄りツールです。素材が多く、短尺展開を継続している人ほど恩恵を受けやすいリポジトリです。
