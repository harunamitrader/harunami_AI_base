これは何か

system_prompts_leaks は、人気 AI チャットやコーディング支援ツールの system prompt、system message、developer instructions を整理して集めたリポジトリです。README では ChatGPT、Codex、Claude、Gemini、Grok、Perplexity などを継続更新対象として明示しています。

何ができるか

Anthropic、OpenAI、Google、xAI、Perplexity、Misc といったフォルダ単位で prompt 群をたどれます。OpenAI セクションだけでも GPT-5.4、GPT-5.3、Codex、Codex CLI、各種ツール指示まで分かれており、モデルごとの差分や製品ごとの設計思想を比較しやすい構造です。

目立つポイント

単に断片を並べるのではなく、モデル名ごとにリンクを整理していて追跡しやすいのが特徴です。更新頻度も高く、PR 歓迎を明記しているため、一次資料を集約するコミュニティ型アーカイブとして機能しています。

セットアップや使い方の流れ

利用自体にセットアップはほぼ不要で、README から各モデルや製品の prompt ファイルへ直接たどれます。調査用途では特定ベンダーだけを見るより、同じカテゴリの prompt を横並びで比較しながら読む使い方が向いています。

どんな人向けか

AI プロダクトの振る舞いを観察したい人、プロンプト設計やガードレール比較をしたい人、記事や研究の一次資料を探している人に向いています。特にプロダクト分析や評価観点づくりの出発点として使いやすいです。

注意点

収集物の性質上、内容の完全性や鮮度は常に変動します。README にも定期更新と PR ベースの拡充が書かれているので、引用や断定に使うときは対象ファイルの日付や文脈を別途確認した方が安全です。

まとめ

system_prompts_leaks は、主要 AI 製品の内部指示を横断して追える観測拠点として価値が高いリポジトリです。2026-04-03 の GitHub Trending 3 位に入っているのも、AI の振る舞いを可視化したい需要の強さをそのまま表しています。
