これは何か

Google の最新画像生成モデル「Nano Banana 2」（技術名: Gemini 3.1 Flash Image）を、Anthropic の Claude Code や Gemini CLI から直接呼び出すための特化型スキルです。コーディング作業などのターミナル・ワークフローを中断することなく、自然言語の指示で高品質な画像資産を即座に生成することができます。

何ができるか

このスキルを導入することで、AI エージェントは以下の画像生成機能を獲得します。

  • 高解像度出力: 512px から最高 4K までの解像度指定、および自由なアスペクト比での生成。
  • 高度な画像処理: FFmpeg や ImageMagick と連携した、放送品質のグリーンスクリーン透過処理（アルファチャンネル付与）。
  • リファレンス活用: 既存の画像を参考にしたスタイル置換や、エディット指示による微調整。
  • Claude による自動制御: あなたが「～の画像を作って」と頼むだけで、Claude が最適なプロンプトを自動構成し、ツールを実行します。

目立つポイント

最大の利点は「ブラウザや専用の生成アプリに切り替える必要がない」という点です。特に Nano Banana 2 の低遅延・高精度な特性をターミナル上に引き込むことで、UI モックアップの作成、ドキュメント用の図解生成、プロトタイプ用の素材作成などが、会話の延長線上で完結します。エンジニアの「集中力を削がない」クリエイティブ・ツールと言えます。

セットアップや使い方の流れ

  1. リポジトリをクローンし、bun install と bun link で CLI をセットアップします。
  2. Google AI Studio から Gemini API キーを取得し、環境変数に設定します。
  3. Claude Code を起動し、このスキルをプロジェクトに導入（/init または設定ファイルの編集）します。
  4. 「ダークモードのダッシュボード UI を 16:9 で生成して」と Claude に指示を出します。

どんな人向けか

  • Claude Code を中心としたターミナル完結の開発環境を構築しているエンジニア。
  • 作成中のアプリの素材やモックアップを、プロンプト一つで素早く用意したいデザイナー・デベロッパー。
  • Google の最新画像生成 AI モデルのポテンシャルを、最も効率的な形で活用したい人。

注意点

動作には Bun ランタイム、および高度な編集機能を利用する場合は FFmpeg/ImageMagick のインストールが必要です。生成には Google Gemini API のトークンを消費します。

まとめ

nano-banana-2-skill は、AI コーディングにおける表現力の幅を大きく広げるツールです。「文字」だけでなく「視覚情報」を AI と共に自在に生成できる環境を作ることで、あなたのプロダクト開発はより豊かでスピード感のあるものになるでしょう。
