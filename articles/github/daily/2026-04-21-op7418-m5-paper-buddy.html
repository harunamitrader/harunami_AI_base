これは何か

m5-paper-buddy は、電子ペーパー端末「M5Paper V1.1」を、Anthropic の AI コーディングエージェント「Claude Code」専用の物理コンパニオン・デバイスに変えるユニークなプロジェクトです。ターミナルの中だけに存在する AI エージェントを物理デバイスと連携させ、手元でデバッグ状況を確認したり、物理ボタンでコマンド実行の承認を行ったりすることを可能にします。

何ができるか

現在実行中の Claude Code セッション情報を M5Paper の画面にリアルタイムでミラーリングできます。プロジェクト名、ブランチ名、コンテキストの使用量、直近のログなどを一目で把握できるほか、AI が Bash コマンドの実行やファイルの編集を求めてきた際、その内容（diff 等）を M5Paper の大画面に表示し、端末の物理ボタンやタッチ操作で「承認（YES）」または「拒否（NO）」を指示できるのが最大の特徴です。BLE（Bluetooth）と USB の両方に対応しています。

目立つポイント

単なる「外部モニター」ではなく、エージェントとの「対話の物理化」を実現している点が非常にユニークです。キーボードを打つ手を止めずに、手元の M5Paper をタップして AI の質問に答えたり、実行を許可したりする体験は、AI コーディングにおける新しいワークフローを感じさせます。また、電子ペーパーの特性を活かした高い視認性と、日本語（CJK）表示への標準対応により、日本のエンジニアにとっても実用性の高いツールとなっています。

セットアップや使い方の流れ

M5Paper 側に専用のファームウェアを書き込み、ホスト PC 側で Claude Code のプラグインとして /buddy-install コマンドを実行することで導入できます。一度セットアップすれば、ターミナルで Claude Code を起動するたびに M5Paper が起動し、自動的にセッションを追跡し始めます。複数のセッションを開いている場合でも、画面へのタッチで簡単に切り替えることが可能です。

どんな人向けか

Claude Code をメインのコーディングパートナーとして活用しており、デスクトップ環境をより機能的、かつガジェット的に面白くしたいエンジニアに最適です。画面を占有しがちな AI の実行ログや承認待ちの情報を物理デバイスに追い出すことで、メインディスプレイを最大限に活用したいと考えている方にとって、最高の「相棒」となるでしょう。

注意点

現時点ではハードウェアとして「M5Paper V1.1」に特化しており、他の M5Stack シリーズやマイコンボードではそのまま動作しません。また、Claude Code 本体の仕様変更に追従するためのアップデートが必要になる場合がありますが、開発が非常に活発であり、プラグイン形式で提供されているため、メンテナンス性は考慮されています。

まとめ

m5-paper-buddy は、ソフトウェアとしてのエージェントに「身体性」を与える、非常に興味深い試みです。情報の可視化に留まらず、AI のアクションを「物理的にグリップする」感覚は、自律型 AI との共同作業をより直感的で信頼できるものにします。デスクの上に電子ペーパーの相棒を置き、次世代のコーディング体験を楽しんでみてください。
