これは何か

mvanhorn/last30days-skill は、AI エージェント（Claude Code 等）がインターネット上の広範なソースを巡回し、特定のトピックについて直近 30 日間の「新鮮な情報」だけを抽出・要約するための専用スキルです。

何ができるか

Reddit、X（旧 Twitter）、YouTube、Hacker News、TikTok などの主要プラットフォームからデータを収集し、それらを一つの物語（ナラティブ）としてまとめ上げます。YouTube の再生回数考慮やトランスクリプトの自動取得、競合比較を行う「X vs Y」モードなど、実用性の高いリサーチ機能が目白押しです。

目立つポイント

単なる検索要約ではなく、各主張にしっかりとした引用（ソースリンク）を付与する点が強力です。また、ウォッチリスト機能により、競合他社の動向や特定の人物の活動を定期的に自動チェックする「自分専用のニュースレター」のような運用も可能です。

セットアップや使い方の流れ

Python ベースで構築されており、環境構築後に API キーなどを設定して使用します。npx last30days-skill research "トピック名" のように実行すると、数分で Markdown 形式のレポートが ~/Documents/Last30Days/ フォルダに保存されます。

どんな人向けか

トレンド追従が欠かせないデジタルマーケターや、競合調査を頻繁に行うリサーチャー、あるいは日々膨大な情報の海から重要なシグナルを見落としたくない全ての知識労働者に適しています。

注意点

SNS 等のスクレイピング制限や API 制限に依存するため、安定した動作には適切なプロキシ設定やプロバイダーの環境が必要になる場合があります。また、YouTube の完全な解析には yt-dlp などの外部ライブラリも必要です。

まとめ

情報の「新しさ」に特化し、複数のプラットフォームを串刺しにして AI に推論させるこのスキルは、AI を単なるチャット相手から、自律的なリサーチャーへとアップグレードさせる強力な武器となります。
