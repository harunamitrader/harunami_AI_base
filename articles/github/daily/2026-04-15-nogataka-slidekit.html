これは何か

SlideKit は、Claude Code や Cline、Grok CLI といった AI エージェントが「スライド作成」を自律的に行えるように設計されたスキル・ユーティリティ・キットです。 従来、AI にスライドを作らせるには構成案を出させてから手動で貼り付ける等の手間がありましたが、SlideKit を導入することで、構成の検討からレイアウト決定、画像生成、そして最終的なパワポ（pptx）出力までを、対話インターフェース上から一貫して実行可能になります。

何ができるか

  • スライド構成の自動生成: /slidekit-create スキルにより、トピックから論理的なスライド構成を提案。
  • テンプレートの適用: 11 種類以上の標準テンプレートから最適なデザインを選択。
  • 画像素材の自動生成: /imgen により、コンテキストに合わせた図解やイメージを AI が生成。
  • PowerPoint 形式への書き出し: /pptx により、その場で編集可能な pptx ファイルとして保存。
  • セッション管理: 進行中のスライド作成状況を /slidekit-status で常に把握。

目立つポイント

最大の特徴は、AI エージェントが「自ら道具を使い、成果物を仕上げる」ことに特化している点です。 Marp（Markdown ベースのスライド作成ツール）を活用しており、AI にとって扱いやすい Markdown 形式を中間言語として、最終的にプロフェッショナルなプレゼン資料へと変換します。 特にエンジニアが技術解説スライドを爆速で作るためのワークフローとして非常に優れています。

セットアップや使い方の流れ

Claude Code の場合、以下のコマンドで即座に導入可能です。 claude install-skill https://github.com/nogataka/SlideKit インストール後は、以下のようなプロンプトを投げるだけで作業が始まります。 「最新の AI エージェントの動向について、5 枚のスライドを作って。画像も適宜入れて。」

どんな人向けか

日常的に Claude Code や Cline を使いこなしており、資料作成の時間を劇的に削減したいエンジニアやプロジェクトマネージャーに最適です。 また、勉強会の資料配布や社内報告など、スピードが求められるシーンでの活用が期待されます。

注意点

現時点ではスター数からも分かる通り、非常に新しいプロジェクトです。 画像生成等には別途 API キーが必要な場合があり、AI エージェントの権限設定によっては、ローカルでの Marp CLI の動作を確認しておく必要があります。

まとめ

nogataka/SlideKit は、AI エージェントを単なる「相談役」から「実行役」へと進化させる、極めて実用的かつ野心的なツールキットです。 資料作成というクリエイティブかつ事務的なタスクを AI に任せきりにできる未来を、今すぐ手元で体験できるでしょう。
