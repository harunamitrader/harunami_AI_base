OpenRouter Agent TUI (create-agent-tui)

AIエージェントのための「create-react-app」。OpenRouterが提供する、カスタマイズ可能なターミナルUI（TUI）付きエージェント・プロジェクトを瞬時に生成するツール。

これは何か

OpenRouter チームが公開した、AI エージェント開発のためのスキャフォールディング（足場作り）ツールです。React 開発における create-react-app のように、数コマンドで「独自の TUI（ターミナル UI）」を備えたエージェント・プロジェクトの雛形を生成します。Claude Code や Cursor といった既存のエージェントに「新しいエージェントを作らせる」ためのスキルとしても機能します。

何ができるか

このツールを使用すると、TypeScript ベースの完全なエージェント・プロジェクトが作成されます。生成されるプロジェクトには、ストリーミング回答、セッション保存、トークン計算、そして高度にカスタマイズ可能なターミナル表示ロジックが最初から組み込まれています。開発者は、エージェントに持たせたい独自のツール（API連携やデータベース操作など）を実装することだけに集中できます。

目立つポイント

UI の「表現力」へのこだわりが最大の特徴です。ツール呼び出しの表示スタイルを「Emoji（絵文字のみ）」「Grouped（ツリー形式）」「Minimal（一行）」から選択できたり、ターミナルの背景色に合わせた半透明の入力ボックス、グラデーションのローディングアニメーション、さらにはプロジェクト名に合わせた ASCII アートのロゴ表示など、エンジニアの所有欲を満たすリッチな TUI を簡単に構築できます。

セットアップや使い方の流れ

OpenRouter のスキルプラグインを導入済みの Claude Code 等で /understand-build-tui（または相当するコマンド）を実行するか、CLI から npx skills add OpenRouterTeam/skills を通じて利用可能です。対話形式で必要な機能（ファイル操作、シェル実行、Web検索など）を選択するだけで、依存関係が整った実行可能なプロジェクトディレクトリが生成されます。

どんな人向けか

汎用の AI エージェントではなく、特定のタスクやドメイン（自社システムやプライベートなAPI）に特化した「自分たちのエージェント」を構築したい開発者に最適です。また、製品のデモ用に見栄えの良いターミナル UI を素早く作りたい場合や、エージェントのループ構造（ツール実行やコスト制限のロジック）を深く学びたい学習者にも向いています。

注意点

生成されたプロジェクトは OpenRouter の @openrouter/agent ライブラリをベースにしており、モデルの呼び出しには OpenRouter の API キーが必要です。また、現在は TypeScript 環境が前提となっており、Node.js v18 以上が必要です。

まとめ

エージェント開発において、UI や周辺のインフラ部分（設定、セッション、ツール呼び出しの可視化）は毎回似たような実装になりがちです。create-agent-tui はその「車輪の再発明」を終わらせ、誰もが「マエストロ（指揮者）」として、美しく機能的なエージェント・インターフェースを即座に手に入れることを可能にします。
