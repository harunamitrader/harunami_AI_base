これは何か

TradingAgents-CN は、TradingAgents 系のマルチエージェント投資分析フレームワークを中国語圏ユーザー向けに強化した派生版です。README では学習・研究用途の多智能体股票分析平台と説明されており、A 股対応、中国語ローカライズ、多モデル接続が前面に出ています。

何ができるか

複数エージェントによる市場分析、ニュース調査、レポート生成、モデル切り替え、カスタム endpoint 接続、Docker 配備、レポート出力までをまとめて扱えます。単一モデルに全部任せるのではなく、役割分担した agent 群で分析フローを組み立てるタイプです。

目立つポイント

目立つのは、中国市場データと中国語運用へかなり寄せている点です。README では complete A-share support、Google AI integration、native OpenAI support、model selection persistence などが並んでおり、単なる翻訳版ではなく、実際の運用文脈を調整したフォークに近い印象です。

セットアップや使い方の流れ

リポジトリ構成を見ると、backend、frontend、docker、web、reports、cli が分かれており、ローカル検証だけでなく self-host 前提の作りです。README でも Docker 系ファイルが整っており、まずは `.env.example` を元に設定し、Docker か Python 環境で立ち上げて、多モデル接続を設定する流れになります。

どんな人向けか

中国市場を対象に、LLM とマルチエージェントで調査フローを作りたい人に向いています。特に個人学習や研究用途で「ニュース、銘柄、レポート生成をまとめて回したい」という人には使いやすそうです。

注意点

README には学習・研究用途であり、実盤の売買指示を出すものではないと明記されています。またライセンスは混合型で、`app/` と `frontend/` は商用利用に別ライセンスが必要です。実務導入を考えるなら、そこは先に確認した方が安全です。

まとめ

TradingAgents-CN は、中国語圏・中国市場向けにマルチエージェント投資分析を現実寄りに寄せたフレームワークです。A 股や中国語 UI を前提に agent 型の分析環境を組みたいなら、かなり参考になる repo です。
