これは何か

Andrej Karpathy氏が提唱した「autoresearch（自律研究）」の概念を、暗号資産の量化（クオンツ）トレード戦略の探索に応用した実験的なプロジェクトです。FreqTradeのバックテスト環境と組み合わせ、LLMエージェントが自立的に取引戦略のコードを書き、テストし、結果を評価して改善するループを構築します。

何ができるか

AIエージェント（Claude CodeやCursorなど）にこのリポジトリを与えると、自動でバックテスト（run.py）を実行し、Sharpeレシオなどの結果を確認しながら、戦略の進化、フォーク、あるいは破棄を自律的に判断します。何百回もの実験ループを人手の介入なしに回し続けることが可能です。

目立つポイント

AIエージェントの行動を固定のオーケストレーターではなく、自然言語で記述されたprogram.mdというプロンプトファイルで制御している点がユニークです。また、Git操作（commitやreset）をエージェント自身に行わせることで試行錯誤を記録しつつ、results.tsvでイベントログを保持する堅牢な設計となっています。

セットアップや使い方の流れ

Python 3.11以上とuv、そしてTA-LibのCライブラリが必要です。uv syncで依存関係をインストール後、prepare.pyでBinanceから過去データをダウンロードします。その後、AIエージェントを起動し、「program.mdを見て新しい実験を開始して」と指示するだけで自律ループがスタートします。

どんな人向けか

AIエージェントによるコード生成や自律的な研究ループの実装に興味がある開発者、およびシステマティックトレードの新しいアプローチを模索しているクオンツ・リサーチャー。

注意点

本プロジェクトは「AIが自律研究ループを回せるか」を検証するためのプロトタイプであり、実際の資金を運用するための投資推奨ではありません。過学習（ハルシネーションの一種）が含まれる可能性もあるため、結果の解釈には十分な注意が必要です。

まとめ

Auto-Quantは、LLMエージェントを単なるプログラマーではなく、仮説検証を繰り返す「研究者」として機能させるためのミニマルな基盤です。トレード戦略の探索という具体的なタスクを通じて、未来のAI駆動型リサーチの可能性を提示しています。
