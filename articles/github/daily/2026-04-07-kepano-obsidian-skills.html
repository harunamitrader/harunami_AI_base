リポジトリ概要

Obsidian Skills は、Obsidian の CEO である Steph Ango 氏が公開した、AI コーディングエージェント用の命令セットとツール（SKILL.md）です。Claude Code や Codex などのエージェントに、Obsidian 特有の構文や機能を正しく扱わせるための「知恵」を与えます。

主な特徴と機能

  • Obsidian Markdown 対応: ウィキリンク（[[link]]）やコールアウト、タグなどの Obsidian 特有の構文を、AI が崩さずに編集できるようにします。
  • Bases (Database) 操作: 最新の Obsidian Bases 機能を AI が理解し、データのフィルタリングや集計、構造変更を行えるようにします。
  • JSON Canvas 操作: 視覚的な思考ツールである Obsidian Canvas を AI が読み取り、ノードの追加や接続の整理を行えるようにします。
  • defuddle 統合: ウェブページからクリーンな Markdown を抽出するツールが含まれており、AI のトークン消費を抑えながら情報を収集できます。

最近のアップデート (2026年4月)

Claude Code のプラグインマーケットプレイスへの対応が完了し、設定一つでこれらのスキルをエージェントに読み込ませることが可能になりました。また、OpenCode などの新しい標準エージェントプロトコルにも準拠しています。

導入と活用方法

  1. リポジトリの内容をローカルの Obsidian ボルトの特定のディレクトリに配置します。
  2. お使いの AI エージェント（Claude Code 等）に、これらのスキルファイルを読み込ませるよう設定します。
  3. 「ボルト内の未整理のメモを Bases にまとめて」といった、Obsidian の機能を駆使した高度な指示が可能になります。

まとめ

個人の知識を蓄積する Obsidian と、高い処理能力を持つ AI エージェントを直結する Obsidian Skills は、ナレッジマネジメントの未来を象徴するプロジェクトです。AI はもはや外部の知性ではなく、あなたの思考の延長線上にあるパートナーとなるでしょう。
