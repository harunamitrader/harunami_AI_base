これは何か

council-of-high-intelligence は、歴史的な哲学者、科学者、そして現代のテックリーダーなど18名の人格（AI Persona）からなる「最高知能評議会」を召集し、あなたの複雑な意思決定を審議させるためのツールです。 単一の AI に答えを求めるのではなく、複数の LLM プロバイダーを横断して異なる知性を戦わせることで、バイアスのない深い洞察を得ることを目的としています。

何ができるか

  • 多段階審議プロトコル： 独立分析 → 相互反対尋問 → 最終結論という3段階のプロセスを経て議論を深めます。
  • 多様なプロバイダーの活用： 反応の均質化を防ぐため、評議会員を Claude, OpenAI, Gemini, Ollama(ローカル) など異なるプロバイダーへ自動的に振り分けます。
  • 18名の評議会員： アリストテレス、ファインマン、カーネマン、リーナス・トーバルズ、宮本武蔵、アンドレイ・カーパシーなど、古今東西の知性が揃っています。

目立つポイント

最大の特徴は、**「多層的な反対意見（ディセント）を強制的に生成させる仕組み」**です。 「ソクラテス vs ファインマン」といった対立軸を持つペア（Polarity Pairs）での議論や、意見の一致が早すぎる場合に反対意見を強制する「反対意見ノルマ」機能を搭載。 これにより、単一モデルの自信満々な回答（ハルシネーション含む）を多角的に検証し、隠れたリスクを表面化させます。

セットアップや使い方の流れ

リポジトリをクローンし、./install.sh を実行して Claude Code や Codex にインストールします。 /council [内容] を入力すると審議が始まります。徹底審議のフルモード、短時間の判断のための --quick、2名での対立軸を探る --duo モードが選べます。

どんな人向けか

重要なビジネス戦略、技術選定、あるいは倫理的なジレンマに直面しているリーダーに最適です。 単一の AI の「尤もらしい回答」ではなく、あえて「批判的な視点」や「多様な代替案」を求めている人にとって、非常に強力な意思決定支援ツールとなるでしょう。

注意点

全ポテンシャルを引き出すには、主要な LLM (Claude, OpenAI, Gemini 等) の API キーが設定されていることが望ましいです。 また、複数名のエージェントが対話を繰り返すため、単一プロンプトよりもトークン消費量（コスト）が多くなる点に留意が必要です。

まとめ

この「最高知能評議会」は、AI を単なる『回答者』から、自らの思考を磨き上げるための『知的パートナー』へとアップグレードさせるツールです。 歴史上の賢人たちが現代の課題について激論を交わす体験は、実利的であると同時に、AI エージェントの新しい活用形を示唆しています。
