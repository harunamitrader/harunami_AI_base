リポジトリの概要

「このユーザー名は他のサイトでも使われているか？」という疑問に、一瞬で答えてくれるのが Python 製 OSINT ツール「Sherlock」です。5万スターを超えてなお進化を続ける、公開情報調査（Open Source Intelligence）のデファクトツールです。

注目ポイント

  • 圧倒的な網羅性: 400 サイト以上（カスタマイズでさらに数千）を対象としたユーザー名検索。
  • 並列実行による高速化: asyncio を活用し、数分以内に主要な SNS/Web サービスの存在確認を完結。
  • 信頼性の追求: 単なる HTTP ステータスコードの確認だけでなく、コンテンツベースのチェックで誤検知を最小限に。

まとめ

セキュリティリサーチやデジタルフォレンジック、そして個人のアイデンティティ管理においても、Sherlock は強力なパートナーとなります。
