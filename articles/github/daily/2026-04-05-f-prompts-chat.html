これは何か

prompts.chat は、もともと Awesome ChatGPT Prompts として知られていた prompt collection を、公開サイト、データセット、self-host 可能な web app まで拡張した project です。README では「世界最大級の open-source prompt library」として位置づけられています。

何ができるか

prompt の閲覧だけでなく、CSV や markdown、Hugging Face dataset として再利用でき、投稿した prompt が GitHub 側へ同期される運用も用意されています。さらに CLI、Claude Code plugin、remote / local MCP server も提供しており、prompt 集をそのまま agent workflow に組み込めます。

目立つポイント

package.json を見ると Next.js 16、Prisma、NextAuth、MDX、MCP SDK を使う構成で、単なる静的リストではなく application として作り込まれています。self-host setup、認証、branding、Docker、book PDF 生成まで含めており、「prompt を配るサイト」から一段進んだ運用基盤です。

セットアップや使い方の流れ

最低限なら公開サイトや PROMPTS.md をそのまま使えます。自前運用する場合は npx prompts.chat new ... か repo clone 後に npm install と npm run setup を進め、必要に応じて MCP や plugin を接続する流れです。

どんな人向けか / 注意点

社内 prompt library を育てたいチーム、公開 prompt を dataset や MCP として再利用したい人、AI 利用ルールを共有したい教育用途に向いています。一方で stars 数の大きさに比べると対象範囲が広く、prompt 品質は利用者側の選別が要るので、そのまま本番投入するより curated 運用の前提で見る方が妥当です。

まとめ

prompts.chat は、prompt engineering の知識集を asset 化し、そのまま product や team workflow に接続できるところが強みです。2026-04-05 の Trending Rank 6 は、prompt library 需要がまだ単発の参考資料ではなく運用基盤として伸びていることを示しています。
