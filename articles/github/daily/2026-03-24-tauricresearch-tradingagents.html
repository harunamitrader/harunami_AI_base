これは何か

TradingAgents は、金融判断を複数の専門エージェントに分業させるマルチエージェント framework です。 ひとつのモデルに全部やらせるのではなく、役割を分けた上で最終判断を組み立てる設計になっています。

何ができるか

README では、ファンダメンタル、センチメント、テクニカル、リスク管理、トレーダー役のような構成が紹介されています。 つまり、投資判断を会議体のように分解して扱うのが基本思想です。

目立つポイント

面白いのは、マルチエージェントという言葉を抽象論ではなく、金融判断フローにかなり素直に落としている点です。 役割の分離がそのまま説明可能性にもつながるので、題材との相性が良いです。

セットアップや使い方の流れ

README では Python 3.13 前提で clone 後に `pip install .` を行い、CLI か Python API から使う流れです。 研究用途として触る入口はそこまで重くありません。

どんな人向けか

マルチエージェント構成を金融ドメインで試したい人にはかなりわかりやすい教材です。 逆に、実売買でそのまま使う前提で見ると、考えるべきことが一気に増えます。

注意点

README 側でも研究用途であり投資助言ではないことが明示されています。 マルチエージェント化しても、市場データ品質、遅延、過学習、責任範囲の問題は消えません。

まとめ

TradingAgents は、投資判断を役割分担で回す構造をそのまま framework 化した repo でした。 マルチエージェントの具体例として見やすく、Trending に入るのも自然です。
