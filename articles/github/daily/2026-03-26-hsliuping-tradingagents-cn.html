これは何か

TradingAgents-CN は、プロの投資会社の意思決定プロセスを「複数の AI エージェント」で再現する金融リサーチ・トレードフレームワークです。特に中国市場（A株、香港株）への最適化が施されており、複雑な市場データの解析を自動化します。

何ができるか

市場アナリスト、テクニカル調査員、ファンダメンタル調査員、ニュース監視役、そしてリスクマネージャーなど、異なる役割を持つエージェントがチームを組んで協議します。各エージェントは自らの専門領域に基づき意見を出し合い、最終的に「弱気と強気の両陣営がディベートする」といったプロセスを経て、客観性の高い投資レポートを生成します。

目立つポイント

中国国内の有力モデル（Alibaba 系の BaiLian や DeepSeek 等）と OpenAI/Gemini 等の海外モデルを柔軟に使い分ける「ハイブリッド構成」に対応しています。データのクレンジングから、テクニカル分析、センチメント分析、そして PDF/Markdown 形式でのプロフェッショナルなレポート出力までを一貫して自動完了する完成度の高さが特徴です。

セットアップや使い方の流れ

Docker を用いたコンテナ環境での導入が推奨されています。環境変数に各種 LLM の API キーを設定し、対象銘柄を指定してコマンドを実行するだけで、数分から数十分かけて多角的な市場レポートを書き上げます。

どんな人向けか

体系的なマルチエージェント・トレード理論を学びたいリサーチャーや、中国市場の膨大なニュース・財務データを AI で効率的にフィルタリングしたい個人投資家、企業の投資部門に適しています。

注意点

本プロジェクトは「投資の学習と研究」を目的としており、自動的に取引注文を実行するものではありません。生成されるレポートやシグナルはあくまで参考情報であり、最終的な判断には人間の検証と法令遵守が不可欠です。

まとめ

金融という極めてノイズの多い領域において、役割分担されたエージェントの「合議制」で精度を上げる TradingAgents-CN のアプローチは、AI による投資の未来を予感させる実務的な試みです。
