これは何か

金融市場のデータ分析とリサーチに特化した、自律型の AI エージェントです。「考える、計画する、学ぶ」というプロセスを繰り返し、複雑な金融関連の問い合わせ（財務諸表の分析など）に対して、正確な答えを導き出すように設計されています。

何ができるか

複雑な質問を、まず具体的なステップに分け（タスクプランニング）、市場データを検索・解析し、その結果が正しいか自分自身で振り返る（自己リフレクション）ことで、精度の高い回答を作成します。損益計算書、貸借対照表、キャッシュフロー計算書などの財務情報を直接取得して解析することが可能です。

目立つポイント

リアルタイムの市場データにアクセスできる点と、WhatsApp 連携機能によりモバイルアプリ経由でリサーチを指示・受信できる点が特徴的です。また、無限ループを検知して停止する安全機能も備わっており、安心して動かし続けることができます。

セットアップや使い方の流れ

Bun ランタイム環境で動作します。リポジトリをクローンした後、OpenAI や金融データ提供サービス（Financial Datasets 等）の API キーを設定します。後は bun start で対話モードを起動し、リサーチしたいトピックを入力するだけで完了です。

どんな人向けか

投資判断や企業分析を効率化したいと考える金融アナリスト、個人投資家、企業の財務担当者など、正確なデータに基づいた深いリサーチを必要とする人に最適です。

注意点

高度な金融データの取得には外部 API （Financial Datasets AI など）のキーが必要になります。無料枠があるものも含め、いくつかのサービスに登録する必要があります。

まとめ

dexter は、金融という専門性の高い領域において、人間のアナリストのような論理思考とデータ収集を代行してくれる強力なツールです。「Claude Code の金融特化版」のような利便性を備え、高度な金融分析を自動化するための有力な選択肢となるでしょう。
