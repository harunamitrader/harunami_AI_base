プロジェクトの概要

Kronos は、金融市場の K-line、つまりローソク足系列に特化した foundation model です。 45 以上のグローバル取引所データで事前学習され、OHLCV のような多次元時系列をトークン化して Transformer で扱う構成を取っています。

機能・できること

  • 市場系列のトークン化: 連続値のローソク足データを階層的な離散トークンへ変換します。
  • 予測 API: KronosPredictor を使い、過去の OHLCV から将来期間の系列を生成できます。
  • モデルサイズの選択: mini、small、base などの事前学習済みモデルが Hugging Face Hub で提供されています。
  • ファインチューニング: 独自データや A-share 市場を例にした追加学習パイプラインが用意されています。

リポジトリの構成

model/ にモデル実装、examples/ に予測例、finetune/ と finetune_csv/ に追加学習用のコードが置かれています。 webui/ もあり、研究コードだけでなくデモや実験環境も含む構成です。

セットアップ・導入のイメージ

Python 3.10 以降で依存関係を入れ、Hugging Face Hub から tokenizer と model を読み込む流れです。 入力 DataFrame には open、high、low、close が必要で、volume と amount は任意です。

  1. pip install -r requirements.txt で環境を作ります。
  2. KronosTokenizer と Kronos を Hugging Face Hub から読み込みます。
  3. KronosPredictor.predict() で予測期間の OHLCV 系列を生成します。

ターゲット層 / 注目ポイント

金融時系列を扱う研究者、クオンツ、トレード分析者に向いています。 汎用の時系列モデルではなく、ローソク足の構造に寄せた tokenizer と事前学習を持つ点が特徴です。

制約・リスク

リポジトリにも、ファインチューニング例は本番用の取引戦略ではないと明記されています。市場予測はノイズが大きく、実運用ではリスク管理、検証、手数料、スリッページを別途設計する必要があります。

まとめ

Kronos は、金融市場を自然言語のような系列モデリング対象として扱う実験的かつ実用志向のプロジェクトです。 投資判断そのものではなく、市場データを読むモデル基盤として評価すると、注目される理由が見えます。
