プロジェクトの概要

Anthropic 社が提供するターミナルベースのAIエージェント「Claude Code」に対して、Android アプリ（APK / XAPK / JAR / AAR）のリバースエンジニアリング能力を付与するプラグイン（スキル）です。 AIに対して自然言語やスラッシュコマンド（/decompile）で指示するだけで、ローカルのデコンパイラ群を駆使し、アプリの内部構造を解析してくれます。

機能・できること

  • 自動デコンパイル: jadx や Fernflower/Vineflower といった強力なデコンパイラを Claude Code の中からシームレスに呼び出します。
  • API 及び通信フローの抽出: Retrofit のエンドポイント、OkHttp のコール履歴、ハードコードされた URL、あるいは認証トークンやヘッダー情報を自動的に見つけ出しドキュメント化します。
  • アーキテクチャ解析: Activity や Fragment から ViewModel を経由して HTTP コールに至るまでの「コールフロー」を追跡し、コードのない状態からアプリの動きを再現・解析するのを助けます。

リポジトリの構成

Claude Code 向けの .claude-plugin マニフェストや、AIへのシステムプロンプト的な役割を果たす SKILL.md、そしてデコンパイラを動かす実体となるいくつかのシェルスクリプトで構成されています。

セットアップ・導入のイメージ

ローカルの PC に Java 17+ と jadx がインストールされている必要があります。 その上で、Claude Code のプロンプトから /plugin marketplace add SimoneAvogadro/android-reverse-engineering-skill コマンドを実行するだけで、以降のすべてのセッションでこのスキルが永続的に利用可能となります。

ターゲット層 / 注目ポイント

セキュリティリサーチャーやマルウェア・アナリスト、あるいは他社アプリのAPI構造を研究（互換層の実装など）したい開発者に非常におすすめです。「複雑なリバースエンジニアリングツールの出力を、自然言語で統合・解釈する」というAIエージェントの強みを最大限に活かした拡張機能です。

制約・リスク

他者のアプリのリバースエンジニアリングは、各国の著作権法や各サービスの利用規約（TOS）に違反する可能性があります。セキュリティ研究や法的例外（互換性の確保など）の範囲内でのみ利用等、自己責任が伴います。

まとめ

コード生成や開発補助にとどまりがちな AI コーディングアシスタントを「強力な解析ツール」へと瞬時に変貌させる、非常に専門的で興味深い拡張エコシステムの一例です。
