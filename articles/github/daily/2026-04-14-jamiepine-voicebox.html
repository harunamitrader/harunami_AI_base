プロジェクトの概要

voicebox は、音声クローン、音声生成、後処理エフェクト、複数話者のタイムライン編集をローカル環境で扱うための音声合成スタジオです。 README では ElevenLabs のオープンソース代替として説明されており、モデルと音声データを手元に置いたまま作業できる点を前面に出しています。

機能・できること

  • 複数 TTS エンジン: Qwen3-TTS、LuxTTS、Chatterbox、Chatterbox Turbo、TADA を切り替えて使えます。
  • 多言語音声: 23言語対応のエンジンを含み、日本語やアラビア語、ヒンディー語なども対象に入っています。
  • 音声編集: ピッチ、リバーブ、ディレイ、コーラス、コンプレッサー、フィルターなどの後処理を適用できます。
  • 長文生成: 長いテキストを文単位に分割し、各チャンクを生成してクロスフェードでつなげます。
  • API 利用: ローカル REST API から音声生成や voice profile 操作を呼び出せます。

リポジトリの構成

デスクトップアプリは Tauri / Rust、フロントエンドは React / TypeScript / Tailwind CSS、バックエンドは Python FastAPI で構成されています。 主要ディレクトリは app/、tauri/、web/、backend/、landing/、scripts/ です。

セットアップ・導入のイメージ

macOS と Windows には配布バイナリが用意され、Docker では docker compose up で起動できます。 開発環境では Bun、Rust、Python 3.11 以上、Tauri の前提環境を揃え、just setup と just dev でバックエンドとデスクトップアプリを起動する流れです。

ターゲット層 / 注目ポイント

音声アプリ、ゲーム音声、ポッドキャスト制作、アクセシビリティツール、音声アシスタントを自前環境で組みたい開発者に向いています。 生成だけでなく、voice profile、バージョン管理、非同期キュー、GPU 対応、REST API までそろっているため、実験用 UI ではなく小さな制作基盤として見られます。

制約・リスク

ローカル実行型のため、GPU、OS、依存ライブラリの相性が品質と速度に直結します。音声クローンは権利・同意・なりすましリスクも大きいため、公開利用や商用利用ではライセンスと利用許諾を明確にする必要があります。

まとめ

voicebox は、ローカルファーストの音声合成環境を、デスクトップ UI、編集機能、API、モデル管理まで含めてまとめた実用寄りのプロジェクトです。 音声生成を外部 SaaS だけに依存せず、制作フローやアプリに組み込みたい場合に検討しやすいリポジトリです。
