これは何か

Everything Claude Code は、Claude Code 向けの config 集を超えて、agent harness performance optimization system を目指した大規模 repo です。README でも skills、instincts、memory、security、research-first development を含む一式として説明されています。

何ができるか

plugin、agents、skills、hooks、rules、MCP config、legacy command shims をまとめて導入し、計画、検証、セキュリティ、学習、multi-agent orchestration まで広く扱えます。しかも Claude Code だけでなく Codex、Cursor、OpenCode、Antigravity まで射程に入っています。

目立つポイント

圧倒的なのはスコープです。README 時点でも数十 agents、百超 skills、multi-platform support、hook runtime controls、install profile、cross-tool parity まであり、単なる dotfiles 集よりかなり「運用基盤」寄りです。

セットアップや使い方の流れ

基本は plugin install か repo clone 後に install script を使います。Windows でも install.ps1 が用意されていて、language / target ごとの selective install ができるようになっています。

どんな人向けか

Claude Code や Codex を本格運用し、rules、hooks、skill 群をまとめて整えたい人に向いています。特に team で harness を標準化したい場合や、複数 agent tool をまたいで整えたい場合に相性が良いです。

注意点

かなり大型の repo なので、全部入りで入れる前に必要な profile や language を絞った方が安全です。README 自体も長く、思想と運用ルールを理解してから取り込む方が失敗しにくいです。

まとめ

everything-claude-code は、Claude Code 周辺の運用を極端に強化したい人向けの大規模基盤です。単発の skill より、harness 全体を設計したい人ほど刺さります。
