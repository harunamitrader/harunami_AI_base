これは何か

video-use は、人気ブラウザエージェント「browser-use」のチームが開発した、Claude Code 専用の動画編集拡張スキルです。 動画素材が入ったフォルダで Claude を起動し、「これをローンチ動画に編集して」と指示するだけで、 カット、カラーグレーディング、字幕入れ、アニメーション合成といった一連の編集工程を AI が自律的に実行します。

何ができるか

  • 不要箇所の自動カット： 「えー」「あのー」などのフィラーワード（Filler Words）や無音区間を削除。
  • 自動カラーグレーディング： FFmpeg を駆使し、シネマティックな補正やパンチのある色調への自動調整。
  • 字幕（テロップ）生成： 文脈に合わせたスタイルでの字幕焼き付け。
  • アニメーション合成： Manim や Remotion を使用した動的なオーバーレイの生成・合成。
  • 自己評価ループ： レンダリング後に AI 自らが映像を確認し、ジャンプカットやノイズを修正。

目立つポイント

画期的な点は、「AI が動画を直接観るのではなく、『読む』」というアプローチです。 ElevenLabs 等を利用した高精度なトランスクリプト（単語単位の時間情報付き）を主軸に据え、文字ベースで編集点を判断します。 これにより、数万フレームを AI に解析させるトークンの無駄を省きつつ、単語境界での精密なカットを実現しています。 必要な瞬間だけフィルムストリップ（静止画）を生成して視認性を確保する、「DOM 構造でブラウザを操る browser-use」の思想が動画編集にも適用されています。

セットアップや使い方の流れ

リポジトリをクローンし、Claude Code のスキルディレクトリ（~/.claude/skills/video-use）にシンボリックリンクを貼ることで導入できます。 別途 FFmpeg や yt-dlp などのコマンドラインツールが必要です。 導入後は、編集したい動画のあるフォルダで claude を起動し、「edit these into a launch video」といった自然言語で編集を開始します。

どんな人向けか

YouTube ショートやチュートリアル動画を頻繁に作成するクリエイター、 動画編集に時間をかけたくないエンジニア、 および AI エージェントによる「マルチモーダルなアウトプット生成」の可能性を追求したい開発者に最適です。

注意点

トランスクリプト生成に外部 API（ElevenLabs 等）を利用するため、その利用料金が発生します。 また、複雑な Manim アニメーションなどは生成に時間がかかることがあり、 最終的なレンダリング負荷はマシンの CPU/GPU 性能に依存します。

まとめ

video-use は、動画編集を「UI ポチポチ作業」から「AI との対話」へと昇華させるプロダクトです。 「編集の民主化」を AI エージェントの力で推し進める browser-use チームの野心的な試みであり、 今後の AI アシスタントが備えるべき標準機能のあり方を示唆しています。
