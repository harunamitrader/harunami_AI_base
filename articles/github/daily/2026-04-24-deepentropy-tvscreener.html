これは何か

TradingViewが提供している強力なスクリーナー機能を、PythonからAPI経由で利用するための非公式ライブラリです。株式、暗号資産、FX、債券、先物など、多岐にわたる市場データをプログラムから直接検索・フィルタリングし、PandasのDataFrame形式で取得できます。

何ができるか

13,000以上のフィールド（テクニカル指標、財務データ、推奨など）を駆使した高度なクエリが可能です。「時価総額10億ドル以上かつRSIが30以下の米国株」といった条件をPythonicな構文で記述し、リアルタイムに近い市場データを一括取得できます。また、ストリーミング機能により定期的な自動更新もサポートしています。

目立つポイント

特筆すべきは、最新のModel Context Protocol (MCP) への対応です。これにより、Claude CodeなどのAIエージェントが、このライブラリを介して直接市場データを調査できるようになります。また、GUIでのクエリ構築を助けるCode Generatorも用意されており、複雑なフィルター設定を直感的に行えます。

セットアップや使い方の流れ

pip install tvscreener でインストール可能です。MCP機能を利用する場合は pip install tvscreener[mcp] を使用します。認証なし（非ログイン状態）でも動作し、日足だけでなく分単位（5分、1時間など）のテクニカル指標にもアクセスできる点が非常に強力です。

どんな人向けか

Pythonを用いたシステムトレードや定量的分析（クオンツ）を行っている開発者、およびAIエージェントに市場調査を行わせたいリサーチエンジニア。

注意点

本ライブラリはTradingView公式のものではないため、利用規約の遵守には注意が必要です。また、データの正確性や取得制限（レートリミット）については、商用やクリティカルな用途での使用前に十分な検証が推奨されます。

まとめ

tvscreenerは、TradingViewの膨大な市場データを「Pythonのコード」と「AIの文脈」の双方にシームレスに接続する強力なブリッジです。投資リサーチの自動化や、AIエージェントによる高度な市場分析を実現するための基盤として、非常に高いポテンシャルを持っています。
