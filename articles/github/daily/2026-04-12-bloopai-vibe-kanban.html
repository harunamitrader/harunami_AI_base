これは何か

vibe-kanban は、Claude Code や Gemini CLI といった強力な CLI ベースの AI エージェントを、自律的な「チームメンバー」として運用するために設計された統合ワークスペースです。カンバン形式のタスク管理、内蔵ブラウザ、そして Git 操作の自動化を一つのインターフェースにまとめ、AI との共創を劇的に加速させます。

何ができるか

10種類以上の主要な AI エージェント（Claude Code, Codex, Gemini CLI 等）を切り替えて利用可能で、エージェントは内蔵ブラウザのスクリーンショットやコンソールを確認しながらフロントエンドの修正を行うことができます。また、修正内容に基づいたプルリクエストの作成や、タスクカードによる進捗管理も自動で行われます。

目立つポイント

AI エージェントに「視覚（ブラウザ）」と「組織的な動き（カンバン）」を与えるというコンセプトが非常に強力です。CLI 単体では把握しづらかったフロントエンドの表示崩れや実行エラーも、エージェントがブラウザを同期して確認することで、自律的な修正が可能になります。npx vibe-kanban 一発で起動できる手軽さも魅力です。

セットアップや使い方の流れ

Node.js 環境があれば、コマンド一つで即座に開始できます。

npx vibe-kanban

ブラウザで管理画面が開くので、そこでプロジェクトフォルダを指定。あとはカンバンにタスクを作って、好きなエージェント（Claude Code 等）をアサインするだけで、AI が自律的にコードを書き始めます。

どんな人向けか

最新の AI エディタや CLI エージェントを使いこなしたい、あるいは AI により大きなタスクを任せたいと考えている開発者に最適です。特にフロントエンド開発や、多数のエージェントを使い分けるプロフェッショナルな環境でその真価を発揮します。

注意点

現在は初期の開発フェーズ（0.1.x）にあるため、頻繁なアップデートとそれに伴う仕様変更が予想されます。また、AI エージェントに高度な権限を与える性質上、実行環境のセキュリティ確保と、AI の挙動を適宜レビューすることが推奨されます。

まとめ

vibe-kanban は、AI エージェントのための「OS」とも呼べる野心的なプロジェクトです。エージェントを文字通り「使い捨ての実行機」から、プロジェクトの文脈を理解し自律的に推進する「共創者」へと一段階上に引き上げてくれる、次世代の開発環境です。
