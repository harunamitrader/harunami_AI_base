これは何か

「LLMFeeder」は、ウェブサイトの閲覧中に見つけた有益な情報を、ChatGPT や Claude といった大規模言語モデル（LLM）が理解しやすい「クリーンな Markdown 形式」に瞬時に変換し、クリップボードやファイルとして抽出するためのブラウザ拡張機能です。不要な広告、ヘッダー、サイドバーを自動で除去し、主要なコンテンツのみを正確に抽出することに特化しています。

何ができるか

アクティブなタブのコンテンツを 1 クリックで Markdown 化してコピーできるほか、複数の開いているタブを一括で処理して一つの Markdown ファイルに結合したり、ZIP ファイルとしてまとめてダウンロードしたりすることが可能です。また、コピー前に対象のテキストが何トークン（GPT-4 や Claude 基準）を消費するかを計算する「リアルタイム・トークンカウンター」を搭載しており、モデルの入力制限を意識しながら情報を収集できます。

目立つポイント

「Privacy First」な設計が最大の特徴で、すべての処理はブラウザ内（クライアントサイド）で完結しており、外部サーバーへのデータ送信は一切ありません。また、AI エージェントがプロジェクトを理解しやすくするための `AGENTS.md` ドキュメントを完備しており、開発者自身が LLM と協調してこのツールを改良している背景が見える、非常にモダンなプロジェクトです。画像。URL の保持やリンクの有無なども柔軟にカスタマイズできます。

セットアップや使い方の流れ

Chrome Web Store や Firefox Add-ons から拡張機能をインストールするだけで準備完了です。あとは情報を抽出したいウェブページを開き、拡張機能のアイコンをクリックするか、右クリックメニューから「Copy as Markdown」を選択するだけです。設定画面では、Markdown の出力形式やトークン上限のしきい値を自由に調整できます。

どんな人向けか

ニュース記事や技術ドキュメント、ブログ記事などを LLM に読み込ませて要約や分析を行わせたいすべてのユーザーに最適です。特に、大量のタブを開いてリサーチを行い、それらを一つの「コンテキスト」として AI に一括投入したいと考えているエンジニアやライター、研究者にとって、これほど便利なツールはありません。

注意点

コンテンツの抽出には Readability エンジンを使用しているため、極端に複雑な構造を持つ動的な SPA（Single Page Application）などでは、意図した通りに本文が抽出されない場合があります。その場合は、右クリックによる「選択範囲のみのコピー機能」を活用することで、確実に必要な情報だけを抽出することができます。

まとめ

「LLMFeeder」は、人間と AI の対話を「情報収集」という側面から強力にバックアップする、シンプルながらも計算し尽くされた傑作ツールです。情報の鮮度が重要なリサーチ作業において、この拡張機能は AI を活用するための「最高の餌（Feeder）」となるでしょう。
