これは何か

Crucix は、27 の異なるオープンソース・インテリジェンス (OSINT) フィードからデータを収集し、単一のダッシュボードに統合して表示する、個人向けのインテリジェンス・エージェント（ターミナル）です。NASA の衛星火災検知、連邦準備制度（FRED）の経済データ、航空機や船舶の追跡、放射線監視など、世界中の多岐にわたるリアルタイムデータを「自分専用の監視基地」のように一元管理できます。

何ができるか

  • 並列データ取得: 27 のソースから 15 分ごとに自動更新。
  • ライブダッシュボード: 地図ベースの直感的な UI で情報を可視化。
  • 多機能通知: Telegram や Discord と連携し、異常検知やリモート操作が可能。
  • AI インサイト: Claude, GPT, Gemini 等の LLM を活用し、情報の相関分析やレポート生成が可能。
  • 柔軟な動作モード: 処理速度やレート制限に応じた 3 つのパフォーマンス設定（BALANCED, TURBO, QUIET）。

目立つポイント

最大の魅力は、完全ローカルで動作しプライバシーを重視しながら、圧倒的な情報密度を手に入れられる点です。経済指標から物理的な災害、武力衝突までを網羅しており、単なる監視にとどまらず、AI による「次に何をすべきか」の提案機能（トレード案の生成など）も備えています。自分だけの「デジタル司令センター」を構築できる、非常にギークなツールです。

セットアップや使い方の流れ

1. インストール: リポジトリをクローンし、npm install で依存関係を導入します。

2. 環境設定: .env ファイルを作成し、FRED、NASA、EIA 等の無料 API キーを設定します。

3. 実行: npm run dev で起動し、ブラウザでローカルサーバーのダッシュボードを確認します。

4. 活用: 必要な通知設定（Telegram 等）を行い、リアルタイムな情報の流れを監視・分析します。

どんな人向けか

地政学リスクや経済変動を監視する投資家やアナリストに最適です。また、自宅に「司令センター」を構築したい OSINT 愛好家や、市場の歪みやリアルタイムな変化を素早く捉えたいトレーダーにとっても強力な武器となります。

注意点

同名の暗号資産トークン等とは一切無関係です。また、ADSB のフライト追跡など、一部の高度なデータソースはフル機能の利用に有料の API キーが必要な場合があります。ローカルリソースを一定以上消費するため、マシンスペックにも注意が必要です。

まとめ

Crucix は、分散したグローバルな公開情報を一つに束ね、自分だけの「インテリジェンス・センター」を構築できるツールです。情報の非対称性を解消し、迅速かつ知覚的な意思決定を支援する、まさに現代の「情報の砦」と言えるプラットフォームです。
