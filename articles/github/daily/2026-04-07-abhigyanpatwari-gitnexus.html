これは何か

GitNexus は、ブラウザ単体で動作する「ゼロ・サーバー」のコード・インテリジェンス・ツールです。リポジトリや ZIP ファイルを読み込み、コードの依存関係や構造をインタラクティブなナレッジグラフとして視覚化します。

何ができるか

  • コードの視覚化: 複雑なプロジェクト構造をグラフ形式で把握。
  • Graph RAG エージェント: グラフ解析と LLM を組み合わせた、文脈を汲み取るコード探索。
  • MCP 対応: Model Context Protocol を介して外部エージェントの知識源として統合可能。
  • 完全ローカル実行: データを外部サーバーに送信せず、ブラウザ内の IndexedDB 等で処理。

目立つポイント

セットアップが非常に簡単で、GitHub の URL を入力するだけで解析が始まる手軽さが魅力です。特に、大規模な既存コードベースの全体像を「目で見て理解」し、かつ AI に質問して詳細を探れるハイブリッドな体験を提供します。

セットアップや使い方の流れ

  1. GitNexus ウェブサイトにアクセス。
  2. 解析したい GitHub リポジトリの URL を入力、またはファイルをドロップ。
  3. 生成されたグラフを探索し、チャット画面からコードに関する質問を投げます。

どんな人向けか

  • 初めて触る巨大なコードベースの構造を短時間で把握したい開発者。
  • コードの機密性を保ちつつ、AI による解析支援を受けたいプロジェクトチーム。
  • リサーチ中にコードの依存関係を視覚的に整理したいエンジニア。

注意点

ブラウザのリソースを使用するため、極端に巨大なプロジェクト（数万ファイル規模など）ではメモリ制限やパフォーマンス低下に直面する可能性があります。また、初回インデックス作成には一定の時間がかかります。

まとめ

GitNexus は、AI 時代の新しいコードリーディング・ツールです。「サーバー不要」という強力なプライバシー特性と、グラフ構造を活用した深い洞察を両立しており、開発者の「理解のコスト」を劇的に下げてくれます。
