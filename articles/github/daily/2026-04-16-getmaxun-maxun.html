プロジェクトの概要

「Maxun」は、Webサイトのクローリング、スクレイピング、情報検索を統合したオープンソースのノーコードデータ抽出プラットフォームです。「あらゆるWebサイトを数分で構造化されたAPI（やスプレッドシート）へと変換する」ことをビジョンに掲げ、直感的なUIとAIを組み合わせているのが特徴です。

機能・できること

  • Extract (データ抽出):
    • Recorder Mode: ブラウザ上でのクリックなどのアクションを記録し、再利用可能な抽出ロボット（スクリプト）を自動生成します。
    • AI Mode: 「映画のタイトルと評価を抽出して」と自然言語で指示するだけで、LLM がページを解釈して必要なデータを構造化して取得します。
  • Scrape (スクレイプ): LLM との親和性が高い形（クリーンな Markdown や HTML）でWebページ全体を一括保存します。
  • Crawl / Search: 特定のサイトのフルクローリング、あるいは条件に基づいた自動Web検索をスケジューリングできます。
  • API & エクスポート: 抽出した一連の操作を REST API として外部へ公開したり、Google Sheets や Airtable へ直接エクスポートすることができます。

リポジトリの構成

バックエンドのサーバー・フロントエンドのダッシュボード・ブラウザ制御基盤などで構成される巨大なモノレポ（Monorepo）です。Node.js / TypeScriptエコシステムで構築されており、Docker-compose での運用を前提とした構成になっています。

セットアップ・導入のイメージ

手軽に試す場合は、公式のホスティング版アプリ（app.maxun.dev）が最速です。自社環境へデプロイする場合は、リポジトリをクローンして docker-compose up を実行することで、データベース等を全て含んだシステム一式をローカルサーバー上に展開できます。また開発者向けの SDK も提供されています。

ターゲット層 / 注目ポイント

マーケティング用のリードジェネレーション（顧客開拓）、市場調査アグリゲーションから、LLM / AI 用の学習データ、ナレッジベース構築まで幅広い用途で活躍します。ページネーションや認証が必要な裏側のサイト構造に対しても RPA 的に柔軟に対応できる点が高く評価されています。

制約・リスク

スクレイピングは対象サイトの規約（利用規約や robots.txt）に抵触する可能性があるため、適切な間隔でのアクセスや合法な範囲内での情報の取り扱いが義務付けられます。Maxun 側のプラットフォーム自体は AGPLv3 ライセンスでオープンソースとして提供されています。

まとめ

これまでエンジニアが Python + Playwright / Selenium で苦心して作成していた自動化スクリプトや画面操作の保守を、すべてGUIとLLMで代替できるようにした包括的なスクレイピング・オートメーション製品です。
