これは何か

VS Code 上で動作するオープンソースの自律型 AI コーディングエージェントです。以前は「Cline」（さらに前は Claude Dev）として知られていた人気プロジェクトが、コミュニティ主導の体制に移行し、さらなる進化を遂げた姿です。

何ができるか

単なるコード補完ではなく、ファイルシステムの操作、ターミナルの実行、ブラウザの自動化、そして MCP（Model Context Protocol）を介した外部ツールとの連携を AI が自律的に行い、開発者の指示に基づいて機能実装やデバッグを完遂します。

目立つポイント

「Code」「Architect」「Ask」「Debug」のマルチモード対応、自律的な実行サイクル、MCP サーバーによる外部スキルの統合、そしてコミュニティによる迅速なアップデートが大きな強みです。

セットアップや使い方の流れ

VS Code 拡張機能としてインストール後、各種 LLM（Claude, OpenAI, Gemini 等）の API キーを設定します。サイドバーからエージェントとチャットを開始し、プロジェクト全般の作業を依頼できます。

どんな人向けか

コーディング作業を AI に大幅に委任し、自律的なエージェントと共にプロジェクトを推進したい全ての開発者に最適です。

注意点

エージェントが自律的にファイルを書き換えるため、作業前には Git 等でのバージョン管理を徹底し、AI による変更内容を常にレビューすることを推奨します。

まとめ

Cline の正当な後継として、コミュニティの期待を背負って進化し続ける Roo-Code は、VS Code における自律型 AI エージェントのデファクトスタンダードとなりつつあります。
