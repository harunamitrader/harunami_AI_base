プロジェクト概要

「Roo-Code」は、VS Code 上で動作するオープンソースの自律型 AI コーディングエージェントです。以前は「Cline」（さらに前は Claude Dev）として知られていた人気プロジェクトが、コミュニティ主導の体制に移行し、さらなる進化を遂げた姿です。

単なるコード補完ではなく、ファイルシステムの操作、ターミナルの実行、ブラウザの自動化、そして MCP（Model Context Protocol）を介した外部ツールとの連携を AI が自律的に行い、開発者の指示に基づいて機能実装やデバッグを完遂します。

主な特徴

  • マルチモード対応: 実装を行う「Code Mode」、設計やプランニングに特化した「Architect Mode」、質問に答える「Ask Mode」、そして問題解決のための「Debug Mode」を切り替えて利用可能。
  • 自律的な実行能力: 修正が必要なファイルを特定し、差分を作成・適用し、ビルドやテストを実行して結果を確認するまでの一連のサイクルを自動化。
  • MCP サーバー連携: 外部の API やデータベース、ドキュメント検索エンジンなどを AI の「スキル」として統合。
  • カスタムモード: 特定のプロジェクトやチームに最適化された独自の動作モードを定義可能。
  • 最新モデルへの迅速な対応: OpenAI Codex (GPT-5.5) や Vertex AI 上の最新モデルなど、常に最先端の LLM を活用できます。

なぜトレンドに？

元々の Cline 開発チームが新しい方向性（Roomote）へ注力することを発表した際、コミュニティがこの強力な OSS 資産を維持するために「Roo-Code」としてフォークし、公式な継承を決定しました。

この「コミュニティによる復活」と、GPT-5.5 サポートを含む精力的なアップデート、そして 300 万インストールを超える圧倒的なユーザーベースが相まって、GitHub トレンドの常連となっています。自律型エージェントのデファクトスタンダードとしての地位を確立しつつあります。
