対象 repo 0Chencc/clawgod

これは何か

0Chencc/clawgod は、Anthropic が提供する AI コーディング CLI 「Claude Code」に対して直接適用するランタイム・パッチ（God Mode 改変版）です。サードパーティによるクローンや独立したクライアントではなく、インストール済みの公式 claude コマンドをフックし、独自の強力なサブコマンドや機能をシームレスに統合します。

何ができるか

このパッチを適用することで、公式の Claude Code に以下の新しい「神（God）」級コマンドが加わります：

  • /share: 会話やコードの共有を迅速に行う機能。
  • /teleport: プロジェクト内の異なる場所やコンテキストへの高速な移動操作。
  • /issue: GitHub Issues の確認・管理を CLI 内から直接実行。
  • /bughunter: バグの自動検出や修正提案に特化したワークフロー。

目立つポイント

最大の特徴は、「公式がアップデートされてもパッチが機能し続ける」設計にあります。独立したフォークプロジェクトとは異なり、インストーラーを実行するたびに、その時点の最新の Claude Code ソースに対して実行時にパッチを再適用する仕組みを採用しています。また、パッチ適用済みかどうかは、CLI 上のロゴの色（緑＝ patched / オレンジ＝ original）で視覚的に判別可能です。

セットアップや使い方の流れ

公式の Claude Code がインストールされている環境で、以下のワンラインコマンドを実行するだけです：

  • macOS / Linux: curl -fsSL https://github.com/0Chencc/clawgod/releases/latest/download/install.sh | bash
  • Windows (PowerShell): irm https://github.com/0Chencc/clawgod/releases/latest/download/install.ps1 | iex

パッチ適用後は、通常通り claude と打つだけで God Mode が有効になります（オリジナルの無改変版は claude.orig として保持されます）。

どんな人向けか

  • 公式 Claude Code の強力なツール体験をそのままに、さらに便利なサブコマンドを求めるパワーユーザー。
  • AI エージェントを用いた「Vibe Coding」をさらに効率化したい開発者。
  • 公式コマンドを「自分好みに拡張」したいハッカー気質なユーザー。

注意点

本プロジェクトは Anthropic 非公式 のパッチです。コード改変を伴うため、利用にあたっては自己責任（Use at your own risk）が原則となります。また、ライセンスは GPL-3.0 で公開されています。

まとめ

ClawGod は、Claude Code を「究極のコーディングツール」に仕立て上げるための、最もアグレッシブで実用的、かつスマートな拡張手段と言えるでしょう。公式の最新機能を追いつつ、独自の強力なフックを外せないユーザーにとって、必須のランタイム・カスタマイザーです。
